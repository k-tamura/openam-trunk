Le projet Open Web SSO (OpenSSO), en tant que composant de sécurité dans un reséau donné, fournit des services d'identité essentiels pour simplifier, de manière transparente, l'exécution de la connexion unique (SSO). OpenSSO fournit les bases pour l'intégration de divers applications Web fonctionnant sur un grand nombre de référentiels d'identités différents, installé sur différentes plates-formes, telle que des serveur Web et des serveur d'application. Ce projet est fondé sur le code source de Sun JavaTM System Access Manager, un produit d'infrastructure d'indentité, fournit par Sun Microsystems.
