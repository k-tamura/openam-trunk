Help: Installer Related Changes

This is the information that is required when your changes affect the installer.
  • if a new change causes a file/dir created?
  • if new change needs to be copied to webcontainer's site manually ?
  • if new change causes any of URI to be redeployed in order to work ?
  • if new change causes any further change (AMConfig, DS or WS/AS config)?
Please provide information to the Installation Team as soon as possible so that they can track these changes.
