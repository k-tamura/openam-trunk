The documents below are old versions of the documentation for OpenSSO Enterprise 8.0, published for the early access release in late 2008. The newest documents with the most current information can be found at the OpenSSO Documentation Center or docs.sun.com. DO NOT USE THE DOCUMENTS ON THIS PAGE.

Sun OpenSSO Enterprise 8.0 Early Access Documentation

This table contains links to PDF files (and one ZIP) of documentation written in late 2008 for the Sun productized version of OpenSSO. Be aware that not all procedures and technical information included has been reviewed or tested.
PDF files will open in a new window. Download and decompress the ZIP.

Title

Description

Release Notes

Contains details of the OpenSSO Early Access release, including new features, hardware and software requirements, and issues with workarounds, if available.

Technical Overview

Describes the architecture and how the many features work together to protect resources in your enterprise.

Installation and Configuration Guide

Describes OpenSSO installation procedures and configuration tasks.

Deployment Example: Single Sign-On, Load Balancing and Failover Using OpenSSO Enterprise 8.0

Contains the procedures to install and configure a basic SSO deployment including OpenSSO, Distributed Authentication, an external user data store, and policy agents.

Deployment Example: SAML v2 Using OpenSSO Enterprise 8.0

Contains the procedures for incorporating an environment using SAML v2. Instructions include configuring entity providers and testing the SAMLv2 configurations.

Developer's Guide

Provides information and procedures for use by engineers and developers.

Deployment Planning Guide

Provides information designed to help you build your OpenSSO Enterprise logical architecture.

Performance Tuning Guide

Provides instructions for using the OpenSSO Enterprise tuning script. Includes comprehensive tuning properties reference.

Administration Guide

Describes various administrative tasks such as Realms Management, Policy Management, Authentication and Directory Management.
Administration Reference

Provides information for administrators including sections on error codes, configuration files and attributes, and command-line tools.
Integration Guide

Provides high-level instructions for deploying OpenSSO Express with CA SiteMinder, Sun IdentityManager, and Oracle AccessManager.

C API Reference for Application and Web Policy Agent Developers

Provides overview information and a look-up listing of the C APIs (including functions, syntax, and parameters).

Java API Reference

Describes the Java API Library. (HTML files compressed in a ZIP file).

Upgrade Guide

Describes how to upgrade Sun Java System Access Manager and Sun Java System Federation Manager to OpenSSO Enterprise 8.0.
Policy Agent 3.0 User's Guide for J2EE agents
 Provides an overview of Policy Agent 3.0 and focuses on the features and configuration that is relevant to all the J2EE agents in this release.
Policy Agent 3.0 User's Guide for Web agents Provides an overview of Policy Agent 3.0 and focuses on the features and configuration that is relevant to all the web agents in this release.

3.0 Policy Agent Guide for Application Server 9.1

Provides installation and configuration information about the version 3.0 policy agent for Sun Java System Application Server 9.1 and Glassfish.

3.0 Policy Agent Guide for Web Server 7.0

Provides installation and configuration information about the version 3.0 policy agent for Sun Java System Web Server 7.0.

3.0 Policy Agent Guide for WebLogic Server/Portal 10

Provides installation and configuration information about the version 3.0 policy agent for BEA WebLogic Server/Portal 10.
