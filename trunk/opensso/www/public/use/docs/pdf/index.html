Access Manager 7.1 Documentation

This table contains links to the PDFs for the general release of Sun Java System Access Manager 7.1. (Access Manager is Sun Microsystems' productized version of the OpenSSO source code.) The books have been donated by Sun Microsystems.
NOTE: You can find the documentation in HTML form on docs.sun.com. The PDFs will open in a new window.

Title

Description

Release Notes

Contains information available at the time Access Manager 7.1 was released.

Technical Overview

Describes the architecture of many features of Access Manager 7.1 and how the features work together to protect resources in your enterprise.

Deployment Planning Guide

Provides planning and deployment solutions.

Postinstallation Guide

Describes configuration tasks performed after installing Access Manager.

Performance Tuning Guide

Provides recommendations and procedures on how you can tune Access Manager 7.1 and its related components to improve performance and reliability.

Administration Guide

Describes various administrative tasks such as Realms Management, Policy Management, Authentication and Directory Management.

Administration Reference Provides look-up information for administrators including sections on error codes, configuration files and attributes, and command-line tools.

C API Reference

Provides overview information and a look-up listing of the C APIs (including functions, syntax, and parameters).

Federation and SAML Administration Guide

Provides information about the features based on the Liberty Alliance Project and Security Assertions Markup Language (SAML) specifications including instructions for enabling a Liberty-based environment, and summaries of the application programming interface (API) for extending the framework.

Java API Reference

Describes the Java API Library. (HTML)

Policy Agent User's Guide

Describes Web and J2EE policy agents including installation and configuration information. (Individual agents are described in separate docs.)

Cookie Hijacking

This Technical Note provides information on the precautions you can take in an Access Manager deployment against cookie hijacking.

Localization Guide

This Technical Note describes the organization of localized Access Manager software resource files. It also explains how to customize an existing language's resource files and add a new language option.
