O projeto Open Web SSO (OpenSSO) fornece serviços de identidade de núcleo para simplificar a implementação de um transparente single sign-on (SSO) como um componente de segurança em uma infra-estrutura de rede. O OpenSSO fornece a base para a integridade de diversos aplicativos que poderiam tipicamente operar contra uma configuração diferente dos repositórios de identidade e estão hospedados em uma variedade de plataformas, como as da web e dos servidores dos aplicativos. Este projeto é baseado no código de base do Sun JavaTM System Access Manager, que é um produto de infra-estrutura de identidade de núcleo oferecido pela Sun Microsystems.
