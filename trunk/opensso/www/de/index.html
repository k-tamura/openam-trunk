Das Open Web SSO-Projekt (OpenSSO) bietet Core Identity Services, um die Einführung des transparenten Single Sign-on (SSO) als Sicherheitskomponente in einer Netzwerk-Infrastruktur zu vereinfachen. OpenSSO bietet eine Basis für die Integration von vielfältigen Web-Applikationen, die gegen ungleiche Sätze von Identität-Repositories funktionieren können und auf verschiedenen Plattformen, wie z. B. Web- und Application-Server, gehostet werden. Dieses Projekt basiert auf der Codebasis von Sun JavaTM System Access Manager - ein Core Identity Infrastructure Projekt von Sun Microsystems.
