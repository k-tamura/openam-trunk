Configuración del servicio ID-FF de Liberty

Nombre de cookie de federación

Este atributo especifica el nombre de la cookie de los servicios ID-FF. La cookie se utiliza para recordar si el usuario ya está federado.

Clase de implementación del SPI del buscador de proxy de IDP

Este atributo especifica la clase de implementación de búsqueda del proveedor de identidades favorito que se va a utilizar mediante proxy.

Intervalo de vaciado de la caché de solicitudes

Este atributo especifica el intervalo de vaciado (en segundos) del subproceso de vaciado de solicitudes internas de ID-FF.

Tiempo de espera de la caché de solicitudes

Este atributo especifica el valor de tiempo de espera (en segundos) de la solicitud de autenticación de ID-FF. Cualquier objeto AuthnRequest se purgará de la memoria si supera el valor de tiempo de espera.

URL de inicio de sesión de IDP

Este atributo especifica la URL de inicio de sesión a la que se redirigirá IDP si no se encuentra una sesión válida al procesar la solicitud de autenticación. Si no se especifica la clave, se utilizará una URL de inicio de sesión predeterminada.

Firma XML

Este atributo especifica el nivel de verificación de firmas para las solicitudes y respuestas de Liberty.
