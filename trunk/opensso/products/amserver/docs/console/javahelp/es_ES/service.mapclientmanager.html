Administrador de cliente

El Administrador de cliente es la interfaz que muestra los clientes base, los estilos y las propiedades asociadas, y permite agregar y configurar dispositivos.

Tipos de cliente base

Los tipos de cliente base se muestran en la parte superior del Administrador de cliente. Estos tipos de cliente contienen las propiedades predeterminadas que pueden heredar todos los dispositivos que pertenezcan al tipo de cliente.

Tipo de cliente

El Administrador de cliente agrupa a todos los clientes disponibles, incluido el propio tipo de cliente base, en la lista de tipos de cliente. Puede modificar las propiedades de cada cliente haciendo clic en el nombre del dispositivo. A continuación, las propiedades se mostrarán en la ventana Client Editor (Editor de cliente). Nota: para obtener definiciones específicas de las propiedades, consulte Open Mobile Alliance Ltd. (OMA) Wireless Application Protocol, Version 20-Oct-2001 en http://www1.wapforum.org/tech/terms.asp?doc=WAP-248-UAProf-20011020-a.pdf.

Es posible que para acceder a este documento tenga que registrarse en un foro WAP™. Para obtener más información, visite http://www.wapforum.org/faqs/index.htm.

Plataforma de hardware

Contiene las propiedades del hardware del dispositivo, como el tamaño de visualización, los conjuntos de caracteres admitidos, etc.

Plataforma de software

Contiene las propiedades del entorno de aplicación del dispositivo, el sistema operativo y el software instalado.

Características de red

Contiene las propiedades que describen el entorno de red, incluidos los portadores compatibles.

BrowserUA

Contiene los atributos relacionados con el agente de usuario del explorador que se ejecuta en el dispositivo.

WapCharacteristics

Contiene las propiedades del entorno del Protocolo de aplicaciones inalámbricas (WAP) admitido por el dispositivo.

PushCharacteristicNames

Contiene las propiedades del entorno WAP admitido por el dispositivo.

Propiedades adicionales

Propiedades adicionales. Le permite agregar propiedades adicionales al dispositivo.
