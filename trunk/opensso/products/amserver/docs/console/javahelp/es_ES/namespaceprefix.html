Prefijo de espacio de nombre

El valor de este atributo especifica el prefijo de espacio de nombre utilizados para los mensajes de protocolo XML del Servicio de perfil personal de Liberty. Un namespace diferencia elementos con el mismo nombre que provienen de esquemas XML distintos. El prefijo de espacio de nombre se coloca al principio del elemento y sirve para distinguir los metadatos de los distintos espacios de nombre del esquema XML.
