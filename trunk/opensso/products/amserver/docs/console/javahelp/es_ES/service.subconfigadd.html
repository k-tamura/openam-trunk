Para agregar una configuración secundaria

  1. Haga clic en Nuevo en la lista de instancias de configuración secundaria.
  2. Introduzca un nombre para la nueva configuración secundaria.
  3. Introduzca los datos para los siguientes campos:
    Usuario de almacén de la sesión

    Define el usuario de la base de datos que se utiliza para recuperar y guardar los datos de la sesión.

    Contraseña de almacén de la sesión

    Define la contraseña para el usuario de la base de datos definido en el almacén de la sesión.

    Contraseña de almacén de la sesión (confirmar)

    Confirme la contraseña.

    Tiempo máximo de espera

    Define el tiempo total que esperará un subproceso para adquirir el objeto de conexión de base de datos. El valor se muestra en milisegundos.

    URL de base de datos

    Especifica la dirección URL de la base de datos.

  4. Haga clic en Add.
