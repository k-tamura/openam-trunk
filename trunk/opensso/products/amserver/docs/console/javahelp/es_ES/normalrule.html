Reglas

Una regla contiene un tipo de servicio, una o varias acciones, y un valor. y define fundamentalmente la política.

  • Un tipo de servicio define el tipo de recurso que se está protegiendo.

  • Una acción hace referencia al nombre de una operación que puede realizarse en el recurso; POST y GET son ejemplos de acciones del servidor Web. Una acción permitida para los recursos humanos puede ser, por ejemplo, el cambio de un número de teléfono particular.

  • Un valor define el permiso de la acción, por ejemplo, permitir o denegar.
