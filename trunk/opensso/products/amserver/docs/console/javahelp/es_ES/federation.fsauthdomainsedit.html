Para modificar un perfil de círculo de confianza

Siga este procedimiento para editar los atributos generales configurados de un círculo de confianza existente o para agregar proveedores a éste. El punto de partida es Círculo de confianza en la interfaz Federación.

  1. Haga clic en el nombre de un círculo de confianza configurado para modificar su perfil o para agregarle proveedores.

    Aparece la página Editar círculo de confianza.

  2. Escriba los nuevos valores o edite los valores existentes para los atributos generales del círculo de confianza:
    Nombre

    El valor estático de este atributo es el nombre proporcionado al crear el círculo de confianza.

    Descripción

    El valor de este atributo es una descripción del círculo de confianza. Puede modificar la descripción introducida anteriormente si es necesario.

    URL del servicio de escritor de IDFF

    Este atributo especifica la ubicación del servicio que escribe la cookie del dominio común. La URL presenta el formato http://common-domain-host:port/common/idffwriter .

    URL del servicio de lector de IDFF

    Este atributo especifica la ubicación del servicio que lee la cookie del dominio común. La URL presenta el formato http://common-domain-host:port/common/idffreader .

    URL de escritor de SAML2

    Este atributo especifica la ubicación del servicio de escritor de SAML2 que escribe la cookie en el dominio común. La URL presenta el formato http://common-domain-host: port/common/saml2writer

    URL de lector de SAML2

    Este atributo especifica la ubicación del servicio de escritor de SAML2 que escribe la cookie en el dominio común. La URL presenta el formato http://common-domain-host: port/common/saml2reader

    Estado

    El estado predeterminado es “Activo” (Active). Si se selecciona la opción Inactivo, se deshabilita la comunicación con el círculo de confianza.

  3. Elija uno o varios de los proveedores disponibles y haga clic en la flecha Agregar para seleccionarlos.

    La lista proporcionada contiene los nombres de las entidades creadas y rellenadas con proveedores. Para obtener más información, consulte Para agregar proveedores a un círculo de confianza.

  4. Haga clic en Guardar para completar la operación.
