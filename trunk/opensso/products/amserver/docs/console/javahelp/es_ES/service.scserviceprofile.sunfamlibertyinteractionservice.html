Servicio de interacción de Liberty

WSP de redirección de usuarios para la interacción

Indica si el proveedor de servicios web redirigirá al usuario para obtener su consentimiento. El valor predeterminado es yes.

WSP de redirección de usuarios para la interacción de datos

Esta propiedad inicia una interacción para obtener el consentimiento del usuario o recopilar datos adicionales. Indica si el proveedor de servicios web redirigirá al usuario para recopilar datos adicionales. El valor predeterminado es yes.

Duración prevista de WSP para la interacción

Este atributo indica el periodo de tiempo (en segundos) previsto por el proveedor de servicios web para completar una interacción y devolver el control al cliente del servicio web. Por ejemplo, el proveedor de servicios web recibe una solicitud en la que se indica que el cliente del servicio web esperará un máximo de 30 de segundos (establecido en Duración prevista del WSP para la interacción) a que se realice la interacción. Si este atributo se establece en 40 segundos, el proveedor de servicios web devolverá un error de SOAP (timeNotSufficient ), en el que se indica que no hay suficiente tiempo para la interacción.

WSP para forzar que "returnToURL" sea SSL

Este atributo indica si el proveedor de servicios web debe aplicar el elemento returnToURLHTTPS especificado por el cliente del servicio web. Las especificaciones de Liberty Alliance Project indican que el valor de esta propiedad es siempre "yes" (sí). El valor "false" (falso) se utiliza principalmente para simplificar la implementación de forma escalonada.

WSP para forzar que la devolución al host sea igual al host solicitado

Este atributo indica si el proveedor de servicios web debe aplicar los valores de dirección de returnToHost y requestHost en el caso de que sean iguales. Las especificaciones de Liberty Alliance Project indican que el valor de esta propiedad es siempre "yes" (sí). El valor "false" (falso) se utiliza principalmente para simplificar la implementación de forma escalonada.

Ubicación de la hoja de estilo HTML

Este atributo señala la ubicación de la hoja de estilo utilizada para procesar la página de interacción con el formato HTML.

Ubicación de la hoja de estilo WML

Este atributo señala la ubicación de la hoja de estilo utilizada para procesar la página de interacción con el formato WML.

URL de interacción de WSP

Este atributo indica la URL en la que se ha implementado el servlet WSPRedirectHandler . El servlet administra la parte de las interacciones del proveedor de servicios para las redirecciones de usuarios.

URL de interacción de WSP si se encuentra tras un equilibrador de carga

Define la URL del controlador de redirección de WSP mostrada por el equilibrador de carga.

Lista de URL de interacción del clúster de WSP (sitio) tras el equilibrador de carga

Define las URL del controlador de redirección de WSP de los servidores de confianza del clúster.

Clase de configuración de la interacción

Este atributo especifica la clase que proporciona métodos de acceso para leer las configuraciones de la interacción.

Opciones de WSC para participar en la interacción

Este atributo indica el nivel de la interacción en la que participará WSC si se configura para participar en las redirecciones de usuarios. Los posibles valores son los siguientes: "interactIfNeeded", "doNotInteract" y "doNotInteractForData". El valor "interactIfNeeded" afirmativo se utiliza de forma predeterminada.

WSC para incluir "userInteractionHeader"

Este atributo indica si el cliente del servicio web incluirá un encabezado SOAP para indicar determinadas preferencias de la interacción en función de las especificaciones de Liberty. El valor predeterminado es yes.

WSC de redirección de usuarios para la interacción

Este atributo establece si WSC participará en las redirecciones de usuarios. El valor predeterminado es yes.

Duración prevista de WSC para la interacción

Este atributo define el periodo de tiempo máximo en segundos que el cliente del servicio web está dispuesto a esperar al proveedor de servicios web para que complete su parte de la interacción. El proveedor de servicios web no iniciará ninguna interacción si es posible que ésta tarde más tiempo del que se ha establecido. Por ejemplo, el proveedor de servicios web recibe una solicitud con esta propiedad establecida en un máximo de 30 segundos. Si la propiedad "Duración prevista de WSP para la interacción" del proveedor de servicios web se ha establecido en 40 segundos, el proveedor de servicios web devolverá un error de SOAP (timeNotSufficient), en el que se indica que no hay suficiente tiempo para la interacción.

WSC para forzar que la URL de redirección sea SSL

Este atributo especifica si el cliente del servicio web aplicará HTTPS en las URL de redirección. Las especificaciones de Liberty Alliance Project indican que el valor de esta propiedad es siempre "yes" (sí), lo que implica que el proveedor de servicios web no redirigirá al usuario cuando el valor de redirectURL (especificado por el proveedor de servicios web) no sea una URL HTTPS. El valor "false" (falso) se utiliza principalmente para realizar una implementación sencilla y escalonada.
