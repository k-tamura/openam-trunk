General

Los atributos generales permiten configurar los datos de configuración básicos para la administración centralizada del servidor.

Atributos del sitio

El atributo del sitio es:

Sitio principal

Este atributo asigna el nombre del sitio (Id. de sitio) del equilibrador de carga al servidor de OpenSSO. Tenga en cuenta que debe crear primero el sitio para poder agregarlo.

Atributos del sistema

Los atributos del sistema enumeran la información de ubicación de la instancia del servidor:

Directorio de instalación base

Especifica el directorio base en el que residen los datos del producto.

Configuración regional predeterminada

El valor de configuración regional es el subtipo de idioma predeterminado con el que se instaló OpenSSO. El valor predeterminado es en_us.

URL de notificación

La ubicación del punto final del servicio de notificación. Este valor se establece durante la instalación.

Validación XML

El valor predeterminado es no. Determina si la validación es necesaria al analizar documentos XML utilizando la clase XMLUtils de OpenSSO. Esta propiedad sólo se aplica cuando el valor del atributo de nivel de depuración se establece en advertencia o mensaje. Los valores permitidos son "yes" (sí) y "no". La validación de documentos XML sólo se activa si el valor de esta propiedad es "yes" (sí) y el valor del atributo de nivel de depuración se ha establecido en advertencia o mensaje.

Atributos de depuración

Los atributos de depuración muestran la información básica de comprobación de errores:

Nivel de depuración

Especifica el nivel de depuración. El valor predeterminado es "error". Los valores posibles son:

  • off (desactivado): no se crea ningún archivo de depuración.

  • error: sólo se registran los mensajes de error.

  • warning (advertencia): sólo se registran los mensajes de advertencia.

  • message (mensaje): se registran los mensajes de error, advertencia o de carácter informativo.

Directorio de depuración

Especifica el directorio de salida en el que se crearán los archivos de depuración. Este valor se establece durante la instalación. Ejemplo: OpenSSO-base /debug_URI/debug.

Servidor de correo

Los atributos del servidor de correo muestran el nombre de host y el puerto del servidor de correo:

Nombre de host del servidor de correo

El valor predeterminado es localhost. Especifica el host del servidor de correo.

Número de puerto del servidor de correo

El valor predeterminado es 25. Especifica el puerto del servidor de correo.
