Para crear un nuevo agente

La creación de un nuevo agente también recibe el nombre de creación de un nuevo perfil de agente. Puede realizar esta tarea en la consola de administración de OpenSSO. Algunos de los pasos concretos se aplican únicamente a los tipos de agente de directivas J2EE y web.

  1. Haga clic en la ficha Control de acceso.
  2. Haga clic en el nombre del dominio al que pertenecerá el agente.
  3. Haga clic en la ficha Agents (Agentes).
  4. Seleccione la ficha del tipo de agente adecuado.
    • Agentes web
    • Agentes J2EE
    • Agentes del proveedor de servicios web
    • Agentes del cliente de servicios web
    • Agente de cliente STS

      El agente de cliente del Servicio de token de seguridad permite elegir entre dos tipos, Agente de detección y Agente STS. Para más información sobre estos tipos de agente, consulte Cliente STS.

    • Agentes 2.2.

      La ficha Agentes 2.2 se utiliza para los agentes del conjunto de software Policy Agent 2.2.

    • Autenticador de agentes
  5. En la sección Agente, haga clic en Nuevo.
  6. En el campo Nombre, introduzca el nombre del nuevo perfil de agente.
  7. Introduzca y confirme la contraseña.

    Precaución – Esta contraseña debe ser igual a la que se introduzca en el archivo de contraseñas del perfil de agente especificado al ejecutar el programa agentadmin para instalar el agente.

    Pasos 8–10 Aplicar únicamente a agentes de directiva web y J2EE.

  8. Seleccione la configuración aplicable: local o centralizada

    Cuando se selecciona la configuración local, las propiedades relacionadas a este agente no podrán editarse desde la consola. En un escenario como éste, el agente recupera la información de configuración de los archivos locales, OpenSSOAgentBootstrap.properties y OpenSSOAgentConfiguration.properties , en el directorio de instalación del agente. Para un agente configurado de forma local, cambie los valores de la propiedad editando directamente el archivo OpenSSOAgentConfiguration.properties .

  9. En el campo URL del servidor, introduzca la URL del servidor de OpenSSO.

    Por ejemplo:

    http://OpenSSO_EnterpriseHost.example.com:8080/OpenSSO
  10. En el campo URL del agente, introduzca la URL para la aplicación del agente, agentapp .

    Por ejemplo:

    Agentes web

    http://agentHost.example.com:8090

    Agentes J2EE

    http://agentHost.example.com:8090/agentapp

  11. Haga clic en Create (Crear).

    La Consola crea el perfil del agente y muestra de nuevo la página del agente, con un vínculo al nuevo perfil del agente.

    Si desea realizar la configuración adicional del perfil de agente, haga clic en este vínculo para que se muestre la página Editar agente.
