Para crear o modificar un usuario
  1. Haga clic en la ficha Users (Usuarios).
  2. Haga clic en Nuevo.
  3. Introduzca los datos para los siguientes campos:

    Id. de usuario Este campo incluye el nombre del usuario con el que se iniciará una sesión en Federated Access Manager. Es posible que se trate de un valor diferente a un DN.

    Nombre. Este campo incluye el nombre del usuario.

    Apellidos.: este campo incluye los apellidos del usuario.

    Nombre completo. Este campo incluye el nombre completo del usuario.

    Password (Contraseña). Este campo contiene la contraseña del nombre especificado en el campo "Id. de usuario".

    Contraseña (confirmar). Confirme la contraseña.

    Estado de usuario. Esta opción indica si el usuario puede autenticarse mediante Federated Access Manager.

  4. Haga clic en Aceptar.
  5. Una vez creado el usuario, puede hacer clic en su nombre para editar la información de usuario. Consulte los atributos de usuario para obtener más información. Puede realizar las siguientes modificaciones adicionales:

    • Para agregar servicios a una identidad

    • Para agregar un usuario a roles y grupos
