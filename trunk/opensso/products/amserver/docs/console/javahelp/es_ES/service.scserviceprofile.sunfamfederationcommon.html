Longitud máxima permitida del contenido

Este atributo especifica la longitud máxima permitida de contenido para una Solicitud HTTP que se utilizará en servicios de federación. Se rechazará cualquier solicitud cuyo contenido supere la longitud máxima de contenido permitida.

Clase de implementación del SPI del decodificador de contraseñas

Este atributo especifica la clase de implementación de la interfaz com.sun.identity.saml.xmlsig.PasswordDecoder que se utiliza para decodificar la contraseña guardada del almacén de claves de firmas XML y la contraseña de autenticación básica en SAML 1.x.

Clase de implementación del SPI del proveedor de firmas

Este atributo especifica la clase del proveedor de firmas XML de SAML. El SPI predeterminado es com.sun.identity.saml.xmlsig.AMSignatureProvider .

Clase de implementación del SPI del proveedor de claves

Este atributo especifica la clase del proveedor de claves de firmas XML. El SPI predeterminado es com.sun.identity.saml.xmlsig.JKSKeyProvider.

Comprobar la presencia de los certificados

Si se establece en en, el certificado debe estar presente en el almacén de claves para la validación de firmas XML. Si, por el contrario, se establece en off, se omitirá la comprobación de presencia del certificado. Esto sólo se aplica a SMAL1.x.

Algoritmo de canonicalización XML

Este atributo especifica el algoritmo de canonicalización XML utilizado para la creación y verificación de firmas XML de SAML. El valor predeterminado es http://www.w3.org/2001/10/xml-exc-c14n#.

Algoritmo de firma XML

Este atributo especifica el algoritmo de firma XML utilizado para la creación y verificación de firmas XML de SAML. Si no se especifica o el valor está vacío, se utilizará el valor predeterminado (http://www.w3.org/2000/09/xmldsig#rsa-sha1 ).

Algoritmo de transformación XML

Este atributo especifica el algoritmo de transformación XML utilizado para la creación y verificación de firmas XML de SAML. Si no se especifica o el valor está vacío, se utilizará el valor predeterminado (http://www.w3.org/2001/10/xml-exc-c14n# ).
