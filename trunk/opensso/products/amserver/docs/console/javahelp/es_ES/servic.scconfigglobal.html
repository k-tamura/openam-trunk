Propiedades globales

Las propiedades globales contienen servicios que permiten definir la función de restablecimiento de contraseña y la configuración de directivas para OpenSSO. Los servicios que se pueden configurar son los siguientes:

  • Configuración de la federación común

  • Configuración del servicio ID-FF de Liberty

  • Servicio de seguridad ID-WSF de Liberty

  • Servicio de interacción de Liberty

  • Protocolo de federación múltiple

  • Reinicialización de contraseña

  • Configuración de políticas

  • Configuración del servicio SAMLv2

  • Enlace SOAP de SAMLv2

  • Servicio de testigos de seguridad

  • Sesión

  • Para agregar una configuración secundaria

  • Usuario
