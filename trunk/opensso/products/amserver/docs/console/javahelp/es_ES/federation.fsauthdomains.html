Círculo de confianza.

Un círculo de confianza, denominado anteriormente dominio de autenticación, es una federación de un número cualquiera de proveedores de servicios (y, al menos, un proveedor de identidades) con los que las entidades principales pueden tramitar negocios en un entorno seguro y aparentemente perfecto. Para crear y rellenar un círculo de confianza, debe crear primero una entidad que aloje los metadatos (la información de configuración que define una arquitectura de servicios de identidades determinada) de cada proveedor que se convertirá en miembro del círculo de confianza. Posteriormente, se configura y guarda el círculo de confianza. Por último, para agregar una entidad (un proveedor de configuración) al círculo de confianza, deberá editar las propiedades de la entidad.

Las tareas siguientes están asociadas a círculos de confianza:

  • Para crear un nuevo círculo de confianza

  • Para modificar un perfil de círculo de confianza

  • Para agregar proveedores a un círculo de confianza

  • Para eliminar un perfil de círculo de confianza
