Atributos generales WS-FED

Los siguientes atributos son comunes a los tipos Proveedor de identidades y de servicio:

Nombre de visualización de SP

Este atributo define el nombre del proveedor de servicios WS-FED. El valor predeterminado es el meta alias proporcionado en el momento de la creación.

Nombre de visualización de IDP

Este atributo define el nombre del proveedor de identidades WS-FED. El valor predeterminado es el meta alias proporcionado en el momento de la creación.

Dominio

Muestra el dominio al que pertenece el proveedor.

Número de emisor del token

Define un identificador único para el proveedor de identidades o servicio.

Punto final del emisor del token

Especifica la URL en la que el proveedor de identidades o servicio está proporcionando servicios de WS-FED. Por ejemplo:

https://demo.example.com/opensso/WSFederationServlet/metaAlias/example
