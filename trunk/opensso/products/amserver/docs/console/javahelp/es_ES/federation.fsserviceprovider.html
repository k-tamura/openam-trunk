Para agregar un proveedor de servicios a una entidad de proveedores

El primer paso del proceso de adición de un proveedor de servicios a una entidad de proveedores consiste en definir el tipo de proveedor de servicios que se va a agregar.

  • Un proveedor alojado está alojado en el mismo servidor que Federated Access Manager.

  • Un proveedor remoto no está alojado en el mismo servidor que Federated Access Manager.

El siguiente procedimiento describe este paso inicial. El punto de partida es Entidades en la interfaz Federación.

  1. Seleccione el nombre de una entidad de proveedores configurada para modificar su perfil.

    Aparece la página de perfil de la entidad.

  2. Elija "Proveedor de servicios" (Service Provider) en el menú "Ver".
  3. Seleccione el tipo de proveedor que desea agregar:

    • Seleccione Nuevo proveedor alojado si el proveedor está alojado en el mismo servidor que Federated Access Manager.

    • Seleccione Nuevo proveedor remoto si el proveedor no está alojado en el mismo servidor que Federated Access Manager.

    Aparece la página de perfil del proveedor de servicios correspondiente.

  4. Según su selección en el paso anterior, elija uno de los siguientes procedimientos:
    • Para agregar un nuevo proveedor de servicios alojado a una entidad de proveedores
    • Para agregar un nuevo proveedor de servicios remoto a una entidad de proveedores
