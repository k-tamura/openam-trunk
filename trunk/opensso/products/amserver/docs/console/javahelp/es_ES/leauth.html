Condición de autenticación LE

La política se aplica si el nivel de autenticación del usuario es inferior o igual al nivel de autenticación definido en la condición.

La condición de nivel de autenticación puede utilizarse para especificar niveles diferentes a los del módulo de autenticación registrado para ese dominio. Esta opción resulta de utilidad cuando la política se aplica a un usuario autenticado desde otro dominio.
