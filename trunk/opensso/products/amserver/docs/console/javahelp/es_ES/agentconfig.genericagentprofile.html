Configuración de agentes

Una vez que haya creado un agente puede personalizar la función de cada uno de ellos. Para hacerlo, primero haga clic en el nombre del agente que desea configurar y, a continuación, modifique los atributos del agente. Consulte las siguientes secciones para conocer las definiciones de cada tipo de agente y sus funciones definidas:

  • Proveedor de servicios web

  • Atributos de clientes de servicios web

  • Atributos del Agente de detección

  • Atributos del agente del servicio de token de seguridad

  • Agente de directivas 2.2

  • Autenticador de agentes
