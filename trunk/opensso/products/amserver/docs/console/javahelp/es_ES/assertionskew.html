Factor de contracción de aserción para hora notBefore

Este atributo se utiliza para calcular la hora notBefore de una aserción. Por ejemplo, si IssueInstant es 2002-09024T21:39:49Z y Assertion Skew Factor For notBefore Time se ha establecido en 300 segundos (180 es el valor predeterminado), el atributo notBefore del elemento de condiciones de la aserción será 2002-09-24T21:34:49Z.

Nota – La duración total válida de una aserción se define mediante los valores establecidos en los atributos Assertion Timeout y Assertion Skew Factor For notBefore Time.
