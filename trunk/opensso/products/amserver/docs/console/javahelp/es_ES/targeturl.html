URL de destino

Si la URL de destino recibida mediante cualquier perfil se muestra como un valor de este atributo, las aserciones recibidas se enviarán a la URL de destino mediante el envío de un formulario HTTP (HTTP FORM POST).

Nota – Para editar o duplicar los atributos del perfil de un socio de confianza, haga clic en el botón correspondiente de la columna Acciones situada junto al nombre del socio de confianza configurado.
