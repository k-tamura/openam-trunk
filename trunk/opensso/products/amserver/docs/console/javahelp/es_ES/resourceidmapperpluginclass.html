Clases de complemento del asignador de Id. de recursos

El valor de este atributo es una lista de las clases que generan identificadores para una oferta de recursos configurada para una organización o rol. com.sun.identity.liberty.ws.interfaces.ResourceIDMapper is an interface used to map a user identifier to the resource identifier associated with it. El Servicio de detección proporciona dos implementaciones para esta interfaz:

  • com.sun.identity.liberty.ws.disco.plugins.Default64ResourceIDMapper (format: providerID + "/" + the Base64 encoded userIDs)

  • com.sun.identity.liberty.ws.disco.plugins.DefaultHexResourceIDMapper (format: providerID + "/" + the hex string of userID)

Se pueden desarrollar distintas implementaciones con la clase de implementación y agregarlas como valor de este atributo. Para ello, haga clic en Nuevo y utilice el siguiente formato:

providerid=providerID| class_name_and_path
