Creación de un nuevo complemento de repositorio de Access Manager

En esta sección, se enumeran los pasos para crear un complemento de repositorio.

Para crear un nuevo complemento de repositorio de Access Manager

  1. Seleccione el dominio en el que desea configurar el complemento de repositorio de Access Manager.

  2. Haga clic en la ficha Data Store (Almacén de datos).

  3. Haga clic en New (Nuevo) en la lista de almacenes de datos.

  4. Introduzca un nombre para el complemento de repositorio.

  5. Seleccione Access Manager Repository Plugin (Complemento de repositorio de Access Manager).

  6. Haga clic en Next (Siguiente).

  7. Defina los siguientes campos:

    Nombre de clase de complemento de Access Manager

    Especifica la ubicación del archivo de clase que implementa el complemento de repositorio de Access Manager.

    Organización de Access Manager

    El DN que señala la organización de Directory Server que será administrada por Access Manager. Se trata del DN base de todas las operaciones realizadas en este almacén de datos.

  8. Haga clic en Finish (Finalizar).
