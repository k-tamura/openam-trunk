Lista de asignación de atributos PPLDAP

Cada atributo de identidad definido por el Servicio de perfil personal de Liberty presenta una coincidencia exclusiva con un atributo LDAP. El valor de este atributo consiste en una lista que especifica dichas asignaciones. Por ejemplo, JobTitle=sunIdentityServerPPEmploymentIdentityJobTitle asigna el atributo JobTitle de Liberty al atributo sunIdentityServerPPEmploymentIdentityJobTitle de OpenSSO. Al agregar nuevos atributos a cualquiera de las partes de esta ecuación, asegúrese de que las asignaciones del nuevo atributo estén configuradas en este atributo. Consulte Para configurar una asignación de atributos.
