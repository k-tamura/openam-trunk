Ventana de sesión

Muestra los usuarios que han iniciado sesión en OpenSSO y el tiempo de la sesión de cada uno de ellos. Los campos de visualización son:

ID de usuario

Muestra el Id. del usuario que ha iniciado sesión.

Time Remaining (Tiempo restante, en minutos)

Muestra el tiempo restante (en minutos) que tiene el usuario para dicha sesión antes de tener que volver a autenticarse.

Max Session Time (Tiempo máximo de sesión, en minutos)

Muestra el tiempo máximo (en minutos) que puede estar conectado el usuario antes de que la sesión finalice y deba volver a autenticarse para acceder de nuevo.

Time Idle (Tiempo de inactividad, en minutos)

Muestra el tiempo (en minutos) que el usuario ha estado inactivo.

Max Idle Time (Tiempo máximo de inactividad , en minutos)

Muestra el tiempo máximo (en minutos) que un usuario puede permanecer inactivo antes de tener que volver a autenticarse. El administrador define los límites de tiempo en el servicio Sesión. Consulte Sesión para obtener definiciones de los atributos de sesión.
