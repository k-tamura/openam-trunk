Ofertas de recursos para bootstrapping

Este atributo define una oferta de recursos para iniciar (bootstrapping) un servicio. Esta oferta de recursos y las credenciales asociadas se enviarán al cliente tras un inicio de sesión único (SSO) en la aserción del SSO. Sólo está permitida una oferta de recursos para bootstrapping; de forma predeterminada, esta oferta contiene información relacionada con el Servicio de detección. Entre las tareas asociadas a este atributo, se incluyen:

  • Para configurar una oferta de recursos para Bootstrapping

  • Para configurar una descripción de servicio
