Preferencias de globalización

El servicio de preferencias de globalización contiene atributos globales que permiten configurar OpenSSO para las diferentes configuraciones regionales y conjuntos de caracteres. Los atributos son:

  • Conjunto de caracteres admitido por cada idioma

  • Alias de los conjuntos de caracteres

  • Formato de nombre común generado automáticamente

Conjunto de caracteres admitido por cada idioma

Este atributo muestra los conjuntos de caracteres que admite cada configuración regional, lo que indica la asignación entre las configuraciones regionales y el conjunto de caracteres. El formato es el siguiente:

Para agregar un nuevo juego de caracteres compatible, haga clic en Agregar y defina los siguientes parámetros:

Configuración regional

La nueva configuración regional que desee agregar. Consulte Idiomas admitidos para obtener más información.

Conjuntos de caracteres admitidos

Introduzca el conjunto de caracteres admitido para la configuración regional especificada. Los conjuntos de caracteres están delimitados por punto y coma. Por ejemplo, charset=charset1;charset2;charset3;...;charsetn

Para editar un conjunto de caracteres compatible existente, haga clic en el nombre en la tabla Supported Charsets (Conjuntos de caracteres admitidos). Pulse "Aceptar" cuando haya terminado.

Alias de los conjuntos de caracteres

Este atributo muestra los nombres del conjunto de códigos (que se asignan a los nombres IANA) que se utilizará para enviar la respuesta. Estos nombres del conjunto de códigos no tienen que coincidir con los del conjunto de códigos de Java. En la actualidad, existe una tabla Hash que permite asignar los conjuntos de caracteres de Java a conjuntos de caracteres IANA y viceversa.

Para agregar un nuevo alias de conjunto de caracteres, haga clic en Agregar y defina los siguientes parámetros:

Nombre de MIME

El nombre de asignación IANA. Por ejemplo, Shift_JIS

Nombre de Java

El conjunto de caracteres de Java definido para asignarse al conjunto de caracteres IANA.

Para editar un alias de conjunto de caracteres existente, haga clic en el nombre en la tabla. Pulse "Aceptar" cuando haya terminado.

Formato de nombre común generado automáticamente

Esta opción de visualización permite definir la forma en la que un nombre se genera automáticamente para adaptar los formatos de nombres a distintas configuraciones regionales y conjuntos de caracteres. La sintaxis predeterminada es la siguiente (tenga en cuenta que la inclusión de comas o espacios en la definición se mostrará en el formato del nombre):

en_us = {givenname} {initials} {sn}

Por ejemplo, si deseaba ver un nuevo formato de nombre para un usuario (Usuario uno) con un uid (11111) para el conjunto de caracteres chino, defina:

zh = {sn}{givenname}({uid})

Se mostrará:

Usuario uno 11111
