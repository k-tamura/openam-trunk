Plataforma

El servicio de plataforma es aquél en el que se pueden agregar servidores adicionales a las configuraciones de OpenSSO, así como otras opciones aplicadas en el nivel superior de la aplicación OpenSSO. Los atributos del servicio de plataforma son globales. Los atributos son:

  • Idioma de plataforma

  • Dominios de cookies

  • Idiomas disponibles

  • Conjunto de caracteres del cliente

Idioma de plataforma

El valor de configuración regional de la plataforma es el subtipo de idioma predeterminado con el que se instaló OpenSSO. Los servicios administración, registro y autenticación se administran con el idioma de este valor. El valor predeterminado es en_US. Consulte Idiomas admitidos para obtener una lista de los subtipos de idiomas admitidos.

Dominios de cookies

La lista de dominios que se devolverán en el encabezado de cookie al configurar una cookie en el explorador del usuario durante la autenticación. Si se deja vacío, no se configurará ningún dominio de cookie. Es decir, la cookie de sesión de OpenSSO sólo se reenviará a esta aplicación y no al resto de servidores del dominio.

Si se requiere SSO con otros servidores del dominio, este atributo debe configurarse con el dominio de cookies. Si dispone de dos interfaces en dominios distintos de una instancia de OpenSSO, deberá configurar ambos dominios de cookies en este atributo. Si se utiliza un equilibrador de carga, el dominio de cookies debe ser el del dominio del equilibrador de carga y no los servidores detrás del equilibrador. El valor predeterminado de este campo es el dominio de la instancia de OpenSSO instalada.

Idiomas disponibles

Este atributo almacena todos las configuraciones regionales configuradas para la plataforma. Considere una aplicación que permite al usuario elegir la configuración regional del usuario. La aplicación obtendrá el atributo del perfil de la plataforma y mostrará al usuario la lista de configuraciones regionales. El usuario elegirá una configuración regional y la aplicación la configurará en la entrada de usuario preferredLocale.

Conjunto de caracteres del cliente

Este atributo especifica el conjunto de caracteres establecido para los distintos clientes en el nivel de la plataforma. Contiene una lista con los tipos de clientes y los correspondientes conjuntos de caracteres. Consulte Para crear un conjunto de caracteres nuevo para obtener más información.
