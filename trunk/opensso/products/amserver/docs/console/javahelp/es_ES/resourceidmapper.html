Asignador de ID de recursos

The value of this attribute specifies the implementation of com.sun.identity.liberty.ws.interfaces.ResourceIDMapper. Aunque es posible desarrollar una nueva implementación, OpenSSO proporciona la clase predeterminada com.sun.identity.liberty.ws.idpp.plugin.IDPPResourceIDMapper , que asigna un identificador de recurso de detección a un Id. de usuario.
