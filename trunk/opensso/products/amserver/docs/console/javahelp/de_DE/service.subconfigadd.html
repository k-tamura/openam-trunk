So fügen Sie eine Subkonfiguration hinzu

  1. Klicken Sie in der Liste "Sekundäre Konfigurationsinstanz" auf "Neu".
  2. Geben Sie einen Namen für die neue Unterkonfiguration ein.
  3. Geben Sie in folgenden Feldern die entsprechenden Werte ein:
    Speicherbenutzer der Sitzung

    Legt den Datenbankbenutzer fest, der zum Abrufen und Speichern von Sitzungsdaten verwendet wird.

    Speicherpasswort der Sitzung

    Legt das im Sitzungsspeicher definierte Passwort für die Datenbank fest.

    Speicherpasswort der Sitzung (Bestätigung)

    Bestätigen Sie das Passwort.

    Maximale Wartezeit

    Legt die Gesamtdauer fest, die ein Thread für den Erwerb eines Datenbankverbindungsobjekts bereitsteht. Der Wert wird in Millisekunden angegeben.

    Datenbank-URL

    Gibt den URL der Datenbank an.

  4. Klicken Sie auf „Hinzufügen“.
