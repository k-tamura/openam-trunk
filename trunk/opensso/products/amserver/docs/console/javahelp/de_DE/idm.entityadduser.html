So erstellen oder ändern Sie einen Benutzer
  1. Klicken Sie auf die Registerkarte "Benutzer".
  2. Klicken Sie auf "Neu".
  3. Geben Sie in folgenden Feldern die entsprechenden Werte ein:

    Benutzer-ID. In diesem Feld wird der Name eingegeben, unter dem sich der Benutzer bei OpenSSO anmeldet. Es kann sich hier auch um einen Nicht-DN-Wert handeln.

    First Name. This field takes the first name of the user.

    Nachname In diesem Feld wird der Nachname des Benutzers angegeben.

    Vollständiger Name In diesem Feld wird der vollständige Name des Benutzers angegeben.

    Password. This field takes the password for the name specified in the User Id field.

    Passwort bestätigen Bestätigen Sie das Passwort.

    Benutzerstatus. Mit dieser Option wird angegeben, ob der Benutzer sich über OpenSSO authentifizieren kann.

  4. Klicken Sie auf „OK“.
  5. Nachdem der Benutzer erstellt ist, können Sie auf dessen Namen klicken, um die Benutzerinformationen zu ändern.
