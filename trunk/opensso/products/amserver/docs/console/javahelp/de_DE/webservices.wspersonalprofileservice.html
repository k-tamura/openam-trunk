Persönlicher Liberty-Profildienst

Die Liberty ID-SIS Personal Profile-Dienstspezifikation beschreibt einen Datendienst, der allgemeine Profilinformationen zu einer Identität bereitsstellt. Der Dienst dient als letzter gemeinsamer Nenner für das Speichern von benutzerbasierten Informationen zu einem Principal. OpenSSO hat diese Spezifikation implementiert und die Persönlicher Liberty-Profildienst ausgearbeitet. Die Persönlicher Liberty-Profildienst kann hinsichtlich Identitätsdaten abgefragt werden bzw. diese Daten können aktualisiert werden.

Die folgenden angegebenen globalen Attribute können für Ihre Implementierung der Persönlicher Liberty-Profildienst konfiguriert werden.

  • ResourceID-Mapper

  • Berechtigte Person

  • Attribut-Mapper

  • Anbieter-ID

  • Namensschema

  • NameSpace-Präfix

  • Unterstützte Container

  • PPLDAP-Attributzuordnungsliste

  • Richtlinienüberprüfung bei Änderungen erforderlich:

  • Container-Erweiterungsattribute

  • Erweiterungsattribute NameSpace-Präfix

  • Dienstaktualisierung

  • Dienstinstanz-Aktualisierungsklasse

  • Alternativer Endpunkt

  • Alternativer Sicherheitsmechanismus
