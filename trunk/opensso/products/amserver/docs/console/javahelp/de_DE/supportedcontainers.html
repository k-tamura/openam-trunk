Unterstützte Container

Mit den Werten dieses Attributs wird eine Liste der in Persönlicher Liberty-Profildienst unterstützten Container definiert. In dieser Instanz ist ein Container ein Attribut von Persönlicher Liberty-Profildienst. "Kontakt im Notfall" und "Gemeinsamer Name" sind beispielsweise zwei Standardcontainer.

Hinweis – Der hier beschriebene Begriff Container ist nicht mit dem identitätsbezogenen Objekt in OpenSSO zu verwechseln, das ebenfalls als Container bezeichnet wird.

Zum Hinzufügen eines neuen Containers klicken Sie auf "Neu". Fahren Sie danach mit So konfigurieren Sie einen unterstützten Container fort.
