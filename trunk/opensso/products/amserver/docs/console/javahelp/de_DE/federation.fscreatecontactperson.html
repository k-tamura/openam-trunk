So fügen Sie einem Anbieter in einer Anbieter-Entity eine Kontaktperson hinzu

Im Anbieterprofil einer Anbieter-Entity können Sie Informationen zur Kontaktperson eines bestimmten Dienst- oder Identity-Anbieters eintragen. Der Ausgangspunkt ist der Bildschirm "Entities" unter "Verbindung".

  1. Klicken Sie auf den Namen einer konfigurierten Entity, um deren Profil zu ändern.

    Die Profilseite der Entity wird angezeigt.

  2. Wählen Sie im Menü "Ansicht" die Option "Identity-Anbieter" oder "Dienstanbieter".
  3. Klicken Sie in der Attributgruppe "Kontaktpersonen" auf "Neue Kontaktperson".
  4. Geben Sie die Attributwerte für die neue Kontaktperson ein.
    Vorname

    Geben Sie den Vornamen der Entity-Kontaktperson ein.

    Nachname

    Geben Sie den Nachnamen der Entity-Kontaktperson ein.

    Typ

    Wählen Sie den Kontakttyp im Dropdown-Menü aus:

    • Administrativ

    • Rechnungswesen

    • Technisch

    • Weitere

    Firma

    Geben Sie den Namen der Firma ein, bei der diese Person beschäftigt ist.

    Liberty-Haupt-ID

    Geben Sie einen URI ein, der auf eine Online-Instanz des persönlichen Informationsprofils der Kontaktperson verweist.

    E-Mails

    Geben Sie mindestens eine E-Mail-Adresse für die Kontaktperson ein und klicken Sie dann auf "Hinzufügen".

    Telefonnummern

    Geben Sie mindestens eine Telefonnumer für die Kontaktperson ein und klicken Sie dann auf "Hinzufügen".

  5. Klicken Sie auf "Erstellen", um die Konfiguration der Kontaktperson abzuschließen.
  6. Klicken Sie auf der Seite "Entity-Profil" auf "Speichern", um die Konfiguration der Entity abzuschließen.
