STS-Client

Die Schnittstelle des STS-Clients (Security Token Service, Sicherheitstoken-Dienst) ermöglicht das Erstellen und Konfigurieren eines Clients, der mit dem Sicherheitstoken-Dienst von OpenSSO kommuniziert, um ein Sicherheitstoken abzurufen. OpenSSO bietet einen Mechanismus zum Erstellen der folgenden Typen von STS-Client-Agenten:

Discovery Agent

Ermöglicht die Konfiguration eines Discovery Agent-Clients, der mit dem Liberty Discovery-Dienst kommuniziert, um ein Liberty-basiertes Sicherheits-Token abzurufen. Diese Konfiguration definiert die Attribute für die Sicherung von Liberty-Anforderungen vom Discovery-Client bis zum Liberty Discovery-Endpunkt.

Security Token Service-Agent

Ermöglicht die Konfiguration eines Sicherheitstoken-Dienst-Agenten, der mit dem Sicherheitstoken-Dienst von OpenSSO kommuniziert, um webbasierte Sicherheits-Token abzurufen. Diese Konfiguration definiert die Attribute für die Sicherung von Webdienst-Trust-Anforderungen vom STS-Client bis zum STS-Endpunkt.

An der Schnittstelle des STS-Clients können Sie folgende Aktionen ausführen.

  • Discovery Agent-Attribute konfigurieren

  • Attribute des Agenten für den Sicherheitstoken-Dienst konfigurieren
