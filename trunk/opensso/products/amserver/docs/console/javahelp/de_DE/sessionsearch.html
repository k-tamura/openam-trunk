Suchen

Sie können eine bestimmte Benutzersitzung oder einen bestimmten Bereich von Benutzersitzungen anzeigen, indem Sie eine Benutzer-ID eingeben und auf "Suchen" klicken. Platzhalter können verwendet werden. Um die Anzeige der Benutzersitzung zu aktualisieren, klicken Sie auf die Schaltfläche "Aktualisieren".
