Bereiche

Bei einem Zugriffssteuerungsbereich handelt es sich um eine Gruppe von Authentifizierungseigenschaften und Autorisierungsrichtlinien, die mit einem Benutzer oder einer Gruppe von Benutzern verknüpft werden können. Bereichsdaten werden in einer proprietären Informationsstruktur gespeichert, die von OpenSSO in dem von Ihnen angegebenen Datenspeicher erstellt wird. Das OpenSSO-Framework aggregiert die in den einzelnen Bereichen enthaltenen Richtlinien und Eigenschaften innerhalb der OpenSSO-Informationsstruktur. Standardmäßig fügt OpenSSO 8.0 die OpenSSO-Informationsstruktur zusätzlich zu den Benutzerdaten automatisch als speziellen Zweig in Sun Java Enterprise System Directory Server ein. Zugriffssteuerungsbereiche können in Verbindung mit jeder beliebigen LDAPv3-Datenbank verwendet werden.
