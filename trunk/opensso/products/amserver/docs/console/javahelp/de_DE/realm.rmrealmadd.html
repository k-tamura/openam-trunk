So erstellen Sie einen neuen Bereich

  1. Wählen Sie unterhalb der Registerkarte für die Zugriffssteuerung in der Bereichsliste die Option "Neu".
  2. Legen Sie die folgenden allgemeinen Attribute fest:
    Name

    Geben Sie einen Namen für den Bereich ein.

    Übergeordnet

    Legt den Ort des neu erstellten Bereichs fest. Wählen Sie den übergeordneten Bereich aus, in dem Sie den neuen Bereich erstellen möchten.

  3. Legen Sie die folgenden Bereichsattribute fest:
    Bereichsstatus

    Wählen Sie den aktiven oder inaktiven Status. Standardmäßig ist der aktive Status ausgewählt. Diese Einstellung kann jederzeit während des Bestehens des Bereichs über das Symbol "Eigenschaften" geändert werden. Wenn Sie "Inaktiv" auswählen, wird der Benutzerzugriff bei der Anmeldung deaktiviert.

    Bereich-/DNS-Aliasse

    Ermöglicht das Hinzufügen von Aliasnamen für den DNS-Namen des Bereichs. Dieses Attribut akzeptiert nur ?echte“ Domänenaliasnamen (Zufallszeichenfolgen sind nicht zulässig).

  4. Klicken Sie auf "OK", um die Einstellungen zu speichern, oder auf "Abbrechen", um zur vorherigen Seite zurückzukehren.
