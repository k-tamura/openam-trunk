So erstellen Sie eine neue normale Richtlinie
  1. Wählen Sie den Bereich aus, für den Sie eine Richtlinie erstellen möchten.
  2. Klicken Sie in der Liste "Richtlinien" auf "Neue Richtlinie".
  3. Geben Sie einen Namen und eine Beschreibung für die Richtlinie ein.
  4. Wenn die Richtlinie aktiv sein soll, müssen Sie für das Attribut "Aktiv" die Option "Ja" wählen.

    "Ja" ist standardmäßig ausgewählt. Wenn die Richtlinie während der Richtlinienevaluierung nicht angewendet werden soll, heben Sie die Auswahl dieses Attributs auf, um es inaktiv zu machen.

  5. Es ist nicht erforderlich, zu diesem Zeitpunkt alle Felder für normale Richtlinien zu definieren. Sie können die Richtlinie erstellen und später Regeln, Betreffe, Bedingungen und Response-Anbieter hinzufügen. Falls Sie diese Felder jetzt festlegen möchten, lesen Sie die folgenden Themen:

    • So fügen Sie einer Richtlinie eine Regel hinzu.

    • So fügen Sie einer Richtlinie einen Betreff hinzu.

    • LDAP-Filterbedingung

    • So fügen Sie einer Richtlinie einen Response-Anbieter hinzu.

  6. Klicken Sie auf „OK“.
