So erstellen Sie eine neue Gruppe

Erstellen Sie eine Gruppe, wenn die Agenten bestimmte Eigenschaften aus der Gruppe übernehmen sollen. Agenten können Eigenschaften nur aus ihrem Gruppentyp übernehmen. So können Web-Agenten beispielsweise Eigenschaften aus einer Web-Agenten-Gruppe übernehmen.

  1. Klicken Sie auf die Registerkarte "Zugriffssteuerung".
  2. Klicken Sie auf den Namen des Bereichs, zu dem die Gruppe gehört.
  3. Klicken Sie auf die Registerkarte "Agenten".
  4. Wählen Sie gegebenenfalls die Registerkarte für den entsprechenden Agententyp aus.
  5. Klicken Sie im Abschnitt "Gruppe" auf "Neu".

    Geben Sie im Feld "Name" den Namen der neuen Gruppe ein.

  6. Geben Sie bei Web-Agenten und J2EE-Agenten (und nur dort) die URL des OpenSSO-Servers ein.

    Beispiel: http://OpenSSO_EnterpriseHost.example.com:8080/OpenSSO .

  7. Klicken Sie auf "Erstellen".

    Die Konsole erstellt die Agentengruppe und zeigt die Agentenseite mit einer Verknüpfung zu der Gruppe an.

    Wenn Sie weitere Konfigurationen an der Gruppe vornehmen möchten, klicken Sie auf diese Verknüpfung, um die Seite zur Bearbeitung der Agentengruppe anzuzeigen.

    Zur Konfiguration können Sie dieselben Eigenschaften festlegen wie bei einzelnen Agenten, außer dass die Eigenschaften "Gruppe", "Passwort" und "Passwortbestätigung" auf Gruppenebene nicht verfügbar sind.

    Hinweis – Denken Sie außerdem daran, dass einigen Gruppeneigenschaften bereits Variablenwerte zugewiesen wurden, die in den meisten Fällen nicht geändert werden sollten. Im Folgenden finden Sie ein Beispiel für einen solchen Wert:

    @AGENT_PROTO@://@AGENT_HOST@:@AGENT_PORT@/amagent
