So konfigurieren Sie einen Ziel-URL

Die folgenden untergeordneten Attribute können für jeden Ziel-URL festgelegt (oder geändert) werden, der Assertionen über POST erhält.

  1. Klicken Sie auf "Neu", um einen neuen Ziel-URL hinzuzufügen.

    Die Seite "Neuen Post-Paramter der Ziel-URL hinzufügen" wird angezeigt. Außerdem können Sie diese Seite erreichen, indem Sie die Schaltfläche "Bearbeiten" oder "Duplizieren" eines definierten Ziel-URLs wählen.

  2. Typwerte für die Attribute.
    Protokoll

    Wählen Sie entweder http oder https.

    Servername

    Der Name des Servers, auf dem sich der ZIEL-URL befinden, beispielsweise www.sun.com.

    Port

    Dieses Attribut gibt die Portnummer an, beispielsweise58080.

    Pfad

    Dieses Attribut enthält den URI, beispielsweise /amserver/console .

  3. Klicken Sie auf "OK", um die Konfiguration des Ziel-URL abzuschließen.
  4. Klicken Sie auf der Seite "SAML-Profil" auf "Speichern", um die SAML-Konfiguration abzuschließen.
