SAMLv2-Anbieter-Entity

Der SAMLv2-Entity-Anbietertyp basiert auf der OASIS Security Assertion Markup Language (SAML) Version 2-Spezifikation. Diese Entity unterstützt verschiedene Profile (Single Sign-On, Single-Logout usw.), wenn diese mit Remote-SAMLv2-Entities interagieren. Die SAMLv2-Anbieter-Entity ermöglicht es Ihnen, die folgenden Rollen zuzuweisen und zu konfigurieren:

  • Identity-Anbieter

  • Dienstanbieter

  • XACML PEP

  • XACML PDP

  • Attributautorität

  • Attributabfrage

  • Authentifizierungsautorität

  • Gehostete Mitgliedschaft

Verknüpfte Aufgaben und Themen:

  • Entity-Anbieter

  • SAMLv2-Dienstanbieter-Anpassung

  • SAMLv2-Identity-Anbieteranpassung

  • SAMLv2 XACML PDP-Anpassung

  • SAMLv2 XACML PEP-Anpassung

  • SAMLv2-Attributautoritätsanpassung

  • SAMLv2-Attributsanforderungsanpassung

  • SAMLv2-Authentifizierungsautoritätsanpassung

  • SAMLv2-Host-Zuordnungsanpassung
