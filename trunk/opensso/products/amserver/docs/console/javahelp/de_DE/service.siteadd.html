So erstellen Sie eine neue Site-Instanz

  1. Klicken Sie in der Liste der Sites auf "Neu".
  2. Geben Sie den Hostnamen und den Port im Feld ";Server" ein.
  3. Geben Sie den Site-Namen ein.

    Dieser Wert dient zur eindeutigen Identifikation des Servers und ermöglicht die Angabe einer zweiten Einsprungstelle (zusätzlich zur primären URL) für die Site. Dieser wird auch zur Kürzung der Cookie-Länge verwendet, indem die Server-URL der Server-ID zugeordnet wird.

  4. Geben Sie die Primär-URL für die Site-Instanz ein, einschließlich des Site-URI.
  5. Klicken Sie auf Speichern.

    Die erstellte Site wird in der Site-Liste im richtigen Format angezeigt.
