Betreffe

Die Schnittstelle für Betreffe ermöglich die grundlegende Identitätsverwaltung innerhalb eines Bereichs. Jede Identität, die Sie auf dieser Schnittstelle erstellen, kann in der Betreffdefinition einer Richtlinie verwendet werden, die mit dem Typ "OpenSSO-Identitäts-Betreff" erstellt wurde.

Folgende Identitäten können erstellt und geändert werden:

  • Benutzer

  • Gruppe
