SAML-Produktname

Dieses Attribut weist einem SAML-Produkt einen Variablennamen zu. Das Produkt besteht aus größenbeschränkten Daten, die eine Assertion und eine Quellsite identifizieren. Das Attribut wird als Teil einer URL-Abfragezeichenfolge übertragen und über eine Weiterleitung an die Zielsite weitergegeben. Der Standardname lautet SAMLart. Bei Verwendung des standardmäßigen SAMLart könnte die Zeichenfolge zur Abfrageumleitung wie folgt aussehen: http://host:port /deploy-URI/ SamlAwareServlet?TARGET=target-URL /&;SAMLart=artifact123.
