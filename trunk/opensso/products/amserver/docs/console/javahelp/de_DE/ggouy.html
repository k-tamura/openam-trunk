So legen Sie fest, dass ein Agent Eigenschaften von einer Gruppe übernehmen soll
Bevor Sie beginnen

Die Gruppe, aus der der Agent Eigenschaften übernehmen soll, muss zunächst erstellt worden sein.

  1. Klicken Sie auf die Registerkarte "Zugriffssteuerung".
  2. Klicken Sie auf den Namen des Bereichs, zu dem der Agent gehört.
  3. Klicken Sie auf die Registerkarte "Agenten".
  4. Wählen Sie gegebenenfalls die Registerkarte für den entsprechenden Agententyp aus.
  5. Klicken Sie im Abschnitt "Agent" auf den Namen des zu konfigurierenden Agenten.
  6. Wählen Sie bei ausgewählter Registerkarte "Global" für das Attribut "Gruppe" den Namen der Gruppe aus, aus der der Agent Eigenschaften übernehmen soll.
  7. Klicken Sie auf Speichern.

    Oben auf der Seite wird die Schaltfläche "Vererbungseinstellungen" aktiv.

  8. Klicken Sie auf "Vererbungseinstellungen".

    Eine Liste der Vererbungseinstellungen für die Registerkarte "Global" wird in alphabetischer Reihenfolge angezeigt.

  9. Wählen Sie die Eigenschaften aus, die der Agent aus der Gruppe übernehmen soll.

    Oben auf der Seite wird die Schaltfläche "Vererbungseinstellungen" aktiv.

  10. Klicken Sie auf Speichern.
Nächste Schritte

Diese Aufgabe beschreibt nur, wie die Vererbungseinstellungen für Eigenschaften geändert werden können, die auf der Registerkarte "Global" aufgelistet sind. Für die Vererbungseinstellungen, die auf anderen Registerkarten, wie beispielsweise "Anwendung", aufgelistet sind, klicken Sie auf die gewünschte Registerkarte und bearbeiten Sie die Vererbungseinstellungen auf dieselbe Weise wie in den vorhergehenden Schritten beschrieben.
