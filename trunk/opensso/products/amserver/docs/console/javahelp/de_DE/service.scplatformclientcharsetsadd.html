So erstellen Sie einen neuen Zeichensatz

  1. Klicken Sie in der Liste der Client-Zeichensätze auf "Neu".
  2. Geben Sie einen Wert für den Client-Typ ein.
  3. Geben Sie einen Wert für den Zeichensatz ein. Informationen zu den verfügbaren Zeichensätzen finden Sie unter Unterstützte Sprach- bzw. Ländereinstellungen.
  4. Klicken Sie auf „OK“.
  5. Klicken Sie auf der Hauptseite des Plattformdiensts auf ";Speichern".
