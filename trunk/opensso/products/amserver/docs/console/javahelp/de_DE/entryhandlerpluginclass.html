Plugin-Klasse für Eintrags-Handler:

Der Wert dieses Attributs ist der Name und der Pfad der Plugin-Klasse, die die Schnittstelle DiscoEntryHandler implementiert. Diese Schnittstelle wird für das Festlegen oder Abrufen der Discovery-Einträge eines Principals verwendet. Eine Standardimplementierung wird bereitgestellt. To handle discovery entries differently, implement the com.sun.identity.liberty.ws.disco.plugins.DiscoEntryHandler interface and set the implementing class as the value for this attribute.
