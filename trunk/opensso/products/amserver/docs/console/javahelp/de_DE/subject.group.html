Gruppe

Als Gruppe wird eine Ansammlung von Benutzern bezeichnet, die eine Zuständigkeit, eine Funktion oder ein Interesse gemeinsam haben. Normalerweise sind mit dieser Gruppierung keine Berechtigungen verbunden. Gruppen können auf zwei Ebenen vorhanden sein, und zwar in einem Bereich und innerhalb anderer verwalteter Gruppen.

Weitere Informationen finden Sie unter So erstellen oder ändern Sie eine Gruppe
