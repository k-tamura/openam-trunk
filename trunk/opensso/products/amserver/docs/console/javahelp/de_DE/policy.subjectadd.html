So fügen Sie einer Richtlinie einen Betreff hinzu.
  1. Wenn Sie die Richtlinie bereits erstellt haben, klicken Sie auf den Namen der Richtlinie, für die Sie den Betreff hinzufügen möchten. Wenn die Richtlinie noch nicht erstellt wurde, ziehen Sie So erstellen Sie eine neue normale Richtlinie zurate.
  2. Klicken Sie unter der Liste "Betreff" auf "Neu".
  3. Wählen Sie einen der Standard-Betrefftypen aus. Eine Beschreibung der Betrefftypen finden Sie unter Betreffe
  4. Klicken Sie auf "Weiter".
  5. Geben Sie einen Namen für den Betreff ein.
  6. Aktivieren bzw. deaktivieren Sie das Feld "Exklusiv".

    Falls dieses Feld nicht aktiviert ist (Standard), wird die Richtlinie auf die Identity angewendet, die Mitglied des Betreffs ist. Falls dieses Feld aktiviert ist, wird die Richtlinie auf die Identiät angewendet, die kein Mitglied des Betreffs ist.

    Falls in der Richtlinie mehrere Betreffe vorhanden sind, wird die Richtlinie auf die Identität angewendet, wenn mindestens einer der Betreffe impliziert, dass die Richtlinie auf die angegebene Identity zutrifft.

  7. Führen Sie eine Suche durch, um die Identitäten anzuzeigen, die Sie dem Betreff hinzufügen möchten. Dieser Schritt gilt nicht für die Betreffe "Authentifizierte Benutzer" und "Webdienste-Clients".

    Das Standard-Suchmuster (*) zeigt alle qualifizierten Einträge an.

  8. Wählen Sie die einzelnen Identitäten, die Sie dem Betreff hinzufügen möchten, und klicken Sie auf "Alle hinzufügen", um alle Identitäten gleichzeitig hinzuzufügen. Klicken Sie auf "Hinzufügen", um die Identitäten der Liste der ausgewählten Identitäten hinzuzufügen. Dieser Schritt trifft nicht für den Betreff "Authentifizierte Benutzer" zu.
  9. Klicken Sie auf "Fertig stellen".
  10. Zum Entfernen eines Betreffs aus einer Richtlinie wählen Sie den Betreff aus und klicken Sie auf "Löschen". Zum Bearbeiten einer Betreffdefinition klicken Sie auf den Namen des Betreffs.
