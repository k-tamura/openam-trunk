So erstellen Sie eine neue Serverinstanz

In der Serverliste wird automatisch ein Eintrag für jeden Server erstellt, wenn der OpenSSO Configurator zur Serverkonfiguration ausgeführt wird. Unter normalen Bedingungen sollten diese Schritte nicht erforderlich sein.

  1. Melden Sie sich als Top-Level-Administrator bei der OpenSSO-Konsole an.
  2. Klicken Sie auf die Registerkarte "Konfiguration" und dann auf "Sites und Server".
  3. Klicken Sie in der Liste der Server auf "Neu".
  4. Geben Sie den FQDN des hinzuzufügenden Servers ein, und klicken Sie auf "OK".

    Der FDNQ sollte folgendes Format aufweisen http (s)://host.domaene:anschluss /uri.

  5. Die neu erstellte Serverinstanz wird in der Liste angezeigt.
  6. Zur Bearbeitung des Servers klicken Sie auf seinen Namen Die Konfigurationsattribute für den Server stehen Ihnen zur Anpassung zur Verfügung.

    Die Standardservereinstellungen sind die Menge der Standardwerte für Serverinstanzen. Jede Serverinstanz benötigt eine Mindestmenge von Eigenschaftswerten, und die meisten Eigenschaftswerte können, je nach Bereitstellung, für alle Serverinstanzen gleich sein. Mit dieser Einstellung können Sie alle grundlegenden Eigenschaften an einer zentralen Stelle angeben und müssen sie nicht für jede weitere Serverinstanz ändern.

    Diese Standardwerte können überschrieben werden. Die geschieht durch Klicken auf die Schaltfläche "Vererbungseinstellungen", die sich oben auf der Profilseite der Serverinstanz befindet. Nach dem Klicken auf diese Schaltfläche zeigt die Konsole eine Seite an, auf der Sie an- und abwählen können, welche Werte vererbt bzw. überschrieben werden sollen.
