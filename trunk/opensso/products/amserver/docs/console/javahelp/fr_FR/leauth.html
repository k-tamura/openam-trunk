Condition d'authentification LE

La stratégie s'applique si le niveau d'authentification de l'utilisateur est inférieur ou égal à celui défini dans la condition.

La condition de niveau d'authentification peut être utilisée pour spécifier des niveaux différents des niveaux du module d'authentification enregistré pour ce domaine. Cette possibilité est utile lorsqu'une stratégie doit s'appliquer à un utilisateur authentifié dans un autre domaine.
