Temps (jour, date, heure et fuseau horaire)

Définit la condition en fonction de contraintes temporelles. Ces champs sont les suivants :

Date De/À

Spécifie la plage de dates.

Heure

Spécifie la plage horaire dans la journée.

Jour

Spécifie une plage de jours.

Fuseau horaire

Spécifie un fuseau horaire, qui peut être standard ou personnalisé. Les fuseaux horaires personnalisés doivent obligatoirement correspondre à un ID de fuseau horaire reconnu par Java (par exemple, PST). Si aucune valeur n'est spécifiée, la valeur par défaut est le fuseau horaire défini dans la JVM OpenSSO.
