Domaines

Un domaine de contrôle d'accès est un groupe de propriétés d'authentification et de stratégies d'autorisation que vous pouvez associer à un utilisateur ou un groupe d'utilisateurs. Les données de domaine sont stockées dans une arborescence d'informations créée par OpenSSO dans le magasin de données que vous indiquez. La structure OpenSSO regroupe des règles et des propriétés contenues dans chaque domaine de l'arborescence d'informations OpenSSO. Par défaut, OpenSSO 8.0 insère automatiquement l'arborescence d'informations OpenSSO sous forme de branche spéciale dans Sun Java Enterprise System Directory Server, séparément des données utilisateur. Vous pouvez utiliser les domaines de contrôle d'accès en même temps que n'importe quelle base de données LDAPv3.
