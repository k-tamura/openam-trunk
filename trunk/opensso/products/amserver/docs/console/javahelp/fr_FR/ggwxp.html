Entité de fournisseur SAMLv2

Le type de fournisseur d'entités SAMLv2 est basé sur la spécification SAML (Security Assertion Markup Language) version 2 d'OASIS. Cette entité prend en charge plusieurs profils (connexion unique, déconnexion unique, etc.) lorsqu'elle interagit avec des entités SAMLv2 distantes. Le fournisseur d'entités SAMLv2 vous permet d'attribuer et de configurer les rôles suivants :

  • Fournisseur d'identités

  • Fournisseur de services

  • XACML PEP

  • XACML PDP

  • Autorité d'attribut

  • Requête d'attributs

  • Autorité d'authentification

  • Affiliation hébergée
