Configuration d'un gestionnaire de demandes

L'interface RequestHandler doit être implémentée côté serveur par chaque service SAMLv2 utilisant le protocole Service de liaison SOAP. L'attribut Liste du gestionnaire de demandes stocke les informations sur les classes qui implémentent le gestionnaire de demandes. Les entrées de la liste du gestionnaire de demandes contiennent des paires clé/valeur.

  1. Cliquez sur Nouveau pour afficher les attributs Nouveau gestionnaire de demandes ou cliquez sur une valeur de clé configurée pour modifier les attributs existants.
  2. Fournissez les valeurs pour les attributs en vous basant sur les informations suivantes :
    key

    Le paramètre Key est le méta-alias du point de décision de stratégie.

    class

    Le paramètre Class est le nom de la classe Java qui implémente l'interface RequestHandler capable de traiter les demandes XACML.

  3. Cliquez sur OK pour terminer la configuration du gestionnaire de demandes.
  4. Cliquez sur Enregistrer dans la page Liaison SOAP SQMLv2 pour terminer la configuration du service.
