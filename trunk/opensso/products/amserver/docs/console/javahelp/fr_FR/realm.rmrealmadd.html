Création d'un domaine

  1. Sélectionnez Nouveau dans la liste Domaines située dans l'onglet Contrôle d'accès.
  2. Définissez les attributs généraux suivants :
    Nom

    Entrez le nom du domaine.

    Parent

    Définit l'emplacement du domaine que vous créez. Sélectionnez le domaine parent sous lequel sera situé le nouveau domaine.

  3. Définissez les attributs de domaine suivants :
    État du domaine

    Choisissez l'état actif ou inactif. La valeur par défaut est actif. Ce paramètre peut être modifié à tout moment durant la vie du domaine à l'aide de l'icône Propriétés. Si la valeur inactif est sélectionnée, l'accès utilisateur est désactivé à la connexion.

    Domaine/Alias DNS

    Vous permet d'ajouter des alias pour le nom DNS du domaine. Cet attribut n'accepte que les alias de domaine réels (les chaînes aléatoires ne sont pas autorisées).

  4. Cliquez sur OK pour enregistrer ou sur Annuler pour retourner à la page précédente.
