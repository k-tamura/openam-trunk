Objets

L'interface Objets permet d'assurer une gestion de base des identités dans un domaine. Toute identité créée dans l'interface Objets peut être utilisée dans la définition d'objet pour une stratégie créée avec le type Objet d'identité OpenSSO.

Les identités que vous pouvez créer et modifier sont les suivantes :

  • Utilisateur

  • Groupe
