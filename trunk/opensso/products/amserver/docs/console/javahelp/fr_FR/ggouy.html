Pour permettre à un agent d'hériter des propriétés d'un groupe
Avant de commencer

Le groupe dont vous souhaitez qu'un agent hérite des propriétés doit être créé en premier.

  1. Cliquez sur l'onglet Contrôle d'accès.
  2. Cliquez sur le nom du domaine auquel l'agent appartient.
  3. Cliquez sur l'onglet Agents.
  4. Si nécessaire, sélectionnez l'onglet correspondant au type d'agent approprié.
  5. Dans la section Agent, cliquez sur le nom de l'agent que vous souhaitez configurer.
  6. L'onglet Global étant sélectionné, pour l'attribut Groupe, sélectionnez le nom du groupe dont l'agent doit hériter des propriétés.
  7. Cliquez sur Enregistrer.

    En haut de la page, le bouton Paramètres d'héritage devient actif.

  8. Cliquez sur Paramètres d'héritage.

    Dans l'onglet Global, les paramètres d'héritage apparaissent dans l'ordre alphabétique.

  9. Sélectionnez les propriétés que l'agent doit hériter du groupe.

    En haut de la page, le bouton Paramètres d'héritage devient actif.

  10. Cliquez sur Enregistrer.
Étapes suivantes

Cette tâche décrit comment modifier les paramètres d'héritage des propriétés répertoriées dans l'onglet Global. Pour les paramètres d'héritage des propriétés répertoriées dans d'autres onglets (par exemple, Application), cliquez sur l'onglet souhaité et modifiez les paramètres d'héritage en suivant la procédure décrite dans les étapes précédentes.
