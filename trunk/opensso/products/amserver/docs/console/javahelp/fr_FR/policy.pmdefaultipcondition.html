Adresse IP/Nom DNS

Définit la condition en fonction d'une plage d'adresses IP. Les champs que vous pouvez définir sont les suivants :

Adresse IP De/À

Indique la plage d'adresses IP.

Nom DNS

Spécifie le nom DNS. Ce champ peut contenir un nom d'hôte complet ou une chaîne ayant l'un des formats suivants :

nomdomaine

*.nomdomaine
