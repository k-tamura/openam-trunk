Ajout d'un service à un domaine

  1. Cliquez sur le nom du domaine auquel vous souhaitez ajouter un nouveau service.
  2. Sélectionnez l'onglet Services.
  3. Cliquez sur Ajouter dans la liste Services.
  4. Sélectionnez le service à ajouter au domaine.
  5. Cliquez sur Suivant.
  6. Configurez le service en définissant les attributs de domaine. Consultez la définition des attributs pour les services suivants :

    • Administration

    • Configuration de règles

    • Service de découverte

    • Paramètres de globalisation

    • Réinitialisation du mot de passe

    • Session

    • Utilisateur

  7. Cliquez sur Terminer.
  8. Pour modifier les propriétés d'un service, cliquez sur son nom dans la liste des services.
