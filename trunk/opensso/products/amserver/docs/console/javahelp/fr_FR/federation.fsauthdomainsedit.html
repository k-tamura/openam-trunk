Modification du profil d'un cercle de confiance

Suivez cette procédure pour modifier les attributs globaux d'un cercle de confiance ou pour lui ajouter des fournisseurs. Le point de départ est Cercle de confiance dans l'interface Fédération.

  1. Cliquez sur le nom d'un cercle de confiance pour modifier son profil ou lui ajouter des fournisseurs.

    La page Modifier le cercle de confiance apparaît.

  2. Tapez les nouvelles valeurs ou modifiez celles affichées dans les attributs généraux du cercle de confiance :
    Nom

    La valeur statique de cet attribut est le nom fourni lors de la création du cercle de confiance.

    Description

    Cet attribut est une description du cercle de confiance. Si nécessaire, vous pouvez modifier la description déjà saisie.

    URL du service écriture IDFF

    Cet attribut indique l'emplacement du service qui écrit le cookie du domaine commun. L'URL a le format http://common-domain-host:port/common/idffwriter.

    URL du service lecture IDFF

    Cet attribut indique l'emplacement du service qui lit le cookie du domaine commun. L'URL a le format http://common-domain-host:port/common/idffreader.

    URL du écriture SAML2

    Spécifie l'emplacement du service écriture SAML2 qui écrit le cookie dans le domaine commun. L'URL a le format http://common-domain-host:port/common/saml2writer.

    URL du lecture SAML2

    Spécifie l'emplacement du service écriture SAML2 qui écrit le cookie dans le domaine commun. L'URL a le format http://common-domain-host:port/common/saml2reader.

    Statut

    Le statut par défaut est Actif. Si vous sélectionnez Inactif, vous désactivez la communication dans le cercle de confiance.

  3. Choisissez un ou plusieurs fournisseurs disponibles et cliquez sur la flèche Ajouter pour les sélectionner.

    La liste fournie contient les noms des entités créées et renseignées avec des fournisseurs. Pour plus d'informations, reportez-vous à la rubrique Ajout de fournisseurs à un cercle de confiance.

  4. Cliquez sur Enregistrer pour terminer l'opération.
