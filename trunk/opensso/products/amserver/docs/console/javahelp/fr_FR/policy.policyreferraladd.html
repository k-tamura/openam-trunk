Création d'une stratégie d'orientation
  1. Choisissez le domaine pour lequel vous souhaitez créer la stratégie.
  2. Cliquez sur Nouvelle orientation dans la liste Stratégies.
  3. Ajoutez un nom et une description pour la stratégie.
  4. Si vous souhaitez que la stratégie soit active, sélectionnez Oui pour l'attribut Active.
  5. Il n'est pas nécessaire de définir tous les champs des stratégies normales à ce stade. Vous pouvez créer la stratégie, puis ajouter les règles et les orientations ultérieurement. Si vous souhaitez définir les champs maintenant, reportez-vous aux rubriques suivantes :

    • Ajout d'une règle à une stratégie

    • Ajout d'orientations à une stratégie

  6. Cliquez sur Créer.
