Signer la réponse SAML

Cet attribut indique si toutes les réponses SAML seront signées numériquement (XML DSIG) avant d'être distribuées. Cocher cette case active cette fonction.

Remarque – Toutes les réponses SAML utilisées par le profil POST du navigateur Web seront signées numériquement, que cette option soit activée ou non.
