Propriétés générales

Les propriétés générales contiennent des services qui vous permettent de définir la fonctionnalité de réinitialisation de mot de passe et la configuration de stratégies pour OpenSSO. Les services qui peuvent être configurés sont les suivants :

  • Configuration des fédérations communes

  • Configuration du service Liberty ID-FF

  • Service de sécurité de client de service Web Liberty ID

  • Service d'interaction Liberty

  • Protocole multi-fédération

  • Réinitialisation du mot de passe

  • Configuration de règles

  • Configuration du service SAMLv2

  • Liaison SOAP SAMLv2

  • Service de jeton de sécurité

  • Session

  • Ajout d'une configuration secondaire

  • Utilisateur
