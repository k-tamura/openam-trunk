D'ordre général

Les attributs d'ordre général définissent les données de configuration de base pour la gestion centralisée de votre serveur.

Attributs de site

L'attribut de site est :

Site parent

Cet attribut mappe le nom de site (ID de site) d'équilibreur de charge sur le serveur OpenSSO. Pour pouvoir ajouter le site, vous devez d'abord le créer.

Attributs système

Les attributs système répertorient les informations d'emplacement relatives à l'instance de serveur :

Répertoire d'installation de base

Spécifie le répertoire de base contenant les données du produit.

Paramètre régional par défaut

La valeur du paramètre régional correspond au sous-type de langue par défaut avec lequel OpenSSO a été installé. La valeur par défaut est en_us.

URL de notification

Emplacement du point d'arrivée de service de notification. Cette valeur est définie pendant l'installation.

Validation XML

La valeur par défaut est no. Détermine si la validation est requise lors de l'analyse de documents XML avec la classe XMLUtils d'OpenSSO. Cette propriété n'a d'effet que lorsque la valeur de l'attribut Niveau de débogage est définie sur warning ou message. Les valeurs autorisées sont yes et no. La validation de documents XML n'est activée que si cette propriété a pour valeur yes et si l'attribut Niveau de débogage a pour valeur avertissement ou message.

Attributs de débogage

Les attributs de débogage répertorient les informations de vérification d'erreur de base :

Niveau de débogage

Spécifie le niveau de débogage. La valeur par défaut est error. Les valeurs possibles sont les suivantes :

  • off — Aucun fichier de débogage n'est créé.

  • error — Seuls les messages d'erreur sont consignés.

  • warning — Seuls les messages d'avertissement sont consignés.

  • message — Les messages d'erreur, d'avertissement et d'information sont consignés.

Répertoire de débogage

Spécifie le répertoire de sortie dans lequel sont créés les fichiers de débogage. Cette valeur est définie pendant l'installation. Exemple : base OpenSSO /URI_débogage/debug.

Serveur de courrier électronique

Les attributs de serveur de courrier électronique répertorient le nom d'hôte et le port du serveur de courrier électronique :

Nom d'hôte du serveur de courrier électronique

La valeur par défaut est localhost. Spécifie l'hôte du serveur de courrier électronique.

Numéro de port du serveur de courrier électronique

La valeur par défaut est 25. Spécifie le port du serveur de courrier électronique.
