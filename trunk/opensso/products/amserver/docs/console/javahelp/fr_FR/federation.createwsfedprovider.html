Entité de fournisseur WS-FED

Le type de fournisseur d'entités WS-FED est basé sur le protocole WS-Federation. L'implémentation de ce protocole permet une connexion unique entre OpenSSO et le service de fédération de Microsoft Active Directory. L'entité de fournisseur WS-FED vous permet d'attribuer et de configurer les rôles suivants :

  • Fournisseur d'identités

  • Fournisseur de services

Tâches et rubriques connexes :

  • Création d'un fournisseur d'entités WS-FED

  • Attributs globaux WS-FED

  • Personnalisation d'un fournisseur d'identités WS-FED

  • Personnalisation d'un fournisseur de services WS-FED
