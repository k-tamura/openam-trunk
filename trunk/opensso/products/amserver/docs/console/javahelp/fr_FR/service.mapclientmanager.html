Gestionnaire de clients

Le Gestionnaire de clients est l'interface répertoriant les clients de base, leurs styles et leurs propriétés, et qui vous permet d'ajouter et de configurer des périphériques.

Type de client de base

Les types de client de base sont répertoriés en haut du gestionnaire de clients. Ces types de client contiennent les propriétés par défaut dont peuvent hériter tous les périphériques faisant partie de ce type de client.

Type de client

Profil de style : le Gestionnaire de clients regroupe tous les clients disponibles, y compris le client de base, dans la liste des types de client. Vous pouvez modifier les propriétés de chaque client en cliquant sur le nom du périphérique. Les propriétés sont ensuite affichées dans la fenêtre Éditeur du client. Plate-forme matérielle

Contient les propriétés concernant le matériel du périphérique, par exemple la taille de l'écran, les jeux de caractères pris en charge, etc.

Plate-forme logicielle

Contient les propriétés de l'environnement d'application du périphérique, du système d'exploitation et des logiciels installés.

Caractéristiques du réseau

Contient les propriétés décrivant l'environnement réseau, notamment les services support pris en charge.

BrowserUA

Contient les attributs liés à l'agent utilisateur du navigateur exécuté sur le périphérique.

WapCharacteristics

Contient les propriétés de l'environnement WAP (Wireless Application Protocol) pris en charge par le périphérique.

PushCharacteristicNames

Contient les propriétés de l'environnement WAP pris en charge par le périphérique.

Autres propriétés

Autres propriétés Vous permet de définir d'autres propriétés pour le périphérique.
