URL cibles

Si l'URL cible reçu par l'intermédiaire d'un profil SAML est mentionné comme valeur pour cet attribut, les assertions reçues seront adressées à l'URL cible via une commande HTTP FORM POST.

Remarque – Pour modifier ou dupliquer les attributs d'un profil de partenaire de confiance, cliquez sur le bouton Propriétés du site local et cliquez sur Nouveau dans le tableau URL cible.
