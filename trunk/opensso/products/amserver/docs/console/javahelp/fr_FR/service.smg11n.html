Paramètres de globalisation

Le service Paramètres de globalisation contient des attributs généraux qui vous permettent de configurer OpenSSO pour différents paramètres régionaux et jeux de caractères. Ces attributs sont les suivants :

  • Jeux de caractères compatibles par version des paramètres locaux

  • Désignation des jeux de caractères

  • Format de nom commun généré automatiquement

Jeux de caractères compatibles par version des paramètres locaux

Cet attribut répertorie les jeux de caractères pris en charge pour chaque langue, indiquant le mappage entre la langue et le jeu de caractères. Le format est le suivant :

Pour ajouter un nouveau jeu de caractères pris en charge, cliquez sur Ajouter et définissez les paramètres suivants :

Paramètres locaux

Les nouveaux paramètres locaux que vous souhaitez ajouter. Pour plus d'informations, reportez-vous à Environnements linguistiques pris en charge.

Jeux de caractères supportés

Entrez le jeu de caractères supporté pour la langue spécifiée. Les jeux de caractères sont délimités par un point-virgule. Par exemple, charset=jeu_caractères1;jeu_caractères2;jeu_caractères3;...;jeu_caractèresn

Pour modifier tout jeu de caractères supporté, cliquez sur le nom voulu dans le tableau des jeux de caractères supportés. Cliquez sur OK lorsque vous avez terminé.

Désignation des jeux de caractères

Cet attribut répertorie les noms de jeu de codes (mappés sur des noms IANA) qui seront utilisés pour envoyer la réponse. Ces noms de jeu de codes ne doivent pas nécessairement correspondre à des noms de jeu de codes Java. Actuellement, il existe une table de hachage qui fournit la correspondance entre les jeux de caractères Java et les jeux de caractères IANA.

Pour ajouter une nouvelle désignation de jeu de caractère, cliquez sur le bouton Ajouter, puis définissez les paramètres suivants :

Nom MIME

Nom de mappage IANA. Par exemple, Shift_JIS

Nom Java

Jeu de caractères Java à mapper sur le jeu de caractères IANA.

Pour modifier toute désignation de jeu de caractères, cliquez sur le nom voulu dans le tableau. Cliquez sur OK lorsque vous avez terminé.

Format de nom commun généré automatiquement

Cette option d'affichage vous permet de définir la manière dont un nom est automatiquement généré pour adapter les formats de nom à différents paramètres locaux et jeux de caractères. La syntaxe par défaut est la suivante (notez que les virgules et/ou les espaces inclus dans la définition s'afficheront dans le format du nom) :

en_us = {nomdonné} {initiales} {sn}

Par exemple, si vous voulez afficher un nouveau format de nom pour un utilisateur (Utilisateur Un) avec un uid (11111) pour le jeu de caractères chinois, définissez :

zh = {sn}{nomdonné}({uid})

L'affichage est le suivant :

Utilisateur 11111
