Authentification

OpenSSO est installé avec un jeu de types de modules d'authentification par défaut. Une instance de module d'authentification est un plug-in qui collecte les informations de l'utilisateur, par exemple l'ID utilisateur et le mot de passe, compare ces informations avec les entrées de la base de données et autorise ou refuse en conséquence l'accès à l'utilisateur. Plusieurs instances du même type peuvent être créées et configurées séparément.
