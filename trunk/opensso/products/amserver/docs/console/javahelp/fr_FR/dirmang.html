Gestion d'annuaire

L'onglet Gestion d'annuaire ne s'affiche que quand OpenSSO est installé en mode hérité. Cette fonctionnalité de gestion d'annuaire fournit une solution de gestion des identités pour les déploiements OpenSSO compatibles avec Sun Java System Directory Server.

Pour plus d'informations sur les options d'installation en mode hérité, reportez-vous au Guide d'installation et de configuration de Sun OpenSSO.
