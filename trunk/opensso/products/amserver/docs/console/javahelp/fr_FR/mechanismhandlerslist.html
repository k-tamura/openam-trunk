Liste des gestionnaires de mécanismes

L'attribut Liste des gestionnaires de mécanismes stocke des informations concernant les mécanismes SASL pris en charge par le Service Web d'authentification. It displays entries that contain key/value pairs separated by a pipe as in: key=PLAIN|class=com.sun.identity.liberty.ws.authnsvc.mechanism.PlainMechanismHandler . Ces paramètres sont expliqués dans le tableau suivant.

Paramètre

Valeur

key

Le paramètre obligatoire key définit le mécanisme SASL pris en charge par le Service Web d'authentification.

class

Le paramètre obligatoire class spécifie le nom de la classe d'implémentation du mécanisme SASL. La couche Service Web d'authentification fournit une interface de gestion à implémenter pour que chaque mécanisme SASL traite le message demandé et renvoie une réponse.

Pour ajouter un nouveau gestionnaire de mécanismes, cliquez sur Ajouter et reportez-vous à Configuration d'un nouveau gestionnaire de mécanismes.
