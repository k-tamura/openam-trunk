Agent de stratégie 2.2

Federated Access Manager est compatible de manière ascendante avec Policy Agent 2.2. Policy Agent 2.2 doit majoritairement être configuré localement à partir du conteneur de déploiement sur lequel il est installé. Par conséquent, à partir de la console de Federated Access Manager , un nombre d'options Policy Agent 2.2 très limité peut être configuré.

Mot de passe

Le mot de passe a été défini lors de la création du profil d'agent. Cependant, vous pourrez changer de mot de passe à tout moment.

Confirmation de mot de passe

Le mot de passe a été confirmé lors de la création du profil d'agent. Si vous modifiez le mot de passe, vous devez confirmer votre modification.

Statut

L'option Actif est sélectionnée lors de la création de l'agent. Ne choisissez Inactif que si vous voulez supprimer la protection que l'agent fournit.

Description

Description de l'agent, que vous pouvez ajoutez si vous le souhaitez.
