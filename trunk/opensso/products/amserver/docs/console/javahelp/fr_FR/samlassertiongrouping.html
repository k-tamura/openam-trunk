Assertion

Voici les attributs du groupe Assertions. (La durée de validité totale d'une assertion est définie par les valeurs spécifiées dans les deux attributs.)

Délai d'assertion

Cet attribut spécifie le nombre de secondes avant expiration du délai de validité d'une assertion. La valeur par défaut est 420.

Facteur d'inclinaison de l'assertion pour la durée notBefore :

Cet attribut est utilisé pour calculer la durée notBefore d'une assertion. Par exemple, si IssueInstant est égal à 2002-09024T21:39:49Z, et Assertion Skew Factor notBefore Time est égal à 300 secondes (180 est la valeur par défaut), l'attribut notBefore de l'élément de conditions de l'assertion serait 2002-09-24T21:34:49Z.
