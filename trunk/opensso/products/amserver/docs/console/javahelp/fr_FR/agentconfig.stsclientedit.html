Client STS

L'interface du client du service de jeton de sécurité (STS) vous permet de créer et de configurer un client qui communique avec le service de jeton de sécurité d'OpenSSO pour obtenir un jeton de sécurité. OpenSSO fournit le mécanisme permettant de créer les types suivants d'agents clients de STS :

Agent de découverte

Permet de configurer un client d'agent de découverte qui communique avec le service de découverte Liberty pour obtenir un jeton de sécurité Liberty. Cette configuration définit les attributs permettant de sécuriser les demandes Liberty envoyées par le client de découverte à l'extrémité de découverte Liberty.

Agent du service de jeton de sécurité

Permet de configurer un agent du service de jeton de sécurité qui communique avec le service de jeton de sécurité d'OpenSSO pour obtenir des jetons de sécurité basés sur des services Web. Cette configuration définit les attributs permettant de sécuriser les demandes d'approbation de services Web envoyées par le client STS à l'extrémité STS.

Dans l'interface du client STS, vous pouvez effectuer les opérations suivantes :

  • configurer les Attributs de l'agent de découverte ;

  • Configurer les Attributs de l'agent du service de jeton de sécurité
