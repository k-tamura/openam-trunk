Personnalisation d'un PEP XACML SAMLv2

Le PEP XACML contient les attributs personnalisables suivants :

  • Énumération des protocoles pris en charge

  • Alias de la clé de signature

  • Alias de la clé de chiffrement

  • Autorisation standard

  • Réponse de décision d'autorisation signée

  • Assertion chiffrée

Énumération des protocoles pris en charge

Affiche la version du PEP XACML prise en charge par ce fournisseur.

Alias de la clé de signature

Définit l'alias de clé utilisé pour signer les demandes et les réponses.

Alias de la clé de chiffrement

Définit l'alias de clé pour le chiffrage XACML.

Autorisation standard

L'authentification standard peut être activée pour protéger les extrémités SOAP. Tout fournisseur qui accède à ces extrémités doit avoir le nom d'utilisateur et le mot de passe définis dans les propriétés suivantes : User Name et Password.

Réponse de décision d'autorisation signée

Lorsqu'il est activé, cet attribut demande la signature de toutes les réponses pour la décision d'autorisation XACML.

Assertion chiffrée

Lorsqu'il est activé, cet attribut demande le chiffrement de toutes les assertions.
