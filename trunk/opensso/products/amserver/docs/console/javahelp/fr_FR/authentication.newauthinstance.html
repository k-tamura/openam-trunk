Création d'une instance de module d'authentification

  1. Cliquez sur le nom du domaine auquel vous souhaitez ajouter une nouvelle instance de module d'authentification.
  2. Sélectionnez l'onglet Authentification.
  3. Cliquez sur Nouveau dans la liste Instances de module.
  4. Entrez le nom de l'instance de module d'authentification. Ce nom doit être unique.
  5. Sélectionnez le type de module d'authentification du domaine.
  6. Cliquez sur OK.
  7. Cliquez sur le nom de la nouvelle instance de module et modifiez les propriétés de ce module. Reportez-vous à la rubrique Attributs du module d'authentification pour consulter la définition des propriétés de chaque type de module.
  8. Répétez ces étapes si vous voulez ajouter plusieurs instances de module.
