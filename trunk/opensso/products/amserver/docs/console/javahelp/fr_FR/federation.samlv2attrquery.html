Personnalisation d'une requête d'attributs SAMLv2

La requête d'attributs SAMLv2 contient les attributs personnalisables suivants :

  • Format NameID

  • Alias cert

Format NameID

Définit les formats d'identificateur de nom pris en charge par le fournisseur de requête d'attributs. Les identificateurs de nom permettent aux fournisseurs d'échanger des informations sur un utilisateur. Les interactions de connexion unique prennent en charge trois types d'identificateurs :

  • Un identificateur X509SubjectName définit le nom d'objet du type de chiffrement X509.

  • Un identificateur persistant est enregistré dans une entrée du magasin de données de l'utilisateur. Il reprend la valeur de deux attributs.

  • Un identificateur transitoire est temporaire et aucune donnée n'est consignée dans le magasin de données persistant de l'utilisateur.

Alias cert

Cet attribut définit les éléments de l'alias du certificat pour le fournisseur. signing spécifie l'alias du certificat du fournisseur, utilisé pour trouver le certificat de signature correct dans le keystore. Encryption spécifie l'alias du certificat du fournisseur, utilisé pour trouver le certificat de chiffrement correct dans le keystore.
