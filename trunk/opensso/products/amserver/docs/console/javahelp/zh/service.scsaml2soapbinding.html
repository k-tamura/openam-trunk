SAMLv2 SOAP 绑定

SAMLv2 SOAP 绑定服务在 OpenSSO 客户机和 OpenSSO 服务器之间提供基于 SOAP 的 SAMLv2 请求和响应消息交换。收到的请求将委托给请求处理程序进行进一步处理。请求处理程序和元数据别名的关键字位于 SOAP 绑定服务 URL 中。元数据别名和 RequestHandler 的映射保存在 SAMLv2 SOAP 绑定服务中，可从 OpenSSO 配置存储库读取。

请求处理程序列表

RequestHandlerList 是关键字/值对的条目列表，包含元数据别名与 RequestHandler 实现的映射。如果将 OpenSSO 8.0 服务器配置成策略决策点 (Policy Decision Point, PDP)，则必须设置该属性。

关键字是策略决策点元数据别名，类是 Java 类名，该类是处理 XACML 请求的 RequestHandler 接口的实现。

例如，如果 XACML 策略决策点的元数据别名是 /pdp，接口实现是 com.sun.identity.xacml.plugins.XACMLAuthzDecisionQueryHandler，则应将关键字设置为 /pdp，将类设置为 com.sun.identity.xacml.plugins.XACMLAuthzDecisionQueryHandler。
