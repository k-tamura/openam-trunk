向提供者实体添加服务提供者

要将服务提供者添加到提供者实体，首先应定义要添加的服务提供者的类型。

  • 托管提供者的托管服务器与 Federated Access Manager 所在的服务器相同。

  • 远程提供者的托管服务器不同于 Federated Access Manager 所在的服务器。

以下过程对此初始步骤进行了说明。起点是联合界面下的“实体”。

  1. 选择已配置提供者实体的名称以修改其配置文件。

    随即显示实体的配置文件页面。

  2. 从“查看”菜单中选择“服务提供者”。
  3. 选择要添加的提供者类型：

    • 如果提供者的托管服务器与 Federated Access Manager 所在的服务器相同，请选择“新建托管提供者”。

    • 如果提供者的托管服务器不同于 Federated Access Manager 所在的服务器，请选择“新建远程提供者”。

    随即显示相应的服务提供者配置文件页面。

  4. 根据您在上一步骤所做的选择，选择以下某一过程：
    • 向提供者实体添加新的托管服务提供者
    • 向提供者实体添加新的远程服务提供者
