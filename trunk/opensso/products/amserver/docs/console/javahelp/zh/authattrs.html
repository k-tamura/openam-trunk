验证

安装 Federated Access Manager 时会随附一组默认的验证模块类型。验证模块实例是一个插件，用来收集类似用户 ID 和密码的用户信息并根据数据库中的条目检查信息，最终允许或拒绝用户访问。能分别创建和配置多个同类型的实例。

本节提供了配置默认验证模块类型的属性说明。

  • 验证配置

  • 证书

  • 核心

  • 数据存储库
