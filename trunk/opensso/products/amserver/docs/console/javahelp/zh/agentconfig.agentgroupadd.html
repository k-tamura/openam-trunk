创建新组

如果要使代理继承组的特定属性，则创建一个组。代理仅能继承它们的组类型的属性。例如，Web 代理能够继承 Web 代理组的属性。

  1. 单击“访问控制”选项卡。
  2. 单击组所属的领域的名称。
  3. 单击“代理”选项卡。
  4. 如有必要，选择适当代理类型的选项卡。
  5. 在“组”部分中单击“新建”。

    在“名称”字段中输入新组的名称。

  6. 仅为 Web 代理和 J2EE 代理输入 OpenSSO 服务器 URL。

    例如，http://OpenSSO_EnterpriseHost.example.com:8080/OpenSSO。

  7. 单击“创建”。

    控制台创建代理组，并再次显示含有到组的链接的代理页面。

    要对组进行其他配置，单击该链接显示“编辑”代理组页面。

    您可设置用于配置组的属性与用于个体代理的属性相同，仅有的不同之处是不能在该组级别使用“组”、“密码”、“密码确认”属性。

    注 – 此外，要注意到一些组属性已经指定变量值，而且在大多数情况下都不应更改。此类值的一个示例如下：

    @AGENT_PROTO@://@AGENT_HOST@:@AGENT_PORT@/amagent
