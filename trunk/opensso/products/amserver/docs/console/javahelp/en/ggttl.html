To Enable an Agent to Inherit Properties From a Group
Before You Begin

The group from which you want an agent to inherit attributes must be created first.

  1. In the Agent section, click the name of the agent you want to configure.
  2. With the Global tab selected, for the attribute labeled Group, select the name of the group from which you want the agent to inherit attributes.

    At the top of the page, the Inheritance Settings button becomes active.

  3. Click Inheritance Settings.
  4. Select the Attributes (properties) that you want the agent to inherit from the group.
  5. Click Save.
