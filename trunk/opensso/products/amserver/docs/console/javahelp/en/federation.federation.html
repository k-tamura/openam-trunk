Federation

The Federation interface is the OpenSSO implementation of the Liberty Alliance Project Identity Federation Framework (Liberty ID-FF), SAMLv2 and WS-FED specifications. These define a set of protocols, bindings and profiles that provide a solution for identity federation, cross-domain authentication and session management. The Federation interface in the OpenSSO Console provides a means for configuring and maintaining the following:

  • Circle of Trust

  • Entity Providers

  • SAML1.x
