To Configure an Attribute Mapping

A mapping is an attribute that defines a holder for a piece of identity data. Following is the procedure for adding new attributes to the Liberty Personal Profile Service. The starting point is the Liberty Personal Profile Service configuration screen within the Web Services module.

  1. Click New to display the new Attribute Mapping attributes or click on the name of a configured mapping to modify it.
  2. Provide values for the container attributes based on the following information:
    Name Prefix

    Enter the name of the Liberty Personal Profile Service identity attribute to be mapped.

    LDAP Attribute

    Enter the name for the OpenSSO LDAP attribute to which the Name Prefix maps.

  3. Click OK to complete the Mapping configuration.
  4. Click Save on the Liberty Personal Profile Service page to complete the service configuration.
