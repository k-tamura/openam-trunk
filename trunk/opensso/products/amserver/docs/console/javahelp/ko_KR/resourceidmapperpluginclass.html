자원 아이디 매퍼 플러그인 클래스

이 속성 값은 조직 또는 역할에 대해 구성된 자원 오퍼링의 식별자를 생성하는 클래스 목록입니다. com.sun.identity.liberty.ws.interfaces.ResourceIDMapper는 사용자 아이디를 연관된 자원 아이디에 매핑하는 데 사용하는 인터페이스입니다. 검색 서비스는 이 인터페이스에 대한 두 가지 구현을 제공합니다.

  • com.sun.identity.liberty.ws.disco.plugins.Default64ResourceIDMapper(형식: providerID + "/" + Base64 인코딩 사용자 아이디)

  • com.sun.identity.liberty.ws.disco.plugins.DefaultHexResourceIDMapper(형식: providerID + "/" + 사용자 아이디의 16진 문자열)
