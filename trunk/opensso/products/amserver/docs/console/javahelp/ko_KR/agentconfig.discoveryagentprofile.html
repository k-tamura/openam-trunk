검색 에이전트 속성

검색 에이전트 프로필에는 웹 서비스의 클라이언트/프로필에서 웹 서비스 조회, 등록 및 보안 자격 증명 획득을 위해 리버티 검색 서비스와 통신하는 데 사용할 신뢰 기관 구성이 포함되어 있습니다.

그룹

그룹 메커니즘을 사용하면 유사한 에이전트 유형 모음을 정의할 수 있습니다. 그룹은 특정 에이전트를 유형 모음에 포함하기 전에 정의해야 합니다.

비밀번호

검색 에이전트의 비밀번호를 정의합니다.

비밀번호 확인

비밀번호를 확인합니다.

상태

에이전트를 시스템에서 활성화할지 또는 비활성화할지를 정의합니다. 기본적으로 이 속성은 활성 상태로 설정되며, 이 상태에서는 에이전트가 웹 서비스 클라이언트에서 보낸 웹 서비스 요청을 보호하고 웹 서비스 공급자에서 보낸 서비스 응답의 유효성을 검사합니다.

에이전트 구성 저장소 위치

이 속성은 검색 에이전트에 대한 구성 저장소의 에이전트 위치를 정의합니다.

개인 키 별칭

이 속성은 웹 서비스 요청에 서명하거나 웹 서비스 응답을 해독하는 데 사용되는 개인 인증서 키 별칭을 정의합니다.

검색 서비스 종단점

이 속성은 신뢰 기관 클라이언트가 서비스 등록 및 조회를 위해 통신을 연결하는 검색 서비스 종단점을 정의합니다.

인증 웹 서비스 종단점

이 속성은 웹 서비스 클라이언트가 검색 서비스 자원 오퍼링(부트스트랩 자원 오퍼링이라고도 함)을 수신하기 위해 최종 사용자의 SSO 토큰을 사용하여 인증하는 데 사용하는 인증 서비스 종단점을 정의합니다.)
