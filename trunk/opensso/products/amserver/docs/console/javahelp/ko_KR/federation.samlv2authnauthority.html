SAMLv2 인증 권한 사용자 정의

SAMLv2 인증 권한에는 사용자 정의에 필요한 다음과 같은 속성이 포함되어 있습니다.

  • 서명 및 암호화

  • 인증 쿼리 서비스

  • AssertionID 요청

  • 인증서 별칭

서명 및 암호화
키 크기

다음 엔티티와 상호 작용할 때 속성 권한 엔티티에서 사용하는 키의 길이입니다.

알고리즘

다른 엔티티와 상호 작용하는 데 사용되는 암호화 알고리즘입니다.

인증 쿼리 서비스

이 속성은 인증 쿼리를 보낼 URL을 정의합니다.

AssertionID 요청

해당 명제를 검색하기 위해 클라이언트에서 Identity 공급자로 AssertionID를 보내는 URL을 정의합니다. 위치는 요청을 보낼 대상 공급자의 URL을 지정합니다. AssertionID 요청 유형은 다음과 같습니다.

  • SOAP

  • URI

인증서 별칭

이 속성은 공급자의 인증서 별칭 요소를 정의합니다. 서명은 키 저장소에서 올바른 서명 인증서를 찾는 데 사용되는 공급자 인증서 별칭을 지정합니다. 암호화는 키 저장소에서 올바른 암호화 인증서를 찾는 데 사용되는 공급자 인증서 별칭을 지정합니다.
