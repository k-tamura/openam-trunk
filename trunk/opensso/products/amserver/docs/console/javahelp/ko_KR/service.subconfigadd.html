하위 구성 추가

  1. [보조 구성 인스턴스] 목록에서 [새로 만들기]를 누릅니다.
  2. 새 하위 구성의 이름을 입력합니다.
  3. 다음 필드에 데이터를 입력합니다.
    세션 저장소 사용자

    세션 데이터를 검색하고 저장하는 데 사용되는 데이터베이스 사용자를 정의합니다.

    세션 저장소 비밀번호

    세션 저장소에 정의된 데이터베이스 사용자의 비밀번호를 정의합니다.

    세션 저장소 비밀번호(확인)

    비밀번호를 확인합니다.

    최대 대기 시간

    스레드가 데이터베이스 연결 객체를 얻기 위해 대기하는 총 시간을 정의합니다. 값은 밀리초 단위입니다.

    데이터베이스 URL

    데이터베이스의 URL을 지정합니다.

  4. [추가]를 누릅니다.
