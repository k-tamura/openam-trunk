Liberty ID-FF 서비스 구성

연합 쿠키 이름

이 속성은 ID-FF 서비스 쿠키의 이름을 지정합니다. 쿠키는 사용자가 이미 연합되어 있는지 여부를 기억하는 데 사용됩니다.

IDP 프록시 검색기 SPI 구현 클래스

이 속성은 프록시로 사용할 기본 Identity 공급자를 찾기 위한 구현 클래스를 지정합니다.

요청 캐시 정리 간격

이 속성은 ID-FF 내부 요청 정리 스레드의 정리 간격(초)을 지정합니다.

요청 캐시 시간 초과

이 속성은 ID-FF 인증 요청의 시간 초과 값(초)을 지정합니다. 해당 시간 초과 값을 초과하는 경우 모든 AuthnRequest 객체가 메모리에서 제거됩니다.

IDP 로그인 URl

이 속성은 인증 요청을 처리하는 동안 유효한 세션을 찾지 못하는 경우 IDP가 리디렉션되는 로그인 URL을 지정합니다. 해당 키를 지정하지 않으면 기본 로그인 URL이 사용됩니다.

XML 서명

이 속성은 리버티 요청 및 응답에 대한 서명의 유효성 검사 수준을 지정합니다.
