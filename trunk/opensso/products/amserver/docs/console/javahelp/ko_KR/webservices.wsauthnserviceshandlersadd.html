새 메커니즘 처리기 구성
  1. [추가]를 눌러 새 메커니즘 처리기 속성을 표시하거나 구성된 키 값을 눌러 수정합니다.
  2. 다음 정보를 기반으로 속성에 대한 값을 제공합니다.
    키

    이 필수 매개 변수는 인증 웹 서비스에서 지원하는 SASL 메커니즘을 정의합니다.

    클래스

    이 필수 매개 변수는 SASL 메커니즘의 구현 클래스 이름을 지정합니다. 인증 웹 서비스 계층은 각 SASL 메커니즘이 요청된 메시지를 처리하고 응답을 반환하기 위해 구현해야 하는 처리기 인터페이스를 제공합니다.

  3. [확인] 또는 [저장]을 눌러 요청 처리기 구성을 완료합니다.
  4. 인증 웹 서비스 페이지에서 [저장]을 눌러 서비스 구성을 완료합니다.
