새 사이트 인스턴스를 만들려면

  1. 사이트 목록에서 [새로 만들기]를 누릅니다.
  2. 서버 필드에 호스트 이름과 포트를 입력합니다.
  3. 사이트 이름을 입력합니다.

    이 값은 서버를 고유하게 식별하고 기본 URL에 추가로 해당 사이트에 대한 보조 시작점을 지정할 수 있게 해줍니다. 또한 이 식별자는 서버 URL을 서버 아이디에 매핑하여 쿠키 길이를 줄이는 데 사용됩니다.

  4. 사이트 URI를 포함하여 사이트 인스턴스의 기본 URL을 입력합니다.
  5. [저장]을 누릅니다.

    만든 사이트가 올바른 형식으로 사이트 목록에 표시됩니다.
