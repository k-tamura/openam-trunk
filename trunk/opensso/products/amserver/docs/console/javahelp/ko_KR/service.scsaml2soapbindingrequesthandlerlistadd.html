요청 처리기 구성

RequestHandler 인터페이스는 SOAP 바인드 서비스를 사용하는 각 SAMLv2 서비스가 서버측에서 구현해야 합니다. 요청 처리기 목록 속성은 요청 처리기를 구현하는 구현 클래스에 대한 정보를 저장합니다. 요청 처리기 목록에는 키/값 쌍이 포함된 항목이 표시됩니다.

  1. [새로 만들기]를 눌러 새 요청 처리기 속성을 표시하거나 구성된 키 값을 눌러 기존 속성을 수정합니다.
  2. 다음 정보를 기반으로 속성에 대한 값을 제공합니다.
    키

    키는 정책 결정 지점(PDP) 메타 별칭입니다.

    클래스

    클래스는 Java 클래스 이름으로, XACML 요청을 처리할 수 있는 RequestHandler 인터페이스의 구현입니다.

  3. [확인]을 눌러 요청 처리기 구성을 완료합니다.
  4. SAMLv2 SOAP 바인딩 페이지에서 [저장]을 눌러 서비스 구성을 완료합니다.
