요청 처리기 목록

SOAP 바인드 서비스는 요청 메시지를 처리하고 응답을 반환하는 RequestHandler 인터페이스를 제공합니다. 이 인터페이스는 SOAP 바인드 서비스를 사용하는 각 리버티 기반 Identity 서비스에 의해 서버측에서 구현되어야 합니다. 요청 처리기 목록 속성은 요청 처리기를 구현하는 웹 서비스의 구현 클래스에 대한 정보를 저장합니다. 요청 처리기 목록에서는 다음과 같이 파이프로 구분된 키/값 쌍이 포함된 항목을 표시합니다(예: key=disco|class=com.example.identity.liberty.ws.discoDiscoveryService). 해당 매개 변수는 다음 표에 설명되어 있습니다.

매개 변수

헤더

key

필수 key 매개 변수는 SOAP 종점에 대한 URI 경로의 마지막 부분입니다. OpenSSO에서 SOAP 종점은 SOAP 수신자 서블릿입니다. SOAP 수신자에 대한 URI는 protocol:// host:port/deploy-uri/Liberty/key입니다. disco를 key로 정의하면 해당 검색 서비스에 대한 SOAP 종점의 URI 경로는 다음과 같습니다. protocol://host:port /amserver/Liberty/disco. 서비스 클라이언트마다 서로 다른 키를 사용하여 SOAP 수신자를 연결합니다.

class

필수 class 매개 변수는 특정 Identity 서비스에 대한 요청 처리기 구현 클래스의 이름을 지정합니다. 예를 들면 다음과 같습니다. class=com.example.identity.liberty.ws.disco.DiscoveryService

새 요청 처리기를 추가하려면 요청 처리기 구성을 참조하십시오.
