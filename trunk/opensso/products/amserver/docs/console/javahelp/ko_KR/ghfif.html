에이전트 인증자

에이전트 인증자는 에이전트가 인증된 후 에이전트 인증자에서 읽기 위해 선택한 에이전트 프로필의 읽기 전용 데이터를 얻을 수 있는 에이전트 유형입니다. 에이전트 프로필은 J2EE, WSP, 검색 등의 모든 유형이 될 수 있지만 동일한 영역에 있어야 합니다. 에이전트 인증자 자격 증명(사용자 이름 및 비밀번호)이 있는 사용자는 에이전트 프로필 데이터를 읽을 수 있지만 에이전트 관리를 만들거나 업데이트하거나 삭제할 수 있는 권한은 없습니다.

에이전트 인증자에는 다음과 같은 속성이 있습니다.

비밀번호

비밀번호는 에이전트 인증자 프로필을 만들 때 설정됩니다. 그러나 이후에 언제든지 비밀번호를 변경할 수 있습니다.

비밀번호 확인

비밀번호 확인은 에이전트 인증자 프로필을 만들 때 수행됩니다. 비밀번호를 변경하면 변경된 비밀번호를 확인해야 합니다.

상태

에이전트 인증자를 만들면 활성 옵션이 선택됩니다. 에이전트가 제공하는 보호 기능을 제거하려는 경우에만 비활성을 선택하십시오.

읽을 수 있는 에이전트 프로필

이 속성은 에이전트 인증자가 프로필 데이터를 읽을 OpenSSO 에이전트의 목록을 정의합니다. 에이전트는 J2EE, WSP, 검색 등의 모든 유형이 될 수 있지만 동일한 영역에 있어야 합니다. 목록에 에이전트를 추가하려면 해당 에이전트 이름을 선택하고 [추가]를 누릅니다.
