WS-FED 공급자 엔티티

WS-FED 엔티티 공급자 유형은 WS-Federation 프로토콜을 기반으로 합니다. 이 프로토콜을 구현하면 OpenSSO와 Microsoft Active Directory Federation Service 간에 단일 사인온이 가능해집니다. WS—FED 공급자 엔티티를 사용하면 다음과 같은 역할을 할당하고 구성할 수 있습니다.

  • Identity 공급자

  • 서비스 공급자

연관된 작업 및 항목은 다음과 같습니다.

  • WS-FED 엔티티 공급자를 만들려면

  • WS-FED 일반 속성

  • WS-FED Identity 공급자 사용자 정의

  • WS-FED 서비스 공급자 사용자 정의
