SAMLv2 XACML PEP 사용자 정의

XACML PEP에는 사용자 정의에 사용할 수 있는 다음과 같은 속성이 포함되어 있습니다.

  • 프로토콜 지원 열거

  • 서명 키 별칭

  • 암호화 키 별칭

  • 기본 인증

  • 서명된 인증 결정 응답

  • 명제 암호화됨

프로토콜 지원 열거

이 공급자가 지원하는 XACML PEP 릴리스를 표시합니다.

서명 키 별칭

요청 및 응답에 서명하는 데 사용되는 키 별칭을 정의합니다.

암호화 키 별칭

XACML 암호화에 사용할 키 별칭을 정의합니다.

기본 인증

기본 인증을 사용하면 SOAP 엔드포인트를 보호할 수 있습니다. 이러한 엔드포인트에 액세스하는 공급자는사용자 이름 및 비밀번호 등록 정보를 사용하여 사용자와 비밀번호를 정의해야 합니다

서명된 인증 결정 응답

이 속성을 사용하면 XACML 인증 결정을 위한 모든 응답에 강제로 서명합니다.

명제 암호화됨

이 속성을 사용하면 모든 명제를 강제로 암호화합니다.
