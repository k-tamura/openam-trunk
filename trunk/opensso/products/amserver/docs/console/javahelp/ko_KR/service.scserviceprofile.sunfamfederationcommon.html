공통 연합 구성

Datastore SPI 구현 클래스

이 속성은 기본 데이터 저장소 공급자의 구현 클래스를 지정합니다.

구성 인스턴스 SPI 구현 클래스

이 속성은 구성 데이터를 관리하기 위한 구현 클래스를 지정합니다.

로거 SPI 구현 클래스

이 속성은 로그 공급자의 구현 클래스를 지정합니다.

세션 공급자 SPI 구현 클래스

이 속성은 세션 공급자의 구현 클래스를 지정합니다.

허용되는 최대 컨텐트 길이

이 속성은 연합 서비스에서 사용할 HTTP 요청에서 허용되는 최대 컨텐트 길이를 지정합니다. 컨텐트가 지정된 최대 컨텐트 길이를 초과하는 요청은 모두 거부됩니다.

비밀번호 디코더 SPI 구현 클래스

이 속성은 XML 서명 키 저장소에 저장된 비밀번호와 SAML 1.x의 기본 인증을 위한 비밀번호를 해독하는 데 사용되는 com.sun.identity.saml.xmlsig.PasswordDecoder 인터페이스의 구현 클래스를 지정합니다.

서명 공급자 SPI 구현 클래스

이 속성은 SAML XML 서명 공급자 클래스를 지정합니다. 기본 SPI는 com.sun.identity.saml.xmlsig.AMSignatureProvider 입니다.

키 공급자 SPI 구현 클래스

이 속성은 XML 서명 키 공급자 클래스를 지정합니다. 기본 SPI는 com.sun.identity.saml.xmlsig.JKSKeyProvider입니다.

인증서가 있는지 확인

on으로 설정한 경우 XML 서명 유효성 검사를 위해 키 저장소에 인증서가 있어야 합니다. off로 설정한 경우 인증서가 있는지 확인하지 않습니다. 이 속성은 SMAL1.x에만 적용됩니다.

XML 정규화 알고리즘

이 속성은 SAML XML 서명 생성 및 유효성 검사에 사용할 XML 정규화 알고리즘을 지정합니다. 기본값은 http://www.w3.org/2001/10/xml-exc-c14n#입니다.

XML 서명 알고리즘

이 속성은 SAML XML 서명 생성 및 유효성 검사에 사용할 XML 서명 알고리즘을 지정합니다. 값을 지정하지 않았거나 비어 있는 경우 기본값(http://www.w3.org/2000/09/xmldsig#rsa-sha1)이 사용됩니다.

XML 변환 알고리즘

이 속성은 SAML XML 서명 생성 및 유효성 검사에 사용할 변환 알고리즘을 지정합니다. 값을 지정하지 않았거나 비어 있는 경우 기본값(http://www.w3.org/2001/10/xml-exc-c14n#)이 사용됩니다.
