속성 매핑 구성

매핑은 Identity 데이터의 홀더를 정의하는 속성입니다. 새 속성을 리버티 개인 프로필 서비스에 추가하는 절차는 다음과 같습니다. 시작 위치는 웹 서비스 모듈 안에 있는 리버티 개인 프로필 서비스 구성 화면입니다.

  1. [새로 만들기]를 눌러 새 [속성 매핑] 속성을 표시하거나 구성된 매핑 이름을 눌러 수정합니다.
  2. 다음 정보를 기반으로 컨테이너 속성에 대한 값을 제공합니다.
    이름 접두어

    매핑할 리버티 개인 프로필 서비스 Identity 속성의 이름을 입력합니다.

    LDAP 속성

    이름 접두어가 매핑될 OpenSSO LDAP 속성의 이름을 입력합니다.

  3. [확인]을 눌러 매핑 구성을 완료합니다.
  4. 리버티 개인 프로필 서비스 페이지에서 [저장]을 눌러 서비스 구성을 완료합니다.
