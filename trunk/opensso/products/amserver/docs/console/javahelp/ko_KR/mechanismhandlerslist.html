메커니즘 처리기 목록

메커니즘 처리기 목록 속성은 인증 웹 서비스에서 지원하는 SASL 메커니즘에 대한 정보를 저장하며, 파이프 기호(|)로 구분된 키/값 쌍을 포함하는 항목을 표시합니다(예: key=PLAIN|class=com.sun.identity.liberty.ws.authnsvc.mechanism.PlainMechanismHandler ). 해당 매개 변수는 다음 표에 설명되어 있습니다.

매개 변수

값

key

필수 key 매개 변수는 인증 웹 서비스에서 지원하는 SASL 메커니즘을 정의합니다.

class

필수 class 매개 변수는 SASL 메커니즘의 구현 클래스 이름을 지정합니다. 인증 웹 서비스 계층은 각 SASL 메커니즘이 요청된 메시지를 처리하고 응답을 반환하기 위해 구현해야 하는 처리기 인터페이스를 제공합니다.

새 요청 처리기를 추가하려면 [추가]를 누릅니다. 새 메커니즘 처리기 구성을 참조하십시오.
