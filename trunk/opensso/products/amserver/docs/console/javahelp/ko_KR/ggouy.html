에이전트가 그룹에서 등록 정보를 상속받도록 하려면
시작하기 전에

먼저 에이전트가 등록 정보를 상속할 그룹을 만들어야 합니다.

  1. [액세스 제어] 탭을 누릅니다.
  2. 에이전트가 속하는 영역 이름을 누릅니다.
  3. [에이전트] 탭을 누릅니다.
  4. 필요한 경우 적절한 에이전트 유형에 해당하는 탭을 선택합니다.
  5. [에이전트] 섹션에서 구성할 에이전트의 이름을 누릅니다.
  6. [전역] 탭이 선택된 상태에서 그룹 속성에 대해 에이전트가 등록 정보를 상속할 그룹의 이름을 선택합니다.
  7. [저장]을 누릅니다.

    페이지 맨 위의 [상속 설정] 버튼이 활성화됩니다.

  8. [상속 설정]을 누릅니다.

    [전역] 탭에 상속 설정 목록이 영문자순으로 표시됩니다.

  9. 에이전트가 그룹에서 상속할 등록 정보를 선택합니다.

    페이지 맨 위의 [상속 설정] 버튼이 활성화됩니다.

  10. [저장]을 누릅니다.
다음 순서

이 작업은 [전역] 탭에 나열된 등록 정보의 상속 설정을 변경하는 방법을 설명하는 것입니다. [응용 프로그램] 등의 다른 탭에 나열된 등록 정보의 상속 설정의 경우에는 해당 탭을 누르고 이전 단계에서 설명한 대로 상속 설정을 편집하십시오.
