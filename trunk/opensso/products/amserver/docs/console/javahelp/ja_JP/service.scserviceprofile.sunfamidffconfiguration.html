Liberty ID-FF サービス設定

連携 Cookie 名

この属性は、ID-FF サービス Cookie の名前を指定します。この Cookie は、ユーザーがすでに連携されているかどうかを確認するために使用されます。

IDP プロキシ検索 SPI 実装クラス

この属性は、プロキシ化する優先アイデンティティープロバイダを見つけるための実装クラスを指定します。

要求キャッシュクリーンアップ間隔

この属性は、ID-FF 内部要求クリーンアップスレッドのクリーンアップ間隔を秒単位で指定します。

要求キャッシュタイムアウト

この属性は、ID-FF 認証要求のタイムアウト値を秒単位で指定します。このタイムアウト値を超えると、AuthnRequest オブジェクトはメモリーから消去されます。

IDP ログイン URl

この属性は、認証要求を処理する際に有効なセッションが見つからない場合に IDP がリダイレクトするログイン URL を指定します。キーが指定されていない場合は、デフォルトのログイン URL が使用されます。

XML 署名先

この属性は、Liberty 要求および応答の署名検証のレベルを指定します。
