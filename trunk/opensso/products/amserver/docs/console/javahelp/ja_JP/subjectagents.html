エージェント

Federated Access Manager のポリシーエージェントは、Web サーバーや Web プロキシサーバー上のコンテンツを不正な侵入から保護します。それらのエージェントは、管理者が設定したポリシーに基づいて、サービスや Web リソースへのアクセスを制御します。

「エージェント」オブジェクトはポリシーエージェントプロファイルを定義し、Federated Access Manager リソースを保護している特定のエージェントについての認証情報やその他のプロファイル情報を Federated Access Manager で格納できるようにします。管理者は、Federated Access Manager コンソールを使ってエージェントプロファイルを表示、作成、変更、および削除できます。

「エージェントを作成または変更する」を参照してください。
