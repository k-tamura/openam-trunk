WS 連携の一般属性

次の属性は、アイデンティティープロバイダとサービスプロバイダの両方に共通する属性です。

SP 表示名

この属性は、WS 連携サービスプロバイダの名前を定義します。デフォルトは、作成時に指定されたメタエイリアスです。

IDP 表示名

この属性は、WS 連携アイデンティティープロバイダの名前を定義します。デフォルトは、作成時に指定されたメタエイリアスです。

レルム

プロバイダが属するレルムを表示します。

トークン発行者名

アイデンティティープロバイダまたはサービスプロバイダの一意の ID を定義します。

トークン発行者エンドポイント

アイデンティティープロバイダまたはサービスプロバイダが WS 連携サービスを提供している URL を指定します。次に例を示します。

https://demo.example.com/opensso/WSFederationServlet/metaAlias/example
