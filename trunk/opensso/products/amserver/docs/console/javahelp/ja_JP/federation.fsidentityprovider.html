アイデンティティープロバイダをプロバイダエンティティーに追加する

アイデンティティープロバイダをプロバイダエンティティーに追加する手順の最初の段階では、追加するアイデンティティープロバイダのタイプを定義します。

  • ホストプロバイダは、Access Manager と同じサーバーにホストされます。

  • リモートプロバイダは、Access Manager と同じサーバーにはホストされません。

次に示す手順は、この最初の段階についての説明です。連携の「エンティティー」画面から作業を始めます。

  1. プロファイルを変更する設定済みプロバイダエンティティーの名前を選択します。

    エンティティーのプロファイルページが表示されます。

  2. 「表示」メニューから「アイデンティティープロバイダ」を選択します。
  3. 追加するプロバイダのタイプを選択します。

    • プロバイダが Access Manager と同じサーバーにホストされる場合は、「新規ホストプロバイダ」を選択します。

    • プロバイダが Access Manager と同じサーバーにホストされない場合は、「新規リモートプロバイダ」を選択します。

    それぞれに適した「アイデンティティープロバイダ」プロファイルページが表示されます。

  4. 前の手順で選択したものに基づいて、次のいずれかを選択します。
    • 「新しいホストアイデンティティープロバイダをプロバイダエンティティーに追加する」
    • 「新しいリモートアイデンティティープロバイダをプロバイダエンティティーに追加する」
