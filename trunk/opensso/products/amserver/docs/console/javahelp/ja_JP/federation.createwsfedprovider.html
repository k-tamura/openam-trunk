WS 連携プロバイダエンティティー

WS 連携エンティティープロバイダタイプは、WS 連携プロトコルに基づいています。このプロトコルの実装では、OpenSSO と Microsoft Active Directory 連携サービスの間でシングルサインオンが可能です。WS 連携プロバイダエンティティーでは、次のロールを割り当てて設定できます。

  • アイデンティティープロバイダ

  • サービスプロバイダ

関連する作業と項目は次のとおりです。

  • 「WS 連携エンティティープロバイダを作成する」

  • 「WS 連携の一般属性」

  • 「WS 連携アイデンティティープロバイダのカスタマイズ」

  • 「WS 連携サービスプロバイダのカスタマイズ」
