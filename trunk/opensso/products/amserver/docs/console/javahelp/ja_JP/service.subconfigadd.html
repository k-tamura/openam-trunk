サブ設定を追加する

  1. 「セカンダリ設定インスタンス」リストで「新規」をクリックします。
  2. 新しいサブ設定の名前を入力します。
  3. 次のフィールドのデータを入力します。
    セッションストアユーザー

    セッションデータの取得と格納に使用するデータベースユーザーを定義します。

    セッションストアパスワード

    セッションストアに定義したデータベースユーザーのパスワードを定義します。

    セッションストアパスワード (確認)

    パスワードを確認します。

    最大待ち時間

    スレッドがデータベース接続オブジェクトを取得するときの最大待ち時間を定義します。ミリ秒単位で指定します。

    データベース URL

    データベースの URL を指定します。

  4. 「追加」をクリックします。
