2.2 策略代理程式

Federated Access Manager 與策略代理程式 2.2 向下相容。通常，策略代理程式 2.2 必須從本身安裝所在的部署容器進行本機配置。因此，只有非常有限的策略代理程式 2.2 選項可從 Federated Access Manager 主控台配置。

密碼

密碼是在您建立代理程式設定檔時設定的。不過，您以後隨時可以變更密碼。

密碼確認

密碼確認是在您建立代理程式設定檔時執行。如果您變更密碼，則必須確認變更。

狀態

建立代理程式時，選取 [使用中] 選項。只有在您要移除代理程式提供的保護時，才選擇 [非使用中]。

描述

代理程式的描述，您可以視需要加入。

sunIdentityServerDeviceKeyValue

舊有屬性。
