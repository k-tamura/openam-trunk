OpenSSO 資料存放區類型

在 [OpenSSO 資料存放區選取] 畫面中，可以建立下列資料存放區類型：

Active Directory

此資料存放區類型使用 LDAP 版本 3 規格向 Microsoft Active Directory 的實例寫入識別資料。

如需屬性定義的相關資訊，請參閱 Active Directory 屬性。

通用 LDAPv3

此資料存放區類型可讓您向任何與 LDAPv3 相容的資料庫寫入識別資料。如果您使用的 LDAPv3 資料庫不支援持續搜尋，則無法使用快取功能。

如需屬性定義的相關資訊，請參閱通用 LDAPv3 屬性。

Sun Directory Server OpenSSO 模式

此資料存放區類型位於 Sun Java System Directory Server 實例中，並擁有 OpenSSO 資訊樹狀結構。

如需屬性定義的相關資訊，請參閱具有 OpenSSO 模式的 Sun Directory Server 屬性。
