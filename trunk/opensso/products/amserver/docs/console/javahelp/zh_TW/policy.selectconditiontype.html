將條件增加至策略
  1. 若您已建立策略，按一下您要增加規則的策略名稱。如果尚未建立策略，請參閱建立新的一般策略。
  2. 於 [條件] 清單下，按一下 [新建]。
  3. 選取條件類型並按 [下一步]。
  4. 定義條件類型的欄位。如需有關條件類型的說明，請按一下下列其中一項：

    • 使用中的階段作業時間

    • 按模組鏈認證

    • 按模組實例認證

    • 認證層級 (大於或等於)

    • 認證層級 (小於或等於)

    • 範圍認證

    • 目前的階段作業特性

    • 識別成員身份

    • IP 位址/DNS 名稱

    • LDAP 篩選器條件

    • 時間 (星期幾、日期、時間和時區)

  5. 按一下 [完成]。
