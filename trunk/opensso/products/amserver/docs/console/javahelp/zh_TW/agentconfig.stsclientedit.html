STS 用戶端

安全性記號服務 (Security Token Service, STS) 用戶端介面可讓您建立和配置用戶端，以便與 OpenSSO 的安全性記號服務進行通訊，從而取得安全性記號。OpenSSO 提供建立下列類型 STS 用戶端代理程式的機制：

探索代理程式

可讓您配置探索代理程式用戶端，用來與 Liberty 探索服務進行通訊，以取得基於 Liberty 的安全性記號。此配置定義保護探索用戶端對 Liberty 探索端點的 Liberty 請求的屬性。

安全性記號服務代理程式

可讓您配置安全性記號服務代理程式，用來與 OpenSSO 的安全性記號服務進行通訊，以取得基於 Web 服務的安全性記號。此配置定義保護 STS 用戶端對 STS 端點的 Web 服務信任請求的屬性。

在 STS 用戶端介面中，可以執行下列作業：

  • 配置探索代理程式屬性

  • 配置安全性記號服務代理程式屬性
