建立或修改使用者
  1. 按一下 [使用者] 標籤。
  2. 按一下 [開啟新檔] 。
  3. 輸入下列欄位的資料：

    使用者 ID。 此欄位中為將登入 Fedrated Access Manager 的使用者名稱。此特性可為非 DN 值。

    名字。此欄位中為使用者的名字。

    姓氏。此欄位中為使用者的姓氏。

    全名。此欄位中為使用者的全名。

    密碼。此欄位中為 [使用者 ID] 欄位中所指定名稱的密碼。

    密碼 (確認)。確認密碼。

    使用者狀態。此選項指出是否允許使用者透過 Fedrated Access Manager 認證。

  4. 按一下 [確定]。
  5. 一旦建立了使用者，您可以按一下使用者名稱來編輯使用者資訊。如需使用者資訊，請參閱使用者屬性。您可執行的其他修改：

    • 新增服務至識別

    • 新增使用者至角色與群組
