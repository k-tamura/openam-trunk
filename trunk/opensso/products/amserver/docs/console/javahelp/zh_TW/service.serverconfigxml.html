目錄配置

[目錄配置] 屬性定義內嵌式目錄存放區的基本配置資訊：

目錄配置

[目錄配置] 屬性包括：

最小連線池

指定用於連線至 Directory Server 的連線池最小大小，如 LDAP 伺服器屬性中指定的最小大小。預設值為 1。

最大連線池

此屬性指定用於連線至 Directory Server 的連線池最大大小，如 LDAP 伺服器屬性中指定的最大大小。預設值為 10。

連結 DN

指定 LDAP 伺服器中的連結 DN。

連結密碼

定義用於連結至 LDAP 伺服器的密碼。依預設，會使用安裝期間輸入的 amldapuser 密碼做為連結使用者。

伺服器

此屬性會定義當做 OpenSSO 實例的配置資料存放區使用的目錄伺服器。若要增加配置伺服器，按一下 [增加] 按鈕，然後提供下列屬性的值：

名稱

輸入伺服器的名稱。

主機名稱

指定 Directory Server 的完全合格的主機名稱。例如：

DirectoryServerHost.domainName.com

連接埠號碼

指定 Directory Server 的連接埠號碼。

連線類型

定義 Directory Server 的連線類型。依預設會選取 SIMPLE。您也可以選擇 SSL。

舊有配置

下列屬性會針對 OpenSSO 的舊有模式實例定義基本目錄伺服器配置。這些屬性僅會在 [舊有模式] 安裝中出現。

最小連線池

指定用於連線至 Directory Server 的連線池最小大小，如 LDAP 伺服器屬性中指定的最小大小。預設值為 1。

最大連線池

此屬性指定用於連線至 Directory Server 的連線池最大大小，如 LDAP 伺服器屬性中指定的最大大小。預設值為 10。

伺服器

此屬性會列出負載平衡器協定、主機名稱和連接埠。例如：http://lb.example.com:80。
