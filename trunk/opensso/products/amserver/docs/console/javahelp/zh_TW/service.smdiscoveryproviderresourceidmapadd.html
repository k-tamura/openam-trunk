配置 ResourceID 對映程式

com.sun.identity.liberty.ws.interfaces.ResourceIDMapper 是用來將使用者識別碼對映至與其相關聯之資源識別碼的介面。可以利用實作類別來開發不同的實作並新增至屬性。以下是將新的 ResourceID 對映程式新增至探索服務的程序。起點是 Web 服務模組內的探索服務配置畫面。

  1. 按一下 [新建] 以顯示新的對映程式屬性，或按一下已配置的對映程式名稱以修改它。
  2. 根據下列資訊提供對映屬性值：
    提供者 ID

    此屬性的值為指向探索服務的 URI。屬性值以如下格式書寫：http:// host:port/amserver/Liberty/disco

    ID 對映程式

    輸入實作類別的名稱。

  3. 按一下 [確定] 以完成對映程式的配置。
  4. 按一下探索服務頁面上的 [儲存] 以完成服務配置。
