代理程式

「集中式代理程式配置」為代理程式管理員提供從一個中央位置管理多個代理程式配置的方法。代理程式配置會儲存於 OpenSSO 的資料儲存庫，並由管理員透過 OpenSSO 主控台來管理。任何代理程式配置變更都會傳送至受影響的代理程式，且代理程式將根據更新特性的本質進行回應。您可以建立的特定代理程式設定檔包括：

  • 建立新群組和代理程式

  • Web 策略代理程式

  • J2EE 策略代理程式

  • Web 服務提供者

  • Web 服務用戶端屬性

  • STS 用戶端

  • 探索代理程式屬性

  • 安全性記號服務代理程式屬性

  • 2.2 策略代理程式
