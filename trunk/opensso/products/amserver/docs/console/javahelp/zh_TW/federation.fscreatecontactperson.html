新增連絡人員至提供者實體內的提供者

可將有關特定服務提供者或是識別提供者連絡人員的資訊新增至提供者實體內的提供者設定檔。起點為聯合之下的實體畫面。

  1. 在配置的實體名稱上按一下以修改其設定檔。

    會顯示實體的設定檔頁面。

  2. 從 [檢視] 功能表中選取識別提供者或服務提供者。
  3. 按一下 [連絡人員] 屬性群組之下的 [新建連絡人員]。
  4. 提供 [新建連絡人員] 屬性的值。
    名字

    鍵入實體連絡人員的指定名字。

    姓氏

    鍵入實體連絡人員的姓氏。

    類型

    從下拉式功能表中選擇連絡類型：

    • 管理

    • 帳單

    • 技術

    • 其他

    公司

    鍵入此人所屬公司的名稱。

    Liberty 首要 ID

    鍵入一個 URI，指向連絡人員的個人資訊設定檔的線上實例。

    電子郵件

    鍵入連絡人員的一個或多個電子郵件地址，並按一下 [新增]。

    電話號碼

    鍵入連絡人員的一個或多個電話號碼，並按一下 [新增]。

  5. 按一下 [建立] 以完成連絡人員配置。
  6. 按一下 [實體設定檔 ] 頁面上的 [儲存] 以完成實體配置。
