共用聯合配置

資料存放區 SPI 實作類別

此屬性指定預設資料存放區提供者的實作類別。

配置實例 SPI 實作類別

此屬性指定管理配置資料的實作類別。

記錄程式 SPI 實作類別

此屬性指定記錄提供者的實作類別。

階段作業提供者 SPI 實作類別

這會指定階段作業提供者的實作類別。

允許的內容長度最大值

此屬性會指定聯合服務中將使用之 HTTP 請求所允許的內容長度最大值。內容超出指定內容長度最大值的任何請求都會被拒絕。

密碼解碼器 SPI 實作類別

此屬性指定 com.sun.identity.saml.xmlsig.PasswordDecoder 介面的實作類別，該介面用來解碼儲存在 XML 簽署金鑰庫中的密碼，以及 SAML 1.x 下基本驗證的密碼。

簽名提供者 SPI 實作類別

此屬性指定 SAML XML 簽名提供者類別。預設的 SPI 為 com.sun.identity.saml.xmlsig.AMSignatureProvider。

金鑰提供者 SPI 實作類別

此屬性指定 XML 簽名金鑰提供者類別。預設的 SPI 為 com.sun.identity.saml.xmlsig.JKSKeyProvider。

請檢查憑證是否存在

如果設為 開啟，則金鑰庫中必須存在用於 XML 簽名驗證的憑證。如果設為 關閉，則不會檢查憑證是否存在。這僅適用於 SMAL1.x。

XML Canonicalization (標準化) 演算法

此屬性指定用於產生和驗證 SAML XML 簽名的 XML 標準化演算法。預設值為 http://www.w3.org/2001/10/xml-exc-c14n#。

XML 簽名演算法

此屬性指定用於產生和驗證 SAML XML 簽名的 XML 簽名演算法。未指定或值為空白時，會使用預設值 (http://www.w3.org/2000/09/xmldsig#rsa-sha1)。

XML 變換演算法

此屬性指定用於產生和驗證 SAML XML 簽名的變換演算法。未指定或值為空白時，會使用預設值 (http://www.w3.org/2001/10/xml-exc-c14n#)。
