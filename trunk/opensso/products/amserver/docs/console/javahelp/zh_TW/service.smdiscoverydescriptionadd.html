配置服務描述

服務說明屬性會在不同的協定終點定義執行中的 Web 服務。當您配置啟動程式的資源提供時進行定義。服務實例的相關資訊需要在各種環境中進行通訊。例如，定義的探索服務是識別服務，它會提供資源提供的列舉 (每一個都包括服務實例說明)。

  1. 按一下 [新增說明] 以顯示新的服務說明屬性，或按一下 [編輯] 以修改現有的說明。
  2. 根據下列資訊提供屬性值：
    安全機制 ID

    此屬性是新增探索服務所支援的認證方法的位置。這些安全機制會參考 Web 服務使用者對 Web 服務提供者進行認證，或提供訊息層級安全性的方式。依預設，會選取服務實例支援的所有可用方法。如果未選取認證方法，且 Web 服務使用者使用該方法傳送請求，將拒絕請求。請參閱支援的認證機制。

    端點 URL

    此屬性的值是 SOAP-over-HTTP 端點的 URL。URI 方案必須為如同 https://soap.profile-provider.com/soap 中的 HTTP 或 HTTPS。

    SOAP 動作

    此選擇性屬性的值等同於服務基於 WSDL 之具體說明中 wsdlsoap:operation 元素的 wsdlsoap:soapAction 屬性。

  3. 按一下 [確定] 以完成服務配置。
  4. 按一下探索服務頁面上的 [儲存] 以完成服務配置。
