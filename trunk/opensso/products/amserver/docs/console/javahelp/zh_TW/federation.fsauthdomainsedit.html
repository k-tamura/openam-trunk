修改信任圈設定檔

遵循此程序以編輯現有信任圈中配置的一般屬性，或是新增提供者至其中。起點為聯合介面下的信任圈。

  1. 按一下配置的信任圈名稱以修改其設定檔，或是新增提供者至其中。

    [編輯信任圈] 頁面隨即顯示。

  2. 為信任圈的一般屬性鍵入新值或編輯現有值：
    名稱

    此屬性的靜態值為您建立信任圈時提供的名稱。

    描述

    此屬性的值為信任圈的描述。如果適當的話，您可以修改已輸入的說明。

    IDFF 寫入器服務 URL

    此屬性指定寫入共用網域 cookie 的服務位置。URL 的格式為 http://common-domain-host:port/common/idffwriter。

    IDFF 讀取器服務 URL

    此屬性指定讀取共用網域 cookie 的服務位置。URL 的格式為 http://common-domain-host:port/common/idffreader。

    SAML2 寫入器 URL

    此屬性會指定將 Cookie 寫入共用網域的 SAML2 寫入器服務位置。URL 的格式為 http://common-domain-host: port/common/saml2writer。

    SAML2 讀取器 URL

    此屬性會指定將 Cookie 寫入共用網域的 SAML2 寫入器服務位置。URL 的格式為 http://common-domain-host: port/common/saml2reader。

    狀態

    預設狀態為使用中。選取 [非使用中] 會停用信任圈內的通訊。

  3. 選擇一個或多個可用的提供者，然後按一下 [新增] 箭頭以選取它們。

    提供的清單包含已建立並寫入提供者的實體名稱。如需詳細資訊，請參閱 將提供者新增到信任圈。

  4. 按一下 [儲存] 以完成作業。
