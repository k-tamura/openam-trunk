配置請求處理程式

每一個使用 SOAP 連結服務 的 SAMLv2 服務都必須在伺服器端實作 RequestHandler 介面。[請求處理程式清單] 屬性會儲存實作請求處理程式之實作類別的相關資訊。請求處理程式清單會顯示包含關鍵字/值對的項目。

  1. 按一下 [新建] 以顯示 [新建請求處理程式] 屬性，或按一下已配置的鍵值以修改現有屬性。
  2. 根據下列資訊提供屬性值：
    關鍵字

    關鍵字 是策略決定點中介別名。

    類別

    類別 是 Java 類別名稱，此類別是可以處理 XACML 請求之 RequestHandler 介面的實作。

  3. 按一下 [確定] 以完成請求處理程式的配置。
  4. 按一下 [SAMLv2 SOAP 連結] 頁面上的 [儲存] 以完成服務配置。
