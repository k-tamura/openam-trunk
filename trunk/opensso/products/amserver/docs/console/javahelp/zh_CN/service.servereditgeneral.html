常规

“常规”属性用于配置集中式服务器管理的基本配置数据。

站点属性

站点属性包括：

父站点

此属性将负载平衡器的“站点名称”（站点 ID）映射至 OpenSSO 服务器。请注意，必须先创建站点，然后才能添加该站点。

系统属性

系统属性列出服务器实例的位置信息：

基安装目录

指定产品数据所驻留的基目录。

默认语言环境

语言环境的值是安装 OpenSSO 时使用的默认语言子类型。默认值为 en_us。

通知 URL

通知服务端点的位置。在安装期间设置该值。

XML 验证

默认值为“否”。用于确定在使用 OpenSSO XMLUtils 类解析 XML 文档时是否需要进行验证。此属性仅在“调试级别”属性设置为“警告”或“消息”时有效。允许的值可以为“是”和“否”。仅在该属性的值为“是”且“调试级别”属性设置为“警告”或“消息”时才会打开 XML 文档验证。

调试属性

“调试”属性列出基本的错误检查信息：

调试级别

指定调试级别。默认值为“错误”。可能的值为：

  • 关 — 不创建调试文件。

  • 错误 — 只记录错误消息。

  • 警告 — 只记录警告消息。

  • 消息 — 记录错误、警告和信息消息。

调试目录

指定要在其中创建调试文件的输出目录。在安装期间设置该值。示例：OpenSSO-base/debug_URI/debug。

Mail Server

Mail Server 属性列出邮件服务器的主机名和端口：

Mail Server 主机名

默认值为 localhost。指定邮件服务器的主机。

Mail Server 端口号

默认值为 25。指定邮件服务器的端口号。
