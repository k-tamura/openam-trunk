OpenSSO 数据存储库类型

在“OpenSSO 数据存储库选择”屏幕中，可创建以下数据存储库类型：

Active Directory

该数据存储类型使用 LDAP 版本 3 规范将身份数据写入到 Microsoft Active Directory 的实例中。

有关属性定义的信息，参见Active Directory 属性。

普通 LDAPv3

该数据存储库类型允许将身份数据写入任意与 LDAPv3 兼容的数据库。如果所使用的 LDAPv3 数据库不支持持久性搜索，则无法使用高速缓存功能。

有关属性定义的信息，参见普通 LDAPv3 属性。

Sun Directory Server OpenSSO 模式

该数据存储库类型驻留在 Sun Java System Directory Server 实例中，并拥有 OpenSSO 信息树。

有关属性定义的信息，参见使用 OpenSSO 模式的 Sun Directory Server 属性。
