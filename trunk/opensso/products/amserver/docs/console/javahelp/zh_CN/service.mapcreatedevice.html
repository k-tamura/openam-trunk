添加新的客户机

  1. 在“客户机类型”列表中单击“新建”。
  2. 选择设备类型，包括以下字段：
    式样

    显示设备的基本式样。例如，HTML。

    设备用户代理

    接受设备的名称。

  3. 单击“下一步”。
  4. 输入新设备的以下信息：
    客户机类型名称

    接受设备的名称。此名称必须在所有设备中唯一。

    HTTP 用户字符串

    定义 HTTP 请求标头中的用户代理。例如，Mozilla/4.0。

  5. 单击“完成”。
  6. 要复制设备及其属性，请单击“复制”链接。设备名称必须唯一。默认情况下，OpenSSO 会将设备重命名为 copy_of_devicename。
