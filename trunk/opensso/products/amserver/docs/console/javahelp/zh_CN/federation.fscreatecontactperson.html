向提供者实体中的提供者添加联系人

在提供者实体中，可以将有关特殊服务提供者或身份认证提供者的联系人信息添加到提供者的配置文件中。起点是联合下的“实体”屏幕。

  1. 单击已配置的实体的名称，对其配置文件进行修改。

    随即显示实体的配置文件页面。

  2. 从“查看”菜单中选择“身份认证提供者”或“服务提供者”。
  3. 单击“联系人”属性组下的“新建联系人”。
  4. 提供“新建联系人”属性值。
    名字

    键入实体联系人的名。

    姓氏

    键入实体联系人的姓。

    类型

    从下拉式菜单中选择联系人类型：

    • 管理

    • 记帐

    • 技术

    • 其他

    公司

    键入雇用该人员的公司的名称。

    Liberty 主体 ID

    键入指向联系人个人信息配置文件联机实例的 URI。

    电子邮件

    键入联系人的一个或多个电子邮件地址，然后单击“添加”。

    电话号码

    键入联系人的一个或多个电话号码，然后单击“添加”。

  5. 单击“创建”完成“联系人”的配置操作。
  6. 单击“实体配置文件”页面上的“保存”完成实体的配置操作。
