配置请求处理程序

RequestHandler 接口必须由每一个使用 SOAP 绑定服务 的 SAMLv2 服务在服务器端实现。请求处理程序列表属性存储有关实现请求处理程序的实现类的信息。而且请求处理程序列表显示包含关键字/值对的条目。

  1. 单击“新建”以显示“新建请求处理程序”属性，或单击已配置的关键字值以修改现有属性。
  2. 基于以下信息，为属性提供值：
    关键字

    关键字是策略决策点元数据别名。

    类

    类是 Java 类名，该类是处理 XACML 请求的 RequestHandler 接口的实现。

  3. 单击“确定”完成“请求处理程序”的配置操作。
  4. 单击“SAMLv2 SOAP 绑定”页面上的“保存”完成服务的配置操作。
