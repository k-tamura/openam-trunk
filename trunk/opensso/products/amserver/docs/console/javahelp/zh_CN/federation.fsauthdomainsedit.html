修改信任环配置文件

按照以下步骤编辑现有信任环已配置的常规属性，或向其中添加提供者。起点是联合界面下的“信任环”。

  1. 单击已配置信任环的名称，修改其配置文件，或向其中添加提供者。

    随即显示“编辑信任环”页面。

  2. 键入信任环“常规”属性的新值或编辑其现有值：
    名称

    该属性的静态值是您创建信任环时提供的名称。

    说明

    该属性的值是信任环的说明。可以修改已输入的说明（如果适用）。

    IDFF 记录器服务 URL

    该属性指定了记录通用域 cookie 的服务的位置。该 URL 的格式为 http://common-domain-host:port/common/idffwriter。

    IDFF 读取器服务 URL

    该属性指定了读取通用域 cookie 的服务的位置。该 URL 的格式为 http://common-domain-host:port/common/idffreader。

    SAML2 记录器 URL

    该属性用于指定将 cookie 写入到通用域的 SAML2 记录器服务的位置。该 URL 的格式为 http://common-domain-host:port/common/saml2writer。

    SAML2 读取器 URL

    该属性用于指定将 cookie 写入到通用域的 SAML2 记录器服务的位置。该 URL 的格式为 http://common-domain-host:port/common/saml2reader。

    状态

    默认值为活动。选中“不活动”会禁用信任环内的通信。

  3. 选择一个或多个可用的提供者并单击“添加”箭头来选定它们。

    所提供的列表包含已创建并填充了提供者的实体的名称 。有关详细信息，参见添加提供者到信任环。

  4. 单击“保存”完成该操作。
