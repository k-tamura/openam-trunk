OpenSSO(Open Web SSO 項目) 作爲網絡基礎設施的安全組件，提供核心身份服務以簡化透明的 單次登入(SSO)的實作。OpenSSO提供了集成多樣化Web應用的基礎，其中可能是操作全異身份存 儲庫的Web應用，也可是寄宿於不同平台如Web伺服器和應用程式伺服器上的Web應用。此項目之 代碼基於 Sun JavaTM System Access Manager (由Sun Microsystems公司提供的核心身份基礎設施產品)。
