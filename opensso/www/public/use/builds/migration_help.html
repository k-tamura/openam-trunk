Help: Migration/Upgrading Related Changes

Please make sure that co-existence/backward compatible with previous versions is not broken.

This is the information that is required when your changes affect the migration and/or upgrading.

  • are there new service XML added?
  • are there any changes to existing service schema?
  • are there ldif changes?
  • are there any new properties in AMConfig.properties?
  • are there any schema changes to directory?

If any of the above is true, select the Yes on the migration radio button. Send an email with to is-upgrade@sun.com regarding the change

In the email, the following information need to be furnished:

  • Description of change to be consolidated and filtered for documentation purposes.
  • For Service XML changes, send the XML that can be used by amAdmin script to be uploaded.
  • For Directory schema changes, an LDIF script that must be uploadable.
  • Any other requirements or prerequisites while modifying the service configuration
