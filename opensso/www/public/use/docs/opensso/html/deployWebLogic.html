Deploying OpenSSO on WebLogic 8.1

In order to successfully deploy OpenSSO on WebLogic 8.1, update the startWebLogic.sh or the startWebLogic.cmd script (on Unix and Windows, respectively) to add the following JVM option:

-Djavax.xml.soap.MessageFactory=com.sun.xml.messaging.saaj.soap.ver1_0.SOAPMessageFactory1_1Impl
