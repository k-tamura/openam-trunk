OpenSSO Resource Center

The OpenSSO Resource Center is just what it says. New documentation from Express Build 7 and later, as well as reformatted documentation from docs.sun.com, now on a wiki. Plus links to blogs, training, technical articles and the like round out the Resource Center.

OpenSSO Express Build 7

OpenSSO Express Build 7 documentation is now available on the OpenSSO Documentation wiki.

Sun OpenSSO Enterprise 8.0 Documentation

Sun OpenSSO Enterprise 8.0 Documentation is now available on docs.sun.com in HTML and PDF format.

How Do I... FAQ

How Do I... is an FAQ page that explains how to set up different federation, SAML, and web service-based functions of OpenSSO.

Procedures

Procedures includes links to articles and blog entries written by OpenSSO community members. These documents might include drafts for review, procedures not yet tested, and information culled from the numerous email aliases.

Engineering Documents

Engineering documents includes system architecture, system requirements, system design, technical notes, and use cases.
