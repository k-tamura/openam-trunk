認証

Federated Access Manager は、一連のデフォルト認証モジュールタイプとともにインストールされます。認証モジュールインスタンスとは、ユーザー ID やパスワードなどのユーザー情報を収集して、その情報をデータベース内のエントリと比較して確認し、ユーザーにアクセスを許可したり拒否したりするプラグインのことです。同じタイプのインスタンスを複数作成して個別に設定することができます。

このセクションでは、デフォルトの認証モジュールタイプを設定する属性について説明します。
