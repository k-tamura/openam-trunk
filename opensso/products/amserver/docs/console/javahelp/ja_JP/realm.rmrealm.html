レルム

アクセス制御レルムとは、特定の認証プロパティーと承認ポリシーの集合体のことです。アクセス制御レルムには、ユーザーやユーザーグループを関連付けることができます。レルムのデータは、ユーザーが指定したデータストア内に OpenSSO が作成した専用の情報ツリー内に格納されます。OpenSSO フレームワークは、OpenSSO 情報ツリー内の各レルムに含まれるポリシーおよびプロパティーを集約します。OpenSSO 8.0 はデフォルトで、OpenSSO 情報ツリーを Sun Java Enterprise System Directory Server 内に自動的に挿入しますが、その際、ユーザーデータとは別の特殊なブランチとして挿入します。アクセス制御レルムは、任意の LDAPv3 データベースと併用できます。
