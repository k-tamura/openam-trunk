OpenSSO のデータストアの種類

OpenSSO のデータストア選択画面で作成できるデータストアの種類は次のとおりです。

Active Directory

このタイプのデータストアは、LDAP version 3 仕様に基づいて、Microsoft Active Directory インスタンスにアイデンティティーデータを書き込みます。

属性の定義については、「Active Directory の属性」を参照してください。

汎用 LDAPv3

このタイプのデータストアは、アイデンティティーデータを任意の LDAPv3 準拠データベースに書き込むことができます。使用している LDAPv3 データベースが持続検索をサポートしない場合は、キャッシュ機能を利用できません。

属性の定義については、「汎用 LDAPv3 属性」を参照してください。

OpenSSO スキーマを含んだ Sun Directory Server

このタイプのデータストアは Sun Java System Directory Server インスタンス内に存在し、そこに OpenSSO 情報ツリーが格納されます。

属性の定義については、「OpenSSO スキーマ属性を含んだ Sun Directory Server」を参照してください。
