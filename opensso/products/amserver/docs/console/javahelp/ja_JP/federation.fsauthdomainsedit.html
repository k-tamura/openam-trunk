トラストサークルプロファイルを変更する

次の手順に従って、既存のトラストサークルの設定済み「一般」属性を編集するか、既存のトラストサークルにプロバイダを追加します。連携インタフェースの「トラストサークル」から作業を始めます。

  1. 設定済みのトラストサークルの名前をクリックして、そのプロファイルを変更するか、またはプロバイダを追加します。

    「トラストサークルを編集」ページが表示されます。

  2. トラストサークルの次の「一般」属性に新規の値を入力するか、または既存の値を編集します。
    名前

    この属性の静的な値は、トラストサークルの作成時に指定した名前です。

    説明

    この属性の値はトラストサークルの説明です。適切な場合には、すでに入力されている説明を変更することもできます。

    IDFF ライターサービス URL

    この属性は、共通ドメインからの Cookie を書き込むサービスの場所を指定します。URL の形式は http://共通ドメインホスト:ポート/common/idffwriter です。

    IDFF リーダーサービス URL

    この属性は、共通ドメインからの Cookie を読み込むサービスの場所を指定します。URL の形式は http://共通ドメインホスト:ポート/common/idffreader です。

    SAML2 ライター URL

    この属性は、共通ドメインに Cookie を書き込む SAML2 ライターサービスの場所を指定します。URL の形式は http://共通ドメインホスト:ポート/common/saml2writer です。

    SAML2 リーダー URL

    この属性は、共通ドメインに Cookie を書き込む SAML2 ライターサービスの場所を指定します。URL の形式は http://共通ドメインホスト:ポート/common/saml2reader です。

    状態

    デフォルトの状態は「アクティブ」です。「非アクティブ」を選択すると、トラストサークル内の通信が無効になります。

  3. 1 つ以上の利用可能なプロバイダを選択し、「追加」矢印をクリックして追加します。

    表示される一覧には、作成されプロバイダが設定されたエンティティーの名前が入ります。詳細は、「トラストサークルにプロバイダを追加する」を参照してください。

  4. 「保存」をクリックして、操作を完了します。
