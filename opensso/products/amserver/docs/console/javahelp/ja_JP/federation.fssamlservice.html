SAML1.x

SAML (Security Assertion Markup Language) は、XML フレームワークを使用してオーソリティーと信頼パートナーサイトとの間でセキュリティー情報を交換する、オープン規格のプロトコルです。セキュリティー情報は、対象の認証状態、アクセス許可および属性の情報に関係しています。電子メールアドレスで識別される個人は、プリンタなどと同様に対象の一種です。SAML オーソリティーは、表明機関とも呼ばれますが、SAML API で統合されたプラットフォームまたはアプリケーションであり、セキュリティー情報を中継することができます。たとえば、必要な属性を渡すことによって、対象が当該システムにすでに認証されていることを表明します。信頼パートナーサイトは、信頼できるセキュリティー情報を受け取ることができます。

「ローカルサイトのプロパティー」ボタンをクリックすると、次の SAML 属性を実装に合わせて設定できます。

  • 「ターゲット指定子」

  • 「サイト ID」

  • 「信頼パートナー」

  • 「ターゲット URL」

  • 「表明のタイムアウト」

  • 「notBefore 時間の表明スキュー係数」

  • 「アーティファクトのタイムアウト」

  • 「SAML アーティファクト名」

  • 「署名 SAML 表明」

  • 「署名 SAML 要求」

  • 「署名 SAML 応答」

SAML の設定に関連する作業は、次のとおりです。

  • 「サイト ID を設定する」

  • 「信頼パートナー: パートナータイプおよびプロファイルを選択する」

  • 「信頼パートナー: 信頼パートナー属性を設定する」

  • 「ターゲット URL を設定する」
