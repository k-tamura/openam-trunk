リソース ID マッパープラグイン用のクラス

この属性の値は、組織またはロールに対して設定されるリソースオファリングの ID を生成するクラスのリストです。com.sun.identity.liberty.ws.interfaces.ResourceIDMapper は、ユーザー ID を、それと関連付けられたリソース ID にマップするために使用されるインタフェースです。ディスカバリサービスは、このインタフェースのために次の 2 つの実装プラグインを提供します。

  • com.sun.identity.liberty.ws.disco.plugins.Default64ResourceIDMapper (形式: プロバイダ ID + "/" + Base64 エンコードされたユーザー ID)

  • com.sun.identity.liberty.ws.disco.plugins.DefaultHexResourceIDMapper (形式: プロバイダ ID + "/" + ユーザー ID の 16 進文字列)

実装クラスを使用してさまざまな実装プラグインを開発し、この属性の値として追加することができます。そのためには、「新規」をクリックして次の形式を使用します。

providerid=providerID| class_name_and_path
