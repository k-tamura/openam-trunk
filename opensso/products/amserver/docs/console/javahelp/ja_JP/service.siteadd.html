新しいサイトインスタンスを作成する

  1. 「サイト」リストの「新規」をクリックします。
  2. 「サーバー」フィールドにホスト名とポートを入力します。
  3. サイト名を入力します。

    この値によって、サーバーを一意に識別し、(プライマリ URL に加えて) 2 つ目のエントリポイントをサイトに指定できるようになります。この ID は、サーバー URL をサーバー ID にマッピングして Cookie の文字数を短くするためにも使用されます。

  4. サイトインスタンスのプライマリ URL (サイトの URI を含む) を入力します。
  5. 「保存」をクリックします。

    作成されたサイトが、正しい形式でサイトリストに表示されます。
