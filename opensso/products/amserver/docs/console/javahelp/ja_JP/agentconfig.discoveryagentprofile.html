ディスカバリエージェント属性

ディスカバリエージェントプロファイルは信頼できる発行局の設定を保持します。この設定は、Web サービスの検索と登録、およびセキュリティー証明情報の取得のために、Web サービスのクライアント/プロファイルで Liberty ディスカバリサービスとの通信に使用されます。

グループ

「グループ」メカニズムを使用すると、類似のエージェントタイプのコレクションを定義できます。特定のエージェントをコレクションに追加する前に、グループを定義しておく必要があります。

パスワード

ディスカバリエージェントのパスワードを定義します。

パスワードの確認

パスワードを確認します。

ステータス

エージェントがシステム内でアクティブになるかどうかを定義します。この属性は、デフォルトではアクティブになります。つまり、エージェントは Web サービスクライアントからのアウトバウンドの Web サービス要求のセキュリティー保護に参加し、Web サービスプロバイダからの Web サービス応答を検証します。

エージェント設定リポジトリの場所

この属性は、ディスカバリエージェントの設定リポジトリの場所を定義します。

非公開鍵のエイリアス

この属性は、Web サービス要求の署名または Web サービス応答の復号化に使用される証明書非公開鍵エイリアスを定義します。

ディスカバリサービスエンドポイント

この属性は、信頼できる発行局クライアントがサービスの登録と検索のために通信を確立する、ディスカバリサービスエンドポイントを定義します。

認証 Web サービスエンドポイント

この属性は、Web サービスクライアントがエンドユーザーの SSOToken を使用してディスカバリサービスのリソースオファリング (ブートストラップリソースオファリングとも呼ばれる) を受け取るための認証に使用する、認証サービスエンドポイントを定義します。
