連携

連携インタフェースは、Liberty Alliance Project Identity Federation Framework (Liberty ID-FF) 仕様の Access Manager 実装です。Liberty ID-FF は、アイデンティティー連携、ドメイン間認証およびセッション管理のソリューションを提供する、一連のプロトコル、バインド、およびプロファイルです。Access Manager コンソールの連携インタフェースは、次のものを設定および保守する手段を提供します。

  • 「認証ドメイン」

  • 「エンティティー: プロバイダおよびアフィリエイト」

  • 「SAML」
