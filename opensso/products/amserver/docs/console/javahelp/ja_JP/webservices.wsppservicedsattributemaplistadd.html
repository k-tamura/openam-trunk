属性マッピングを設定する

「マッピング」は、アイデンティティーデータのホルダーを定義する属性の一種です。次に示すのは、新しい属性を Liberty 個人プロファイルサービスに追加する手順です。Web サービスモジュール内の Liberty 個人プロファイルサービス設定画面から作業を始めます。

  1. 「新規」をクリックして新規の属性マッピング属性を表示するか、設定済みマッピングの名前をクリックしてそれを変更します。
  2. 次の情報に基づいてコンテナ属性の値を入力します。
    名前プレフィックス

    マップされる Liberty 個人プロファイルサービスアイデンティティー属性の名前を入力します。

    LDAP 属性

    名前プレフィックスがマップする OpenSSO LDAP 属性の名前を入力します。

  3. 「了解」をクリックしてマッピングの設定を完了します。
  4. Liberty 個人プロファイルサービスページで「保存」をクリックして、サービスの設定を完了します。
