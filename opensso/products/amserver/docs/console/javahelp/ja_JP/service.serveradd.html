新しいサーバーインスタンスを作成する

各サーバーのエントリは、OpenSSO 設定ツールを実行してサーバーを設定したときに、サーバーリスト内に自動的に作成されます。通常の状況では、これらの手順は必要ありません。

  1. OpenSSO コンソールに最上位の管理者としてログインします。
  2. 「設定」タブをクリックし、「サイトおよびサーバー」をクリックします。
  3. 「サーバー」リストの「新規」をクリックします。
  4. 追加するサーバーの FQDN を入力し、「了解」をクリックします。

    FQDN の形式は http (s)://host.domain:port /uri です。

  5. 新しく作成されたサーバーインスタンスがリストに表示されます。
  6. サーバーを編集するには、サーバーの名前をクリックします。サーバーの設定属性が表示され、カスタマイズできるようになります。

    「デフォルトのサーバー設定値」は、サーバーインスタンスのデフォルト値のセットです。各サーバーインスタンスには最小限のプロパティー値セットを設定する必要があり、配備によっては、すべてのサーバーインスタンスのほとんどのプロパティー値を同じにすることができます。この設定を使用すると、基本的なプロパティーを 1 か所で入力でき、サーバーインスタンスを追加するたびにこれらのプロパティーを変更する必要がなくなります。

    これらのデフォルト値は上書きできます。そのためには、サーバーインスタンスプロファイルページの上部にある「継承設定値」ボタンをクリックします。このボタンをクリックすると、継承する値や上書きする値を選択および選択解除できるページがコンソールに表示されます。
