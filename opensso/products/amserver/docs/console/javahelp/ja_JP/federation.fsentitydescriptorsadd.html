プロバイダエンティティーまたはアフィリエイトエンティティーを作成する

エンティティーの確立は 2 段階の処理です。まず、エンティティーを作成します。エンティティーをプロバイダエンティティーにするかアフィリエイトエンティティーにするかを、ここで定義します。エンティティーを作成したら、そのエンティティーにプロバイダを設定します。新しいエンティティーは次の手順で作成します。連携インタフェースの「エンティティー」から作業を始めます。

注 – エンティティーは、いつでも作成できますが、プロバイダを設定してはじめてトラストサークルに割り当てることができます。

  1. 「新規」をクリックして、エンティティー属性を表示します。

    「新規エンティティー」ページが表示されます。

  2. エンティティーの名前を入力します。

    「エンティティー名」属性は、エンティティーの URI (Uniform Resource Identifier) を定義します。この値は、エンティティー名の間で一意でなければなりません。

  3. エンティティーの説明を入力します。
  4. エンティティーの種類を選択します。

    • プロバイダエンティティーを作成する場合は、「プロバイダ」を選択します。

    • アフィリエイトエンティティーを作成する場合は、「アフィリエイト」を選択し、必須の「アフィリエイト名」および「アフィリエイト所有者」属性の値を定義します。

      • 「アフィリエイト名」はアフィリエイトエンティティーの URI を定義します。この値は、アフィリエイト名の間で一意でなければなりません。

      • 「アフィリエイト所有者」は、アフィリエイトの親管理者としてプロバイダを指定します。この属性の値は、アフィリエイト所有者/プロバイダが属するエンティティーの「エンティティー名」属性に定義された値と一致する URI です。

      注 – プロバイダを「アフィリエイト所有者」として定義しても、そのプロバイダがアフィリエイトのメンバーとして自動的に組み込まれることはありません。所有者がメンバーでもある場合は、URI をここでは所有者として定義し、エンティティー内部ではメンバーとして定義する必要があります。アフィリエイトにメンバーを追加する場合は、「アフィリエイトエンティティーにメンバーを追加する」を参照してください。

  5. 「了解」をクリックします。

    新しいエンティティーが「エンティティー」画面に表示されます。
