새 인스턴스 이름 만들기

이름 지정 서비스는 이 속성을 초기화 시점에 읽습니다. 이 목록에는 단일 Federated Access Manager 구성의 Federated Access Manager 세션 서버가 포함되어 있습니다.

  1. 인스턴스 이름 목록에서 [새로 만들기]를 누릅니다.
  2. 서버 필드에 호스트 이름과 포트를 입력합니다.
  3. 사이트 이름을 입력합니다.

    이 값은 서버를 고유하게 식별합니다. 로드 균형 조정 또는 페일오버에 참여하는 각 서버는 고유한 식별자를 가져야 합니다. 또한 이 식별자는 서버 URL을 서버 아이디에 매핑하여 쿠키 길이를 줄이는 데 사용됩니다. 구문은 다음과 같습니다.

    intance_ID(|site_ID)

  4. [확인]을 누릅니다.

    인스턴스 이름을 편집하려면 인스턴스 이름 목록에서 항목을 누르고 값을 변경합니다.

  5. 플랫폼 서비스 주 페이지에서 [저장]을 누릅니다.
