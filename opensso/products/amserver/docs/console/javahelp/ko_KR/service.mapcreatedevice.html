새 클라이언트 추가

  1. [클라이언트 유형] 목록에서 [새로 만들기]를 누릅니다.
  2. 다음 필드를 사용하여 장치 유형을 선택합니다.
    스타일

    장치의 기본 스타일을 표시합니다. 예를 들어 HTML이 있습니다.

    장치 사용자 에이전트

    장치에 대한 이름을 지정합니다.

  3. [다음]을 누릅니다.
  4. 새 장치에 대한 다음 정보를 입력합니다.
    클라이언트 유형 이름

    장치에 대한 이름을 지정합니다. 이름은 모든 장치 간에 고유해야 합니다.

    HTTP 사용자 문자열

    HTTP 요청 헤더에 사용자 에이전트를 정의합니다. 예를 들어 Mozilla/4.0입니다.

  5. [마침]을 누릅니다.
  6. 장치와 장치 등록 정보를 복제하려면 [복제] 링크를 누릅니다. 장치 이름은 고유해야 합니다. 기본적으로 OpenSSO는 장치 이름을 copy_of_devicename으로 변경합니다.
