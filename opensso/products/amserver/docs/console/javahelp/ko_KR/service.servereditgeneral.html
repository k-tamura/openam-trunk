일반

일반 속성은 중앙 집중화된 서버 관리를 위한 기본 구성 데이터를 구성합니다.

사이트 속성

사이트 속성은 다음과 같습니다.

상위 사이트

이 속성은 로드 밸런서 사이트 이름(사이트 아이디)을 OpenSSO 서버에 매핑합니다. 사이트를 추가하기 전에 먼저 사이트를 만들어야 합니다.

시스템 속성

시스템 속성은 서버 인스턴스의 위치 정보를 나열합니다.

기본 설치 디렉토리

제품 데이터가 있는 기본 디렉토리를 지정합니다.

기본 로켈

로켈 값은 OpenSSO를 설치할 때 사용한 기본 언어 하위 유형입니다. 기본값은 en_us입니다.

알림 URL

알림 서비스 엔드포인트의 위치입니다. 이 값은 설치 중에 설정됩니다.

XML 유효성 검사

기본값은 no입니다. OpenSSO XMLUtils 클래스를 사용하여 XML 문서를 구문 분석할 때 유효성을 검사해야 할지 여부를 결정합니다. 이 등록 정보는 [디버그 수준] 속성 값이 경고 또는 메시지로 설정된 경우에만 유효합니다. 허용되는 값은 yes 및 no입니다. XML 문서 유효성 검사는 이 등록 정보 값이 yes이고 [디버그 수준] 속성 값이 경고 또는 메시지로 설정된 경우에만 설정됩니다.

디버깅 속성

디버깅 속성은 기본 오류 검사 정보를 나열합니다.

디버그 수준

디버그 수준을 지정합니다. 기본값은 오류입니다. 사용 가능한 값은 다음과 같습니다.

  • 해제 — 디버그 파일을 만들지 않습니다.

  • 오류 — 오류 메시지만 기록합니다.

  • 경고 — 경고 메시지만 기록합니다.

  • 메시지 — 오류, 경고 및 정보 메시지를 기록합니다.

디버그 디렉토리

디버그 파일을 만들 출력 디렉토리를 지정합니다. 이 값은 설치 중에 설정됩니다. 예: OpenSSO-base /debug_URI/debug

메일 서버

메일 서버 속성은 메일 서버의 호스트 이름과 포트를 나열합니다.

메일 서버 호스트 이름

기본값은 localhost입니다. 메일 서버 호스트를 지정합니다.

메일 서버 포트 번호

기본값은 25이며 메일 서버 포트를 지정합니다.
