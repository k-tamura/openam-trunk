OpenSSO 데이터 저장소 유형

[OpenSSO 데이터 저장소 선택] 화면에서 다음 데이터 저장소 유형을 만들 수 있습니다.

활성 디렉토리

이 데이터 유형은 LDAP 버전 3 사양을 사용하여 Microsoft Active Directory 인스턴스에 Identity 데이터를 기록합니다.

속성 정의에 대한 자세한 내용은 활성 디렉토리 속성을 참조하십시오.

일반 LDAPv3

이 데이터 저장소 유형을 사용하면 Identity 데이터를 모든 LDAPv3 호환 데이터베이스에 기록할 수 있습니다. 사용 중인 LDAPv3 데이터베이스가 영구 검색을 지원하지 않는 경우 캐시 기능을 사용할 수 없습니다.

속성 정의에 대한 자세한 내용은 일반 LDAPv3 속성을 참조하십시오.

Sun Directory Server OpenSSO 스키마

이 데이터 저장소 유형은 Sun Java System Directory Server 인스턴스에 있으며 OpenSSO 정보 트리를 저장합니다.

속성 정의에 대한 자세한 내용은 Sun Directory Server(OpenSSO 스키마 속성 포함)를 참조하십시오.
