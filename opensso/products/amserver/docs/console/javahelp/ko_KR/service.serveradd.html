새 서버 인스턴스를 만들려면

OpenSSO 구성자가 서버 구성을 실행하면 각 서버에 대한 항목이 서버 목록에 자동으로 만들어집니다. 따라서 정상적인 상태에서는 다음 단계가 필요하지 않습니다.

  1. OpenSSO 콘솔에 최상위 관리자로 로그인합니다.
  2. [구성] 탭을 누른 다음 [사이트 및 서버]를 누릅니다.
  3. 서버 목록에서 [새로 만들기]를 누릅니다.
  4. 추가할 서버의 FQDN을 입력하고 [확인]을 누릅니다.

    FQDN은 http (s)://host.domain:port /uri 형식이어야 합니다.

  5. 새로 만든 서버 인스턴스가 목록에 표시됩니다.
  6. 서버를 편집하려면 해당 서버의 이름을 누릅니다. 서버의 구성 속성을 사용자 정의할 수 있습니다.

    [기본 서버 설정]이 서버 인스턴스의 기본값 집합입니다. 각 서버 인스턴스에는 최소 등록 정보 값 집합이 있어야 하며, 배포 환경에 따라 대부분의 등록 정보 값을 모든 서버 인스턴스에서 동일하게 설정할 수 있습니다. 이 설정을 사용하면 추가 서버 인스턴스의 등록 정보를 각각 변경할 필요 없이 기본 등록 정보를 한 곳에 입력할 수 있습니다.

    이러한 기본값은 덮어쓸 수 있습니다. 서버 인스턴스 프로필 페이지의 맨 위에 있는 [상속 설정] 버튼을 누르면 덮어쓰기가 가능합니다. 이 버튼을 누르면 상속하거나 덮어쓸 값을 선택 및 선택 취소할 수 있는 페이지가 콘솔에 표시됩니다.
