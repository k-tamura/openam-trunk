데이터 저장소

데이터 저장소는 사용자 속성 및 사용자 구성 데이터를 저장할 수 있는 데이터베이스입니다.

OpenSSO는 LDAPv3 Identity 저장소 프레임워크에 연결할 Identity 저장소 플러그인을 제공합니다. 이를 통해 기존 사용자 데이터베이스를 변경하지 않고도 OpenSSO 사용자 정보를 확인하고 검색할 수 있습니다. OpenSSO 프레임워크는 Identity 저장소 플러그인의 데이터를 다른 OpenSSO 플러그인의 데이터와 통합하여 각 사용자의 가상 Identity를 형성합니다. OpenSSO는 이제 두 개 이상의 Identity 저장소 간의 인증 및 권한 부여 과정에 범용 Identity를 사용할 수 있습니다. 가상 사용자 Identity는 사용자 세션이 종료되면 소멸됩니다.

OpenSSO는 다음과 같은 데이터 저장소를 지원합니다.

  • 활성 디렉토리

  • 일반 LDAPv3

  • Sun Directory Server(OpenSSO 스키마 포함)

새 데이터 저장소 만들기 및 저장소 플러그인 구성에 대한 자세한 내용은 새 데이터 저장소 만들기를 참조하십시오.
