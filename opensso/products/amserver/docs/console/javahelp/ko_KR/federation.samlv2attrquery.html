SAMLv2 속성 쿼리 사용자 정의

SAMLv2 속성 쿼리에는 사용자 정의에 사용할 수 있는 다음과 같은 속성이 포함되어 있습니다.

  • NameID 형식

  • 인증서 별칭

NameID 형식

속성 쿼리 공급자에서 지원되는 이름 식별자 형식을 정의합니다. 이름 식별자는 공급자가 한 사용자와 관련하여 서로 통신하는 방법입니다. 단일 사인온 상호 작용은 세 가지 유형의 식별자를 지원합니다.

  • X509SubjectName은 X509 암호화 유형의 주제 이름을 정의합니다.

  • 영구 식별자는 특정 사용자의 데이터 저장소 항목에 두 속성의 값으로 저장되어 있습니다.

  • 일시적 식별자는 임시로 사용되며 사용자의 영구 데이터 저장소에는 데이터가 기록되지 않습니다.

인증서 별칭

이 속성은 공급자의 인증서 별칭 요소를 정의합니다. 서명은 키 저장소에서 올바른 서명 인증서를 찾는 데 사용되는 공급자 인증서 별칭을 지정합니다. 암호화는 키 저장소에서 올바른 암호화 인증서를 찾는 데 사용되는 공급자 인증서 별칭을 지정합니다.
