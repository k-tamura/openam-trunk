영역에 서비스 추가

  1. 새 서비스를 추가할 영역의 이름을 누릅니다.
  2. [서비스] 탭을 선택합니다.
  3. 서비스 목록에서 [추가]를 누릅니다.
  4. 영역에 추가할 서비스를 선택합니다.
  5. [다음]을 누릅니다.
  6. 영역 속성을 정의하여 서비스를 구성합니다. 다음 서비스에 대한 속성 정의를 참조합니다.

    • 관리

    • 정책 구성

    • 검색 서비스

    • 국제화 설정

    • 비밀번호 재설정

    • 세션

    • 사용자

  7. [마침]을 누릅니다.
  8. 서비스의 등록 정보를 편집하려면 서비스 목록에서 해당 이름을 누릅니다.
