SAMLv2 공급자 엔티티

SAMLv2 엔티티 공급자 유형은 OASIS SAML(Security Assertion Markup Language) 버전 2 사양을 바탕으로 합니다. 이 엔티티는 원격 SAMLv2 엔티티와 상호 작용하는 경우 다양한 프로필(단일 사인온, 단일 사인아웃 등)을 지원합니다. SAMLv2 공급자 엔티티를 사용하면 다음과 같은 역할을 할당하고 구성할 수 있습니다.

  • Identity 공급자

  • 서비스 공급자

  • XACML PEP

  • XACML PDP

  • 속성 권한

  • 속성 쿼리

  • 인증 기관

  • 호스트 제휴

연관된 작업 및 항목은 다음과 같습니다.

  • 엔티티 공급자

  • SAMLv2 서비스 공급자 사용자 정의

  • SAMLv2 Identity 공급자 사용자 정의

  • SAMLv2 XACML PDP 사용자 정의

  • SAMLv2 XACML PEP 사용자 정의

  • SAMLv2 속성 권한 사용자 정의

  • SAMLv2 속성 쿼리 사용자 정의

  • SAMLv2 인증 권한 사용자 정의

  • SAMLv2 호스트 제휴 사용자 정의
