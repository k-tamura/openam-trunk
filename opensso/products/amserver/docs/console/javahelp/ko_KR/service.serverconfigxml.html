디렉토리 구성

디렉토리 구성 속성은 포함된 디렉토리 저장소의 기본 구성 정보를 정의합니다.

디렉토리 구성

디렉토리 구성 속성은 다음과 같습니다.

최소 연결 풀

LDAP 서버 속성에 지정된 대로 디렉토리 서버에 연결하는 데 사용되는 연결 풀의 최소 크기를 지정합니다. 기본값은 1입니다.

최대 연결 풀

이 속성은 LDAP 서버 속성에 지정된 대로 디렉토리 서버에 연결하는 데 사용되는 연결 풀의 최대 크기를 지정합니다. 기본값은 10입니다.

바인드 DN

LDAP 서버의 바인드 DN을 지정합니다.

바인드 비밀번호

LDAP 서버에 바인드하는 데 사용되는 비밀번호를 정의합니다. 기본적으로 설치하는 동안 입력된 amldapuser 비밀번호가 바인드 사용자로 사용됩니다.

서버

이 속성은 OpenSSO 인스턴스의 구성 데이터 저장소 역할을 할 디렉토리 서버를 정의합니다. 구성 서버를 추가하려면 [추가] 버튼을 누르고 다음 속성의 값을 제공합니다.

이름

서버의 이름을 입력합니다.

호스트 이름

Directory Server의 정규화된 호스트 이름을 지정합니다. 예를 들면 다음과 같습니다.

DirectoryServerHost.domainName.com

포트 번호

Directory Server 포트 번호를 지정합니다.

연결 유형

Directory Server의 연결 유형을 정의합니다. 기본적으로 SIMPLE이 선택되며SSL을 선택할 수도 있습니다.

레거시 구성

다음 속성은 OpenSSO의 레거시 모드 인스턴스에 대한 기본 디렉토리 서버 구성을 정의합니다. 이러한 속성은 레거시 모드 설치에서만 나타납니다.

최소 연결 풀

LDAP 서버 속성에 지정된 대로 디렉토리 서버에 연결하는 데 사용되는 연결 풀의 최소 크기를 지정합니다. 기본값은 1입니다.

최대 연결 풀

이 속성은 LDAP 서버 속성에 지정된 대로 디렉토리 서버에 연결하는 데 사용되는 연결 풀의 최대 크기를 지정합니다. 기본값은 10입니다.

서버

이 속성은 로드 밸런서 프로토콜, 호스트 이름 및 포트를 나열합니다. 예를 들면 다음과 같습니다. http://lb.example.com:80
