자원 아이디 매퍼 구성

com.sun.identity.liberty.ws.interfaces.ResourceIDMapper는 사용자 아이디를 연관된 자원 아이디에 매핑하는 데 사용하는 인터페이스입니다. 구현 클래스를 사용하여 다른 구현을 만들고 이를 속성에 추가할 수 있습니다. 다음은 새 자원 아이디 매퍼를 검색 서비스에 추가하는 절차입니다. 시작 위치는 웹 서비스 모듈 안에 있는 검색 서비스 구성 화면입니다.

  1. [새로 만들기]를 눌러 새 매퍼 속성을 표시하거나 구성된 매퍼 이름을 눌러 매퍼를 수정합니다.
  2. 다음 정보를 기반으로 매퍼 속성에 대한 값을 제공합니다.
    공급자 아이디

    이 속성은 검색 서비스를 가리키는 URI를 값으로 가집니다. 이 값은 http://host: port/amserver/Liberty/disco 형식으로 기록됩니다.

    아이디 매퍼

    구현 클래스의 이름을 입력합니다.

  3. [확인]을 눌러 매퍼 구성을 완료합니다.
  4. 검색 서비스 페이지에서 [저장]을 눌러 서비스 구성을 완료합니다.
