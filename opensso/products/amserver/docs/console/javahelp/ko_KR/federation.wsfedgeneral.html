WS-FED 일반 속성

다음 속성은 Identity 및 서비스 공급자 유형에서 공통으로 사용됩니다.

SP 표시 이름

이 속성은 WS-FED 서비스 공급자의 이름을 정의하며기본값은 생성 시 지정된 메타 별칭입니다.

IDP 표시 이름

이 속성은 WS-FED Identity 공급자의 이름을 정의하며기본값은 생성 시 지정된 메타 별칭입니다.

영역

공급자가 속하는 영역을 표시합니다.

토큰 발급자 이름

Identity 또는 서비스 공급자의 고유 식별자를 정의합니다.

토큰 발급자 엔드포인트

Identity 또는 서비스 공급자가 WS-FED 서비스를 제공 중인 URL을 지정합니다. 예를 들면 다음과 같습니다.

https://demo.example.com/opensso/WSFederationServlet/metaAlias/example
