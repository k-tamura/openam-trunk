공급자 엔티티에 Identity 공급자 추가

Identity 공급자를 추가하는 과정에서 첫 번째 단계는 추가할 Identity 공급자의 유형을 정의하는 것입니다.

  • 호스트 공급자는 Access Manager와 동일한 서버에서 호스트됩니다.

  • 원격 공급자는 Access Manager와 동일한 서버에서 호스트되지 않습니다.

다음 절차에서는 이 초기 단계에 대해 설명합니다. 시작 위치는 연합 아래의 엔티티 화면입니다.

  1. 구성된 공급자 엔티티의 이름을 선택하여 해당 프로필을 수정합니다.

    엔티티 프로필 페이지가 표시됩니다.

  2. [보기] 메뉴에서 Identity 공급자를 선택합니다.
  3. 추가하는 공급자의 유형을 선택합니다.

    • 공급자가 Access Manager와 동일한 서버에서 호스트되는 경우 [새 호스트 공급자]를 선택합니다.

    • 공급자가 Access Manager와 동일한 서버에서 호스트되지 않는 경우 [새 원격 공급자]를 선택합니다.

    해당 Identity 공급자 프로필 페이지가 표시됩니다.

  4. 이전 단계에서 선택한 사항에 기초하여 다음 중 하나를 선택합니다.
    • 공급자 엔티티에 새 호스트 Identity 공급자 추가
    • 공급자 엔티티에 새 원격 Identity 공급자 추가
