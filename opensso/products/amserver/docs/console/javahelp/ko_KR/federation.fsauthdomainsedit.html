CoT(Circle of Trust) 프로필 수정

다음 절차에 따라 기존 CoT(Circle of Trust)의 구성된 일반 속성을 편집하거나 CoT(Circle of Trust)에 공급자를 추가합니다. 시작 위치는 연합 인터페이스 아래의 CoT(Circle of Trust)입니다.

  1. 구성된 CoT(Circle of Trust)의 이름을 눌러 해당 프로필을 수정하거나 공급자를 추가합니다.

    CoT(Circle of Trust) 편집 페이지가 표시됩니다.

  2. 새 값을 입력하거나 CoT(Circle of Trust)의 일반 속성에 대한 기존 값을 편집합니다.
    이름

    이 속성의 정적 값은 CoT(Circle of Trust)를 작성할 때 제공된 이름입니다.

    설명

    이 속성의 값은 CoT(Circle of Trust)에 대한 설명입니다. 적용되는 경우 이미 입력한 설명을 수정할 수 있습니다.

    IDFF 작성기 서비스 URL

    이 속성은 공통 도메인 쿠키를 작성하는 서비스의 위치를 지정합니다. URL의 형식은 http://common-domain-host:port/common/idffwriter입니다.

    IDFF 판독기 서비스 URL

    이 속성은 공통 도메인 쿠키를 판독하는 서비스의 위치를 지정합니다. URL 형식은 http://common-domain-host:port/common/idffreader입니다.

    SAML2 작성기 URL

    이 속성은 쿠키를 공통 도메인에 기록하는 SAML2 작성기 서비스의 위치를 지정합니다. URL 형식은 http://common-domain-host: port/common/saml2writer입니다.

    SAML2 판독기 URL

    이 속성은 쿠키를 공통 도메인에 기록하는 SAML2 작성기 서비스의 위치를 지정합니다. URL 형식은 http://common-domain-host: port/common/saml2reader입니다.

    상태

    기본 상태는 [활성]입니다. [비활성]을 선택하면 CoT(Circle of Trust) 내에서 통신이 비활성화됩니다.

  3. 하나 이상의 사용 가능한 공급자를 선택하고 [추가] 화살표를 눌러 선택합니다.

    제공된 목록에는 공급자로 작성하고 채운 엔티티의 이름이 포함되어 있습니다. 자세한 내용은 CoT(Circle of Trust)에 공급자 추가를 참조하십시오.

  4. [저장]을 눌러 작업을 완료합니다.
