STS 클라이언트

STS(Security Token Service) 클라이언트 인터페이스를 사용하면 보안 토큰을 가져오기 위해 OpenSSO의 보안 토큰 서비스와 통신하는 클라이언트를 만들고 구성할 수 있습니다. OpenSSO는 다음과 같은 유형의 STS 클라이언트 에이전트를 만드는 메커니즘을 제공합니다.

검색 에이전트

Liberty 기반 보안 토큰을 가져오기 위해 Liberty 검색 서비스와 통신하는 검색 에이전트 클라언트를 구성할 수 있습니다. 이 구성은 검색 클라이언트에서 Liberty 검색 종단점으로 보내는 Liberty 요청을 보호하는 데 필요한 속성을 정의합니다.

보안 토큰 서비스 에이전트

웹 서비스 기반 보안 토큰을 얻기 위해 OpenSSO의 보안 토큰 서비스와 통신하는 STS(Security Token Service) 에이전트를 구성할 수 있습니다. 이 구성은 STS 클라이언트에서 STS 종단점으로 보내는 웹 서비스 신뢰 요청을 보호하는 데 필요한 속성을 정의합니다.

STS 클라이언트 인터페이스에서 다음을 수행할 수 있습니다.

  • 검색 에이전트 속성 구성

  • 보안 토큰 서비스 에이전트 속성 구성
