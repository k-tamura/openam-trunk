导言

Sun OpenSSO 可以阻止在未经授权的情况下访问 Web 服务应用程序和 Web 内容。OpenSSO 集成了身份验证和授权服务、策略代理和身份联合功能，可以为保护您的网络资源提供综合的解决方案。

许多企业基于各个应用程序授予对信息的访问权。例如，某个员工可能需要设立一组用户名和密码以访问公司的保健福利管理网站。这个员工又必须使用不同的用户名和密码来访问会计部门的在线报表。在同一个企业内，客户可以设立一组用户名和密码来访问公司网站的公共区域。对于各个网站或服务，管理员需要将企业用户所输入的内容转换成为服务能够识别的数据格式。各个企业新添加的服务都必须单独进行规定和维护。

OpenSSO 能降低管理成本，并且可以消除与各个应用程序解决方案相关的冗余用户信息。OpenSSO 使管理员能够指定特定的规则或策略，来决定各个用户所能够访问的信息或服务。在应用服务器或 Web 服务器上部署策略代理，用于处理 HTTP 请求并强制执行活动的策略。用户信息和相关的访问策略共同构成了用户的企业身份。OpenSSO 使用户仅使用一个身份便能访问企业中的多个资源。
