通用联合配置

数据存储库 SPI 实现类

此属性指定用于默认数据存储库提供者的实现类。

配置实例 SPI 实现类

此属性指定用于管理配置数据的实现类。

日志程序 SPI 实现类

此属性指定用于日志提供者的实现类。

会话提供者 SPI 实现类

此属性指定用于会话提供者的实现类。

最大允许内容长度

此属性指定在联合服务中使用的 HTTP 请求的最大允许内容长度。任何其内容超过特定的最大内容长度的请求都会被拒绝。

密码解码器 SPI 实现类

此属性指定用于com.sun.identity.saml.xmlsig.PasswordDecoder 接口的实现类，用于解码存储在 XML 签署密钥库中的密码和 SAML 1.x 下基本验证的密码。

签名提供者 SPI 实现类

此属性指定 SAML XML 签名提供者类。默认 SPI 为 com.sun.identity.saml.xmlsig.AMSignatureProvider。

密钥提供者 SPI 实现类

此属性指定 XML 签名密钥提供者类。默认 SPI 为 com.sun.identity.saml.xmlsig.JKSKeyProvider。

检查证书是否存在

如果设置为开，则密钥库中必须存在用于 XML 签名验证的证书。如果设置为关，则不会检查是否存在证书。仅适用于 SMAL1.x。

XML 标准化算法

此属性指定用于生成和验证 SAML XML 签名的 XML 标准化算法。默认值为 http://www.w3.org/2001/10/xml-exc-c14n#。

XML 签名算法

此属性指定用于生成和验证 SAML XML 签名的 XML 签名算法。如果未指定值或值为空，则使用默认值 (http://www.w3.org/2000/09/xmldsig#rsa-sha1)。

XML 变换算法

此属性指定用于生成和验证 SAML XML 签名的变换算法。如果未指定值或值为空，则使用默认值 (http://www.w3.org/2001/10/xml-exc-c14n#)。
