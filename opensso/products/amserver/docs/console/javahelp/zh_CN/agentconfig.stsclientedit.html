STS 客户机

可通过安全令牌服务 (Security Token Service, STS) 客户机界面创建和配置与 OpenSSO 的安全令牌服务通信的客户机，以获取安全令牌。OpenSSO 提供创建以下类型的 STS 客户机代理的机制：

搜索代理

可配置与 Liberty 搜索服务通信的搜索代理客户机，以获取基于 Liberty 的安全令牌。该配置定义保护搜索客户机对 Liberty 搜索端点的 Liberty 请求的属性。

安全令牌服务代理

可配置与 OpenSSO 安全令牌服务通信的安全令牌服务代理，以获取基于 Web 服务的安全令牌。该配置定义保护 STS 客户机对 STS 端点的 Web 服务信任请求的属性。

可在 STS 客户机界面执行以下操作：

  • 配置搜索代理属性

  • 配置安全令牌服务代理属性
