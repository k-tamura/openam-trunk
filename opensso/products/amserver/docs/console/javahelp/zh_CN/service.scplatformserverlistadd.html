创建新的实例名称

命名服务会在初始化时读取此属性。该列表包含单个 Federated Access Manager 配置中的 Federated Access Manager 会话服务器。

  1. 在“实例名称”列表中单击“新建”。
  2. 在服务器字段中输入主机名和端口。
  3. 输入站点名称。

    该值唯一标识服务器。参与负载平衡或故障转移的每个服务器都必须具有唯一的标识符。该值还用于缩短 Cookie 的长度，方法是将服务器 URL 映射到服务器 ID。语法为：

    intance_ID(|site_ID)

  4. 单击“确定” 。

    要编辑实例名称，请在“实例名称”列表中单击一个条目，然后相应地更改其值。

  5. 在“平台服务”主页面中单击“保存”。
