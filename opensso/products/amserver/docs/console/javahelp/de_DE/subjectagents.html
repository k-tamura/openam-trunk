Agenten

Richtlinien-Agenten von Federated Access Manager schützen den Inhalt von Webservern und Web-Proxy-Servern vor unbefugtem Eindringen. Sie steuern den Zugriff auf Dienste und Webressourcen basierend auf den von einem Administrator konfigurierten Richtlinien.

Das Agenten-Objekt definiert ein Richtlinien-Agentenprofil und lässt zu, dass Federated Access Manager Authentifizierungs- und andere Profilinformationen zu einem bestimmten Agenten speichert, der eine Federated Access Manager-Ressource schützt. Über die Federated Access Manager-Konsole können Administratoren Agentenprofile anzeigen, erstellen, ändern und löschen.

Weitere Informationen finden Sie unter So erstellen oder ändern Sie einen Agenten
