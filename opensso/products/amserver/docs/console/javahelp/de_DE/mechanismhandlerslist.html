Mechanismus-Handler-Liste

In der Mechanismus-Handler-Liste werden Informationen über den vom Authentifizierungs-Webdienst unterstützten SASL-Mechanismus gespeichert. It displays entries that contain key/value pairs separated by a pipe as in: key=PLAIN|class=com.sun.identity.liberty.ws.authnsvc.mechanism.PlainMechanismHandler . Die Parameter werden in der folgenden Tabelle erläutert.

Parameter

Wert

key

Der erforderliche key-Parameter definiert den vom Authentifizierungs-Webdienst unterstützten SASL-Mechanismus.

class

Der erforderliche class-Parameter gibt den Namen der Implementierungsklasse für den SASL-Mechanismus an. Die Authentifizierungs-Webdienst-Schicht bietet eine Handler-Schnittstelle, die implementiert werden muss, damit jeder SASL-Mechanismus die angeforderte Meldung verarbeiten und eine Antwort zurückgeben kann.

Zum Hinzufügen eines neuen Request-Handlers klicken Sie auf Hinzufügen. Fahren Sie danach mit So konfigurieren Sie einen neuen Mechanismus-Handler fort.
