Erstellen eines neuen Access Manager-Repository-Plug-Ins

In diesem Abschnitt werden die Schritte zum Erstellen eines Repository-Plug-Ins aufgelistet.

So erstellen Sie ein neues Access Manager-Repository-Plug-In

  1. Wählen Sie den Bereich aus, in dem Sie das Access Manager-Repository-Plug-In konfigurieren möchten.

  2. Klicken Sie auf die Registerkarte "Data Stores".

  3. Klicken Sie in der Liste der Data Stores auf "Neu".

  4. Geben Sie einen Namen für das Repository-Plug-In ein.

  5. Wählen Sie "Plugin für Access Manager-Repository" aus.

  6. Klicken Sie auf "Weiter".

  7. Füllen Sie die folgenden Felder aus:

    Access Manager-Plugin-Klassenname

    Gibt den Speicherort der Klassendatei an, die das Access Manager-Repository-Plug-In implementiert.

    Access Manager-Organisation

    Gibt den DN einer Organisation auf dem Directory Server an, die von Access Manager verwaltet werden soll. Dieser DN wird als Basis-DN für alle in diesem Data Store ausgeführten Vorgänge verwendet.

  8. Klicken Sie auf "Fertig stellen"
