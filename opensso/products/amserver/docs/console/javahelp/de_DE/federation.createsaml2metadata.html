Entity-Anbieter

Ein Entity-Anbieter verwaltet die Metadaten für einzelne Typen von Entity-Anbietern. OpenSSO ermöglicht es Ihnen, drei Typen von Entity-Anbietern auf der Grundlage von Liberty-Protokollen zu erstellen: SAMLv2, ID-FF und WS-FED. Für jeden Entity-Anbietertyp können Sie eine bestimmte Anzahl an Entity-Anbieterrollen zuweisen und anpassen, die mithilfe von Attributen der OpenSSO-Konsole spezifische Funktionen in der Verbindungskommunikation erfüllen.

In den folgenden Abschnitten finden Sie Beschreibungen für Entity-Anbietertypen und Entity-Anbieterrollen, die Sie zuweisen können:
