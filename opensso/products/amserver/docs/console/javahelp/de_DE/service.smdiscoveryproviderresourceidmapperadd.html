So konfigurieren Sie einen ResourceID-Mapper

com.sun.identity.liberty.ws.interfaces.ResourceIDMapper is an interface used to map a user identifier to the resource identifier associated with it. Mithilfe der Implementierungsklasse können Sie weitere Implementierungen entwickeln und dem Attribut hinzufügen. Mit dem folgenden Verfahren fügen Sie dem Discovery-Dienst einen neuen ResourceID-Mapper hinzu. Der Discovery-Dienst-Konfigurationsbildschirm im Modul "Webdienste" bildet den Ausgangspunkt.

  1. Klicken Sie auf "Neu", um die neuen Mapper-Attribute anzuzeigen, bzw. auf den Namen eines konfigurierten Mappers, um ihn zu ändern.
  2. Legen Sie die Mapper-Attributwerte entsprechend der folgenden Informationen fest:
    Provider ID

    Dieses Attribut akzeptiert als Wert eine URI, die auf den Discovery-Dienst verweist. The value is written in the format: http:// host:port/amserver/Liberty/disco

    ID-Mapper

    Geben Sie einen Namen für die Implementierungsklasse ein.

  3. Klicken Sie auf "OK", um die Mapper-Konfiguration abzuschließen.
  4. Klicken Sie auf der Discovery-Dienst-Seite auf "Speichern", um die Konfiguration des Dienstes abzuschließen.
