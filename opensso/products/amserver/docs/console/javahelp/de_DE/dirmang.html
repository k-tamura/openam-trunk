Verzeichnisverwaltung

Die Registerkarte "Verzeichnisverwaltung" wird nur angezeigt, wenn OpenSSO im Legacy-Modus installiert wird. Diese Verzeichnisverwaltungsfunktion bietet eine Identitätsverwaltungslösung für Sun Java System Directory Server-fähige OpenSSO-Implementierungen.

Weitere Informationen zu den Installationsoptionen im Legacy-Modus finden Sie im Sun OpenSSO Installation and Configuration Guide.
