So fügen Sie einer Richtlinie eine Bedingung hinzu.
  1. Wenn Sie die Richtlinie bereits erstellt haben, klicken Sie auf den Namen der Richtlinie, für die Sie die Bedingung hinzufügen möchten. Wenn die Richtlinie noch nicht erstellt wurde, ziehen Sie So erstellen Sie eine neue normale Richtlinie zurate.
  2. Klicken Sie unter der Liste "Bedingungen" auf "Neu".
  3. Wählen Sie einen Bedingungstyp aus und klicken Sie auf "Weiter".
  4. Legen Sie die Werte in den Feldern des Bedingungstyps fest. Eine Beschreibung der jeweiligen Bedingungstypen finden Sie unter den folgenden Themen:

    • Aktive Sitzungszeit

    • Authentifizierung durch Modulkette

    • Authentifizierung durch Modulinstanz

    • Authentifizierungsebene (größer als oder gleich)

    • Authentifizierungsebene (kleiner als oder gleich)

    • Authentifizierung des Bereichs

    • Aktuelle Sitzungseigenschaften

    • Identity-Mitgliedschaft

    • IP-Adresse/DNS-Name

    • LDAP-Filterbedingung

    • Zeit (Tag, Datum, Uhrzeit und Zeitzone)

  5. Klicken Sie auf "Fertig stellen".
