Dienste

In OpenSSO versteht man unter einem Dienst eine Gruppe von Attributen, die gemeinsam von der OpenSSO-Konsole verwaltet werden. Bei diesen Attributen kann es sich lediglich um einige verwandte Informationen handeln, beispielsweise um den Namen, Titel und die E-Mail-Adresse eines Mitarbeiters. In der Regel werden Attribute jedoch als Konfigurationsparameter für Softwaremodule, beispielsweise für eine E-Mail-Anwendung oder einen Buchhaltungsdienst, verwendet.

Auf der Registerkarte "Dienste" können Sie einem Bereich verschiedene OpenSSO-Standarddienste hinzufügen und diese konfigurieren. Sie können folgende Dienste hinzufügen:

  • Administration

  • Richtlinienkonfiguration

  • Discovery-Dienst

  • Globalization Settings

  • Passwortzurücksetzung

  • Session

  • Benutzer
