Authentifizierung

Der allgemeine Authentifizierungsdienst muss für einen Bereich als Dienst registriert werden, bevor sich ein Benutzer über die anderen Authentifizierungsmodultypen anmelden kann. Über den Core-Authentifizierungsdienst kann der OpenSSO-Administrator Standardwerte für die Authentifizierungsparameter eines Bereichs festlegen. Diese Werte werden verwendet, wenn keine vorrangigen Werte in den betreffenden Authentifizierungsmodulen angegeben sind. Die Standardwerte des Core-Authentifizierungsdienstes werden nach der Installation in der Datei amAuth.xml festgelegt und in Directory Server gespeichert.

Es gibt folgende Authentifizierungseigenschaften:

Erweiterte Eigenschaften

Über diese Schaltfläche zeigen Sie die Core-Authentifizierungsattribute an, die für den Bereich festgelegt werden können. Die globalen Attribute sind auf einen Bereich nicht anwendbar und werden daher nicht angezeigt.

Standard-Authentifizierungskette

Legt die Authentifizierungskette für den Bereich fest. Jede Kette, die Sie im Attribut Authentifizierungsverkettung erstellen, wird in der Pulldown-Liste angezeigt und kann ausgewählt werden. Weitere Informationen finden Sie unter So erstellen Sie eine neue Authentifizierungskette.

Administrator-Authentifizierungskette

Definiert die Authentifizierungskette ausschließlich für Administratoren. Dieses Attribut kann verwendet werden, wenn für Administratoren eine andere Authentifizierungskette verwendet werden soll als für Endbenutzer. Die in diesem Attribut konfigurierten Module werden beim Zugriff auf die OpenSSO-Konsole übernommen.

Standarderfolgs-URL

Gibt den URL an, an den der Benutzer nach erfolgreicher Authentifizierung beim Bereich weitergeleitet wird.

Modulinstanzen

Definiert und konfiguriert die Standard-Authentifizierungsmodule von OpenSSO für den Bereich. Sie können auch mehrere Instanzen des gleichen Authentifizierungsmoduls hinzufügen. Weitere Informationen finden Sie unter So erstellen Sie eine neue Authentifizierungsmodulinstanz.

Authentifizierungsverkettung

Sie können ein oder mehrere Authentifizierungsmodule konfigurieren, denen ein Benutzer seine Anmeldeinformationen vorlegen muss. Dieser Vorgang wird als Authentifizierungsverkettung bezeichnet. Mit diesem Attribut können Sie die Optionen für die Authentifizierungsverkettung für den Bereich definieren. Die von Ihnen erstellten Ketten werden in der Liste Standard-Authentifizierungskette angezeigt. Weitere Informationen finden Sie unter So erstellen Sie eine neue Authentifizierungskette.
