Globale Eigenschaften

Die globalen Eigenschaften enthalten Dienste, mit denen Sie die Funktion zum Zurücksetzen von Passwörtern und die Richtlinienkonfiguration für OpenSSO definieren können. Folgende Dienste können konfiguriert werden:

  • Gemeinsame Verbindungskonfiguration

  • Liberty ID-FF Service-Konfiguration

  • Liberty ID-WSF Security Service

  • Liberty-Interaktionsdienst

  • Protokoll für Mehrfachverbindungen

  • Passwortzurücksetzung

  • Richtlinienkonfiguration

  • SAMLv2-Dienstkonfiguration

  • SAMLv2 SOAP Binding

  • Sicherheitstoken-Dienst

  • Sitzung

  • So fügen Sie eine Subkonfiguration hinzu
