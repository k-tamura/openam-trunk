So erstellen Sie einen neuen Data Store

  1. Wählen Sie den Bereich aus, dem Sie einen neuen Data Store hinzufügen möchten.
  2. Klicken Sie auf die Registerkarte "Data Stores".
  3. Klicken Sie in der Liste der Data Stores auf "Neu".
  4. Geben Sie einen Namen für den Data Store ein.
  5. Wählen Sie die Art von Data Store aus, die Sie erstellen möchten.
  6. Klicken Sie auf "Weiter".
  7. Konfigurieren Sie den Data Store, indem Sie die entsprechenden Attributwerte eingeben. Links zu den Attributdefinitionen der einzelnen Data Store-Typen finden Sie unter OpenSSO-Data Store-Typen.
  8. Klicken Sie auf "Fertig stellen".
