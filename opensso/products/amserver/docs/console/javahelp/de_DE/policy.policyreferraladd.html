So erstellen Sie eine neue Bezugsrichtlinie
  1. Wählen Sie den Bereich aus, für den Sie die Richtlinie erstellen möchten.
  2. Klicken Sie in der Liste "Richtlinien" auf "Neuer Bezug".
  3. Geben Sie einen Namen und eine Beschreibung für die Richtlinie ein.
  4. Wenn Sie die Richtlinie aktivieren möchten, wählen Sie für das Attribut "Aktiv" die Option "Ja".
  5. Es ist nicht erforderlich, zu diesem Zeitpunkt alle Felder für normale Richtlinien zu definieren. Sie können die Richtlinie erstellen und später Regeln und Bezüge hinzufügen. Falls Sie diese Felder jetzt festlegen möchten, lesen Sie die folgenden Themen:

    • So fügen Sie einer Richtlinie eine Regel hinzu.

    • So fügen Sie einer Richtlinie Bezüge hinzu

  6. Klicken Sie auf "Erstellen".
