So bearbeiten Sie eine Site-Instanz

  1. Klicken Sie in der Site-Liste auf den Namen der zu bearbeitenden Site.
  2. Die Primär-URL für die Site wird im Primär-URL-Attribut angegeben.
  3. Fügen Sie bei Bedarf eine Sekundär-URL hinzu.

    Die Sekundär-URL stellt die Verbindungsinformationen für das Sitzungs-Repository bereit, das in OpenSSO für die Sitzungs-Failover-Funktion verwendet wird. Als Bezeichner für die sekundäre Konfiguration sollte der URL des Lastenausgleichs angegeben werden. Wenn die sekundäre Konfiguration auf diese Weise angegeben ist, wird die Sitzungs-Failover-Funktion automatisch aktiviert und nach dem Server-Neustart wirksam.

  4. Klicken Sie auf Speichern.
