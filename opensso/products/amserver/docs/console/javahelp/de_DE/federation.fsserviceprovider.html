So fügen Sie einer Anbieter-Entity einen Dienstanbieter hinzu

Der erste Schritt beim Hinzufügen eines Dienstanbieters zu einer Anbieter-Entity besteht darin, die Art des Dienstanbieters zu definieren, der hinzugefügt wird.

  • Ein Host-Anbieter befindet sich auf demselben Server wie Access Manager.

  • Ein Remote-Anbieter befindet sich auf einem anderen Server als Access Manager.

Das folgende Verfahren beschreibt diesen ersten Schritt. Der Ausgangspunkt ist "Entities" in der Verbindungsschnittstelle.

  1. Wählen Sie den Namen einer konfigurierten Anbieter-Entity, um deren Profil zu ändern.

    Die Profilseite der Entity wird angezeigt.

  2. Wählen Sie im Menü "Ansicht" die Option "Dienstanbieter" aus.
  3. Wählen Sie den hinzuzufügenden Providertyp aus:

    • Wählen Sie "Neuer Host-Anbieter", wenn sich der Anbieter auf demselben Server wie Federated Access Manager befindet.

    • Wählen Sie "Neuer Remote-Anbieter", wenn sich der Anbieter auf einem anderen Server als Federated Access Manager befindet.

    Die entsprechende Profilseite für Dienstanbieter wird angezeigt.

  4. Wählen Sie abhängig von Ihrer Auswahl im vorangegangenen Schritt eines der folgenden Verfahren aus:
    • So fügen Sie einer Anbieter-Entity einen neuen Host-Dienstanbieter hinzu
    • So fügen Sie einer Anbieter-Entity einen neuen Remote-Dienstanbieter hinzu
