Globale Eigenschaften

Die globalen Eigenschaften enthalten Dienste, mit denen Sie die Funktion zum Zurücksetzen von Passwörtern und die Richtlinienkonfiguration für Federated Access Manager definieren können. Folgende Dienste können konfiguriert werden:

  • Passwortzurücksetzung

  • Richtlinienkonfiguration

  • Sitzung
