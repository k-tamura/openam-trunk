PPLDAP-Attributzuordnungsliste

Für jedes in Persönlicher Liberty-Profildienst definierte Attribut ist ein exakt entsprechendes LDAP-Attribut vorhanden. Der Wert dieses Attributs ist eine Liste mit diesen Zuordnungen. Beispiel: JobTitle=sunIdentityServerPPEmploymentIdentityJobTitle ordnet beispielsweise das Liberty-Attribut JobTitle dem OpenSSO-Attribut sunIdentityServerPPEmploymentIdentityJobTitle zu. Wenn Sie auf einer der beiden Seiten ein neues Attribut hinzufügen, müssen Sie darauf achten, dass Sie die neue Attributzuordnung auch in diesem Attribut konfigurieren. Weitere Informationen finden Sie unter So konfigurieren Sie eine Attributzuordnung.
