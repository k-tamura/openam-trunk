So fügen Sie Benutzer einer Rolle oder Gruppe hinzu
  1. Klicken Sie auf den Namen der Rolle oder Gruppe, der Sie Benutzer hinzufügen möchten.
  2. Klicken Sie auf die Registerkarte "Benutzer".
  3. Wählen Sie die Benutzer, die Sie hinzufügen möchten, aus der Liste der verfügbaren Benutzer aus und klicken Sie auf "Hinzufügen".
  4. Wenn die Benutzer in der Liste der ausgewählten Benutzer angezeigt werden, klicken Sie auf "Speichern".
