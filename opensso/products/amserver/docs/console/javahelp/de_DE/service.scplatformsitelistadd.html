So erstellen Sie einen neuen Site-Namen

  1. Klicken Sie in der Liste der Site-Namen auf "Neu".
  2. Geben Sie den Hostnamen und den Port im Feld ";Server" ein.
  3. Geben Sie den Site-Namen ein.

    Dieser Wert kennzeichnet den Server eindeutig. Jeder Server, der am Lastenausgleich oder am Failover teilnimmt, muss einen eindeutigen, zweistelligen Bezeichner, zum Beispiel 01, haben.

  4. Klicken Sie auf Speichern.

    Zur Bearbeitung eines Site-Namens klicken Sie auf einen Eintrag in der Site-Namen-Liste und ändern Sie die Werte wie gewünscht.
