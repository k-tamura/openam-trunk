ResourceID-Mapper

The value of this attribute specifies the implementation of com.sun.identity.liberty.ws.interfaces.ResourceIDMapper. Sie können eine neue Implementierung entwickeln oder die von OpenSSO bereitgestellte Standardimplementierung com.sun.identity.liberty.ws.idpp.plugin.IDPPResourceIDMapper verwenden, die einer Benutzer-ID eine Discovery-Ressource zuordnet.
