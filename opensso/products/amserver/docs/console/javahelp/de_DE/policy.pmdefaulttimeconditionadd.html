Zeit (Tag, Datum, Uhrzeit und Zeitzone)

Legt die Bedingung basierend auf Zeitbeschränkungen fest. Die Felder sind:

Datum von/bis

Legt den Datumsbereich fest.

Uhrzeit

Legt den Zeitbereich innerhalb eines Tags fest.

Tag

Legt den Bereich der Tage fest.

Zeitzone

Legt eine Zeitzone fest, entweder Standard oder benutzerdefiniert. Bei benutzerdefinierten Zeitzonen darf es sich nur um Zeitzonen-IDs handeln, die von Java erkannt werden (z. B. PST). Wurde kein Wert angegeben, entspricht der Standardwert der in OpenSSO JVM festgelegten Zeitzone.
