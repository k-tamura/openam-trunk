So konfigurieren Sie einen Verbindungspool – Beispiel

Das folgende Beispiel zeigt, wie Sie einen Verbindungspool für Web Server und MySQL 4.0 einrichten:

  1. Erstellen Sie auf der Web Server-Konsole einen JDBC-Verbindungspool mit den folgenden Attributen:
    poolName

    samplePool

    DataSource-Klassenname

    com.mysql.jdbc.jdbc2.optional.MysqlDatacSource

    serverName

    Servername des mySQL-Servers

    Port

    Anschlussnummer des mySQL-Servers

    Benutzer

    Benutzername des Datenbankpassworts:

    Passwort

    Das Passwort des Benutzers.

    databaseName

    Name der Datenbank

    Hinweis – Die jar-Datei, die die DataSource-Klasse und die in den nächsten Schritten erwähnte JDBC-Treiber-Klasse enthält, sollte dem Anwendungsklassenpfad hinzugefügt werden.

  2. Konfigurieren Sie die JDBC-Ressourcen. Erstellen Sie auf der Web Server-Konsole eine JDBC-Ressource mit den folgenden Attributen:
    JNDI-Name

    jdbc/samplePool

    Poolname

    samplePool

    Dataenressource aktiviert

    Aktiviert

  3. Fügen Sie der Datei sun-web.xmlder Anwendung folgende Zeilen hinzu:
    <resource-ref>
      <res-ref-name>jdbc/mySQL</res-ref-name>
      <jndi-name>jdbc/samplePool</jndi-name>
</resource-ref>
  4. Fügen Sie der Datei web.xml der Anwendung folgende Zeilen hinzu:
    <resource-ref>
       <description>mySQL Database</description>
       <res-ref-name>jdbc/mySQL</res-ref-name>
       <res-type>javax.sql.DataSource</res-type>
       <res-auth>Container</res-auth>
</resource-ref>
  5. Nachdem Sie diese Einstellungen vorgenommen haben, lautet der Wert dieses Attributs wie folgt: java:comp/env/jdbc/mySQL.
