So fügen Sie Dienste zu einem Benutzer hinzu
  1. Wählen Sie die Identität aus, der Sie Dienste hinzufügen möchten.
  2. Klicken Sie auf die Registerkarte "Dienste".
  3. Klicken Sie auf „Hinzufügen“.
  4. Je nach Typ der ausgewählten Identität werden die folgenden Dienste angezeigt:

    • Authentifizierungskonfiguration

    • Discovery-Dienst

    • Persönlicher Liberty-Profildienst

    • Sitzung

    • Benutzer

  5. Wählen Sie den Dienst aus, den Sie hinzufügen möchten, und klicken Sie auf "Weiter".
  6. Bearbeiten Sie die Attribute des Dienstes. Eine Beschreibung der Dienstattribute finden Sie unter dem Namen des betreffenden Dienstes in Schritt 4.
  7. Klicken Sie auf "Fertig stellen".
