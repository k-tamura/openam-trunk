So konfigurieren Sie eine Attributzuordnung

Eine Zuordnung ist ein Attribut, das den Eigentümer für bestimmte Identitätsdaten festlegt. Im Folgenden wird die Vorgehensweise zum Hinzufügen neuer Attribute in Persönlicher Liberty-Profildienst beschrieben. Der Persönlicher Liberty-Profildienst-Konfigurationsbildschirm im Modul "Webdienste" bildet den Ausgangspunkt.

  1. Klicken Sie auf "Neu", um die neuen Attributzuordnungs-Attribute anzuzeigen, bzw. auf den Namen einer konfigurierten Zuordnung, um sie zu ändern.
  2. Legen Sie die Werte der Container-Attribute entsprechend der folgenden Informationen fest:
    Namenspräfix

    Geben Sie den Namen des zuzuordnenden Persönlicher Liberty-Profildienst-Identitätsattributs ein.

    LDAP-Attribut

    Geben Sie den Namen des OpenSSO-LDAP-Attributs ein, dem das Namenspräfix zugeordnet ist.

  3. Klicken Sie auf "OK", um die Konfiguration der Zuordnung abzuschließen.
  4. Klicken Sie auf der Persönlicher Liberty-Profildienst-Seite auf "Speichern", um die Konfiguration des Dienstes abzuschließen.
