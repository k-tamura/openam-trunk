命名

“命名”服务用于获取和设置 URL、插件和配置，以及其他各种 OpenSSO 服务（如会话、验证、日志记录、SAML 和联合）的请求通知。

如果平台运行了多个 OpenSSO，该服务可以使客户机找到正确的服务 URL。找到命名 URL 时，命名服务将对用户会话进行解码，并使用会话中的参数动态替换协议、主机和端口。这将确保服务返回的 URL 是创建了用户会话的主机。命名属性包括：

  • 配置服务 URL

  • 会话服务 URL

  • 记录服务 URL

  • 策略服务 URL

  • 验证服务 URL

  • SAML 网络配置文件/附件服务 URL

  • SAML SOAP 服务 URL

  • SAML 网络配置文件/POST 服务 URL

  • SAML 声明管理器服务 URL

  • 联合声明管理器服务 URL

  • 安全令牌管理器 URL

  • JAXRPC 端点 URL

配置服务 URL

该字段采用的值为：

%protocol://%host:%port/Server_DEPLOY_URI/profileservice

该语法允许基于特定会话参数动态替换配置 URL。
