Web 服务

Web 服务模块是自由联盟计划身份认证 Web 服务框架 (Identity Web Services Framework, ID-WSF) 规范的 OpenSSO 实现。ID-WSF 基于自由身份联合框架 (Identity Federation Framework, ID-FF) 构建，这是通过提供对联合网络环境中的身份数据进行检索、更新或执行操作的 Web 服务构建规范来实现的。这些规范概述了构建 Web 服务（如日历服务、 钱包服务或警报服务）所需的技术组件。

注 – 身份数据包括但不限于用于定义名、姓、家庭地址、办公地址和紧急情况联系人的属性。

Web 服务是 Internet 中分布式计算的基础。Web 服务使用方 (WSC) 通过发出请求来查找 Web 服务并调用其提供的操作。Web 服务提供者 (web service provider, WSP) 是根据请求来实现 Web 服务的应用程序；它可以与 WSC 位于同一台 Java™ 虚拟机上，也可以位于数千英里以外。当 WSC 需要从 WSP 检索身份属性时，首先必须联系搜索服务以查找特定属性存储的位置。此信息返回之后，WSC 将联系 WSP（例如，个人配置文件服务）以检索所需的属性。

OpenSSO 控制台中的 Web 服务模块提供用于配置以下各项的接口：

  • Liberty 个人配置文件服务

  • 搜索服务

  • SOAP 绑定服务

  • 验证 Web 服务
