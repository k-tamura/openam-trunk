搜索代理属性

“搜索代理”配置文件保留信任授权机构配置，Web 服务客户机/配置文件使用该配置与 Liberty 搜索服务进行通信，以进行 Web 服务查找、注册和获取安全凭证。

组

可通过“组”机制来定义相似代理类型的集合。组必须在将特定代理纳入集合之前进行定义。

密码

定义“搜索代理”的密码。

密码确认

确认密码。

状态

定义代理将在系统中处于“活动”状态还是“非活动”状态。默认情况下，该属性会设置为“活动”，即代理会参与保护 Web 服务客户机的出站 Web 服务请求，并验证 Web 服务提供者的 Web 服务响应。

代理配置系统信息库的位置

此属性定义搜索代理配置系统信息库的代理位置。

私钥别名

该属性定义用于签署 Web 服务请求（或者解密 Web 服务响应）的私有证书密钥别名。

搜索服务端点

此属性定义信任授权机构客户机为注册和查找服务而在其中建立通信的搜索服务端点。

验证 Web 服务端点

此属性定义验证服务端点， Web 服务客户机使用该端点进行验证，方法是使用最终用户的 SSOToken 来接收搜索服务资源提供（也称为引导资源提供）。
