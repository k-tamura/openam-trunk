配置支持的容器

容器是用于定义身份数据持有者的属性。以下是向 Liberty 个人配置文件服务添加新属性的过程。起点是 Web 服务模块中的 Liberty 个人配置文件服务配置屏幕。

  1. 单击“新建”以显示新的“支持的容器”属性，或单击已配置的容器名称以修改其配置文件。
  2. 基于以下信息，为容器属性提供值：
    容器名称

    输入容器的名称，如 CreditCard。

    插件

    输入处理整个容器的类名。可用它覆盖默认实现 com.sun.identity.liberty.ws.idpp.plugin.IDPPContainer。

  3. 单击“确定”完成“容器”的配置操作。
  4. 单击“Liberty 个人配置文件服务”页面上的“保存”完成服务的配置操作。
