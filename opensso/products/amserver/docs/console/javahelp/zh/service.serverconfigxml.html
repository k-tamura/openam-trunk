目录配置

“目录配置”属性定义嵌入式目录存储库的基本配置信息：

目录配置

“目录配置”属性包括：

最小连接池

指定用于连接 Directory Server 的连接池的最小尺寸，它与 LDAP 服务器属性中指定的一致。默认端口为 1。

最大连接池

该属性指定用于连接 Directory Server 的连接池的最大尺寸，它与 LDAP 服务器属性中指定的一致。默认值为 10。

绑定 DN

指定 LDAP 服务器中的绑定 DN。

绑定密码

定义用于绑定 LDAP 服务器的密码。默认情况下，会将安装过程中输入的 amldapuser 密码用作绑定用户。

服务器

该属性定义将用作 OpenSSO 实例的配置数据存储库的目录服务器。要添加配置服务器，可单击“添加”按钮，然后为以下属性赋值：

名称

输入服务器的名称。

主机名

指定 Directory Server 的全限定主机名。例如：

DirectoryServerHost.domainName.com

端口号

指定 Directory Server 的端口号。

连接类型

定义 Directory Server 的连接类型。默认选择 SIMPLE。也可以选择 SSL。

传统配置

以下属性定义用于 OpenSSO 传统模式实例的基本目录服务器配置。这些属性仅在以传统模式安装时才显示出来。

最小连接池

指定用于连接 Directory Server 的连接池的最小尺寸，它与 LDAP 服务器属性中指定的一致。默认端口为 1。

最大连接池

该属性指定用于连接 Directory Server 的连接池的最大尺寸，它与 LDAP 服务器属性中指定的一致。默认值为 10。

服务器

该属性列出负载平衡器协议、主机名和端口。例如：http://lb.example.com:80。
