创建新的 Access Manager 系统信息库插件

本节列出了创建系统信息库插件的步骤。

创建新的 Access Manager 系统信息库插件

  1. 选择要在其中配置 Access Manager 系统信息库插件的领域。

  2. 单击数据存储库选项卡。

  3. 在“数据存储库”列表中单击“新建”。

  4. 输入系统信息库插件的名称。

  5. 选择"Access Manager 系统信息库插件"。

  6. 单击“下一步”。

  7. 定义以下字段：

    Access Manager 插件类名称

    指定实现 Access Manager 系统信息库插件的类文件的位置。

    Access Manager 组织

    指向 Directory Server 中由 Access Manager 管理的某组织的 DN。它将成为在此数据存储库内执行的所有操作的基本 DN。

  8. 单击"完成"。
