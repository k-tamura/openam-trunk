配置目标 URL

可为将要通过 POST 接收声明的每个目标 URL 定义（或修改）以下子属性。

  1. 单击“新建”添加新的目标 URL。

    随即显示“将新的 Post 添加到目标 URL”页面。也可通过选择所定义目标 URL 的“编辑”或“复制”按钮来进入此页面。

  2. 键入属性值。
    协议

    选择 http 或 https。

    服务器名称

    TARGET URL 所在服务器的名称，如 www.sun.com。

    端口

    此属性包含端口号，如 58080。

    路径

    此属性包含 URI，如 /amserver/console。

  3. 单击“确定”完成“目标 URL”的配置操作。
  4. 单击“SAML 配置文件”页面上的“保存”完成 SAML 的配置操作。
