请求处理程序列表

SOAP 绑定服务提供 RequestHandler 接口以处理请求消息和返回响应。此接口必须由每一个使用 SOAP 绑定服务且基于 Liberty 的身份服务在服务器端实现。请求处理程序列表属性存储有关实现请求处理程序的 Web 服务实现类的信息。请求处理程序列表显示的条目包含用竖线分隔的关键字/值对，格式如下：key=disco|class=com.example.identity.liberty.ws.disco.DiscoveryService。这些参数将在以下表格中加以说明。

参数

标头

关键字

关键字参数是必填参数，它是 SOAP 端点 URI 路径的最后一部分。OpenSSO 中的 SOAP 端点是 SOAP 接收方 servlet。SOAP 接收方的 URI 为： protocol://host:port /deploy-uri/Liberty/key。如果您将 disco 定义为关键字，则相应搜索服务的 SOAP 端点 URI 路径应为： protocol://host:port /amserver/Liberty/disco。在与 SOAP 接收方连接时，不同的服务客户机使用的关键字不同。

类

类参数是必填参数，它用于为特定身份服务指定请求处理程序实现类的名称。例如：class=com.example.identity.liberty.ws.disco.DiscoveryService。

要添加新的“请求处理程序”，参见配置请求处理程序。
