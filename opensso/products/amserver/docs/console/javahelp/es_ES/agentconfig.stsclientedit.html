Cliente STS

La interfaz de cliente Servicio de token de seguridad (STS, Security Token Service) le permite crear y configurar un cliente que se comunique con el servicio de token de seguridad de OpenSSO para obtener un token de seguridad. OpenSSO proporciona el mecanismo necesario para crear los siguientes tipos de agentes de cliente STS:

Discovery Agent

Le permite configurar un Cliente agente de detección que se comunique con el Servicio de detección de Liberty para obtener un token de seguridad basado en Liberty. La configuración define los atributos a la hora de asegurar solicitudes Liberty desde el cliente de detección hasta el punto final de detección de Liberty.

Agente del Servicio de token de seguridad

Le permite configurar un agente del servicio de token de seguridad que se comunique con el servicio de token de seguridad de OpenSSO para obtener tokens de seguridad basados en servicios web. Esta configuración define los atributos de protección de solicitudes fiables de servicio web desde el cliente STS hasta el punto final STS.

En la interfaz de cliente STS puede efectuar las siguientes acciones:

  • Configurar Atributos del Agente de detección

  • Configurar los Atributos del agente del servicio de token de seguridad
