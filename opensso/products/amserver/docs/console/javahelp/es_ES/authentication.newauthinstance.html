Para crear una nueva instancia de módulo de autenticación

  1. Haga clic en el nombre del dominio para el que desea agregar una nueva instancia de módulo de autenticación.
  2. Seleccione la ficha Autenticación.
  3. Haga clic en Nuevo en la lista Module Instances (Instancias del módulo).
  4. Introduzca un nombre para la instancia del módulo de autenticación. Los nombres deben ser exclusivos.
  5. Seleccione el tipo de módulo de autenticación para el dominio.
  6. Haga clic en Aceptar.
  7. Haga clic en el nombre de la instancia recién creada y edite las propiedades del módulo. Consulte los atributos del módulo de autenticación para obtener las definiciones de propiedades de cada tipo de módulo.
  8. Repita estos pasos para agregar varias instancias de módulos.
