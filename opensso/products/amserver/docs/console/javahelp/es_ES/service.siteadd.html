Para crear una nueva instancia del sitio

  1. Haga clic en Nuevo en la lista de sitios.
  2. Introduzca el nombre de host y el puerto en el campo Server (Servidor).
  3. Escriba el Nombre del sitio.

    Este valor identifica de forma exclusiva al servidor, y permite la posibilidad de especificar un segundo punto de entrada (además de la URL principal) al sitio. También se utiliza para acortar la longitud de la cookie mediante la asignación de la URL del servidor al Id. del servidor.

  4. Introduzca la URL principal de la instancia del sitio, incluyendo el URI del sitio.
  5. Cliquez sur Enregistrer.

    El sitio creado aparecerá en la lista de sitios con el formato correcto.
