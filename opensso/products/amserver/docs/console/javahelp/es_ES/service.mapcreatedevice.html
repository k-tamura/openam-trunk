Para agregar un cliente nuevo

  1. Haga clic en Nuevo en la lista de tipos de cliente.
  2. Seleccione el tipo de dispositivo con los siguientes campos:
    Estilo

    Muestra el estilo base del dispositivo. Por ejemplo, HTML.

    Agente de usuario de dispositivo

    Admite el nombre para el dispositivo.

  3. Haga clic en Siguiente.
  4. Introduzca la siguiente información para el nuevo dispositivo:
    Nombre de tipo de cliente

    Admite el nombre para el dispositivo. Debe ser exclusivo en todos los dispositivos

    La cadena de usuario HTTP

    Define el agente-usuario en el encabezado de solicitud HTTP. Por ejemplo, Mozilla/4.0.

  5. Haga clic en Finalizar (Finish).
  6. Para duplicar un dispositivo y sus propiedades, haga clic en el vínculo Duplicate (Duplicar). Los nombres de dispositivo deben ser exclusivos. De forma predeterminada, OpenSSO cambiará el nombre del dispositivo a copy_of_devicename.
