Personalización de SAMLv2 XACML PEP

XACML PEP contiene los siguientes atributos que pueden personalizarse:

  • Enumeración de la compatibilidad del protocolo

  • Alias de la clave firmada

  • Alias de la clave de cifrado

  • Autorización básica

  • Respuesta sobre la decisión de autorización firmada

  • Aserción cifrada

Enumeración de la compatibilidad del protocolo

Muestra la versión de XACML PEP compatible con este proveedor.

Alias de la clave firmada

Define el alias de clave que se utiliza para firmar solicitudes y respuestas.

Alias de la clave de cifrado

Define el alias de clave para el cifrado XACML.

Autorización básica

Para proteger los puntos finales SOAP puede habilitarse la autorización básica. Todo proveedor que acceda a estos puntos finales deberá tener el usuario y la contraseña definidos en las siguientes dos propiedades: Nombre de usuario y Contraseña.

Respuesta sobre la decisión de autorización firmada

Cuando está habilitado, este atributo obliga que todas las respuestas se firmen para la decisión de autorización XACML.

Aserción cifrada

Cuando está habilitado, este atributo fuerza que todas las aserciones se cifren.
