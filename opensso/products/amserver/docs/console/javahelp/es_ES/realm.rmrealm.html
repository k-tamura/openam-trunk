Dominios

Un dominio de control de acceso es un grupo de propiedades de autenticación y políticas de autorización que pueden asociarse a un usuario o grupo de usuarios. Los datos del dominio se almacenan en un árbol de información comercial creado por OpenSSO en el almacén de datos que especifique. La estructura de OpenSSO agrega las directivas y propiedades incluidas en cada dominio del árbol de información de OpenSSO. De manera predeterminada, OpenSSO 8.0 inserta automáticamente el árbol de información de OpenSSO en forma de rama especial en Sun Java Enterprise System Directory Server, separado de los datos de usuarios. Puede utilizar los dominios de control de acceso junto con una base de datos LDAPv3.
