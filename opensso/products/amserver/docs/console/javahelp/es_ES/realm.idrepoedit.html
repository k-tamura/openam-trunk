Configuración de un almacén de datos

Una vez creado el almacén de datos, puede editar los valores de configuración en cualquier momento. Para hacerlo, haga clic en el nombre del almacén de datos que desee editar. Consulte los apartados siguientes para acceder a las definiciones de atributo de cada tipo de almacén de datos:

  • Atributos de Active Directory

  • Atributos LDAPv3 genéricos

  • Sun Directory Server con los atributos de esquema de OpenSSO
