Mecanismos de autenticación admitidos

Este atributo especifica los métodos de autenticación admitidos por el Servicio de detección. Estos mecanismos de seguridad hacen referencia a la forma en la que el consumidor de servicios Web se autentica al proveedor de servicios Web o proporciona seguridad en los mensajes. De forma predeterminada están seleccionados todos los métodos disponibles que admite la instancia de servicios. Si no se selecciona un método de autenticación y un consumidor de servicios Web envía una solicitud mediante este método, la solicitud será rechazada. Consulte Para configurar una descripción de servicio.
