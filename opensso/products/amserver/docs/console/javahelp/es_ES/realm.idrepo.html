Almacenes de datos

Un almacén de datos es una base de datos en la que se pueden almacenar atributos de usuarios y datos de configuración de usuario.

OpenSSO proporciona complementos de repositorios de identidades que se conectan a una estructura de repositorio de identidades LDAPv3. Esta nueva función permite ver y recuperar la información de usuario de OpenSSO sin necesidad de realizar cambios en la base de datos de usuarios existente. La estructura de OpenSSO integra los datos del complemento de repositorio de identidades con los datos de otros complementos de OpenSSO para formar una identidad virtual para cada usuario. OpenSSO puede utilizar la identidad universal en los procesos de autenticación y autorización en varios repositorios de identidades. La identidad de usuarios virtual se destruye al finalizar la sesión del usuario.

OpenSSO admite los siguientes almacenes de datos:

  • Active Directory

  • LDAPv3 genérico

  • Sun Directory Server con esquema OpenSSO

Consulte Para crear un nuevo almacén de datos para obtener información sobre cómo crear un nuevo almacén de datos y configurar los complementos de repositorios.
