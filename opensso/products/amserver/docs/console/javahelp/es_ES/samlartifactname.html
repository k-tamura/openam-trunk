Nombre de artefacto SAML

Este atributo asigna un nombre de variable a un artefacto SAML. El artefacto consiste en información de tamaño limitado que identifica una aserción y un sitio de origen. Se toma como parte de una cadena de consultas URL y se transmite redirigiéndolo al sitio de destino. El nombre predeterminado es SAMLart . Si se utiliza el nombre predeterminado SAMLart, la cadena de consulta de redirección puede ser http://host:port /deploy-URI/ SamlAwareServlet?TARGET=target-URL /&;SAMLart=artifact123.
