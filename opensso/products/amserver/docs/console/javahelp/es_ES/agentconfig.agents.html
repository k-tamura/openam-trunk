Agentes

La configuración centralizada de agentes proporciona a los administradores de agentes un método para administrar varias configuraciones de agentes desde una ubicación central. Las configuraciones de agentes se almacenan en el repositorio de datos de OpenSSO y las gestiona un administrador mediante la consola de OpenSSO. Cualquier cambio realizado en las configuraciones de agentes se comunicará a los agentes afectados y éstos reaccionarán consecuentemente en función de la naturaleza de las propiedades actualizadas. Los perfiles de agentes específicos que puede crear son:

  • Creación de nuevos grupos y agentes

  • Agente de directivas web

  • Agente de directivas J2EE

  • Proveedor de servicios web

  • Atributos de clientes de servicios web

  • Cliente STS

  • Atributos del Agente de detección

  • Atributos del agente del servicio de token de seguridad

  • Agente de directivas 2.2
