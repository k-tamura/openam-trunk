Servicios

En OpenSSO, un servicio es un grupo de atributos administrados conjuntamente por la consola de OpenSSO. Los atributos pueden ser bits de información relacionada como, por ejemplo, el nombre de un empleado, su puesto y su dirección de correo electrónico, aunque se utilizan normalmente como parámetros de configuración de un módulo de software como, por ejemplo, una aplicación de software o un servicio de pago de nóminas.

Mediante la ficha Servicios, puede agregar y configurar una serie de servicios predeterminados de OpenSSO para un dominio. Puede agregar los siguientes servicios:

  • Administración

  • Configuración de políticas

  • Discovery Service

  • Globalization Settings

  • Reinicialización de contraseña

  • Session

  • Usuario
