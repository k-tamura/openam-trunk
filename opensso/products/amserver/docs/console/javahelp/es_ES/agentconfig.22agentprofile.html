Agente de directivas 2.2

OpenSSO proporciona compatibilidad con versiones anteriores de Policy Agent 2.2. Esta aplicación debe configurarse de forma local desde el contenedor de implementación en el que se ha instalado. Por lo tanto, sólo se puede configurar un número muy limitado de opciones de Policy Agent 2.2 desde la consola de OpenSSO.

Mot de passe

La contraseña que se ha definido al crear el perfil de agente. No obstante, puede cambiar esta contraseña en cualquier momento.

Confirmar contraseña

La confirmación de la contraseña se ha realizado al crear el perfil de agente. Si cambia la contraseña, debe confirmar el cambio.

Estado

La opción Activo se selecciona al crear el agente. Seleccione Inactivo sólo si desea suprimir la protección que proporciona el agente.

Descripción

Una descripción del agente, que puede agregar si lo desea.

Valor de clave de agente

Ajuste necesario al habilitar CDSSO y durante la configuración de la implementación para evitar secuestros de cookies.

Este atributo sirve como clave a la hora de emparejar una clave y un valor. OpenSSO utiliza este atributo para recibir las solicitudes del agente para las aserciones de credenciales acerca de los usuarios. En esta pareja clave-valor sólo uno de los atributos es válido. El resto de atributos se ignora. Use el siguiente formato:

agentRootURL=protocol://hostname:port/

La entrada debe ser precisa. Por ejemplo, el ajuste que representa la clave, agentRootURL, distingue entre mayúsculas y minúsculas.
