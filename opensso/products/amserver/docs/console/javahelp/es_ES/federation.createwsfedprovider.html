Entidad de proveedor WS-FED

El tipo de proveedor de entidades WS-FED está basado en el protocolo de Federación WS. La implementación de este protocolo permite el inicio de sesión único entre OpenSSO y el Servicio de federación de Microsoft Active Directory. La entidad proveedor WS-FED le permite asignar y configurar las siguientes funciones:

  • Proveedor de identidades

  • Proveedor de servicios

Temas y tareas asociados:

  • Para crear un proveedor de entidades WS-FED

  • Atributos generales WS-FED

  • Personalización de proveedores de identidad WS-FED

  • Personalización del proveedor de servicios WS-FED
