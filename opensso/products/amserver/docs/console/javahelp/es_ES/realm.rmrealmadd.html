Para crear un nuevo dominio

  1. Seleccione Nuevo de la lista de dominios en la ficha Control de acceso.
  2. Defina los siguientes atributos generales:
    Nombre

    Introduzca un nombre para el dominio.

    Principal

    Defina la ubicación del dominio que está creando. Seleccione el dominio principal bajo el que se creará el nuevo dominio.

  3. Defina los siguientes atributos de dominio:
    Estado de dominio

    Seleccione el estado activo o inactivo. El valor predeterminado es activo. Puede cambiar el estado en cualquier momento a lo largo de la duración del dominio. Para ello, seleccione el icono Properties (Propiedades). Si selecciona el estado inactivo, se deshabilita el acceso de usuario al iniciar la sesión.

    Alias DNS/dominio

    Permite agregar nombres de alias de DNS para el dominio. Este atributo sólo acepta nombres de dominio "reales" (no se permiten cadenas aleatorias).

  4. Haga clic en Aceptar para guardar el dominio o Cancelar para volver a la página anterior.
