Para configurar una URL de destino

Los subatributos siguientes se pueden definir (o modificar) para cada URL de destino que vaya a recibir aserciones mediante POST (envío).

  1. Haga clic en Nuevo para agregar una URL de destino nueva.

    Se muestra la página Agregar nuevo envío a la URL de destino. También puede acceder a esta página seleccionado el botón Editar o Duplicar de una URL de destino definida.

  2. Escriba los valores para los atributos
    Protocolo

    Seleccione http o https.

    Nombre de servidor

    El nombre de servidor en el que reside la URL de destino (TARGET URL) como, por ejemplo, www.sun.com.

    Puerto

    Este atributo contiene el número de puerto como, por ejemplo, 58080.

    Ruta

    Este atributo contiene un URI como, por ejemplo, /amserver/console.

  3. Haga clic en Aceptar para completar la configuración de la URL de destino.
  4. Pulse “Guardar” en la página de perfil SAML para completar la configuración de SAML.
