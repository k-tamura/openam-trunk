SAML1.x

El protocolo de estándares abiertos Lenguaje de marcado de aserción de seguridad (SAML, Security Assertion Markup Language) utiliza un marco XML para intercambiar información de seguridad entre una autoridad y un sitio de socio de confianza La información de seguridad se ocupa de la información de atributos, la autorización de accesos y el estado de autenticación del sujeto. (Una persona identificada por una dirección de correo electrónico es un sujeto, como también lo puede ser una impresora.) Una autoridad SAML (que también recibe el nombre de parte de aserción) es una plataforma o aplicación integrada en la API de SAML, que le permite transmitir información de seguridad. Por ejemplo, afirmando que un asunto se ha autenticado en el sistema mediante la transferencia de los atributos necesarios. Los sitios de socios de confianza reciben información de seguridad y dependen de su autenticidad.

Los siguientes atributos SAML pueden configurarse para su implementación haciendo clic en el botón Propiedades del sitio local:

  • Especificador de destino

  • Identificadores de sitio

  • Socios de confianza

  • URL de destino

  • Tiempo de espera de aserción

  • Factor de contracción de aserción para hora notBefore

  • Tiempo de espera de artefacto

  • Nombre de artefacto SAML

  • Firmar aserción SAML

  • Firmar solicitud SAML

  • Firmar respuesta SAML
