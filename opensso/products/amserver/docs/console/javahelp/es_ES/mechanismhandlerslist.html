Lista de administradores de mecanismos

El atributo Lista de administradores de mecanismos almacena información acerca de los mecanismos SASL admitidos por el Servicio Web de autenticación. It displays entries that contain key/value pairs separated by a pipe as in: key=PLAIN|class=com.sun.identity.liberty.ws.authnsvc.mechanism.PlainMechanismHandler . Los parámetros se explican en la siguiente tabla.

Parámetro

Valor

key

El parámetro key requerido define el mecanismo SASL admitido por el Servicio Web de autenticación.

class

El parámetro class requerido especifica el nombre de la clase de implementación del mecanismo SASL. La capa del Servicio Web de autenticación proporciona una interfaz de administrador que debe ser implementada en cada mecanismo SASL para poder procesar el mensaje solicitado y devolver una respuesta.

Para agregar un administrador de solicitudes nuevo, haga clic en Agregar y consulte Para configurar un administrador de mecanismos nuevo.
