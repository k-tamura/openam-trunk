Para crear un nuevo círculo de confianza

Siga este procedimiento para crear un nuevo círculo de confianza. El punto de partida es Nuevo círculo de confianza en la interfaz Federación.

  1. Haga clic en Nuevo para mostrar los atributos del círculo de confianza.

    Se muestra la página del perfil del nuevo círculo de confianza.

  2. Escriba el nombre del círculo de confianza.
  3. Escriba una descripción del círculo de confianza en el campo Descripción.
  4. Escriba un valor para la dirección URL del servicio de escritor de IDFF.

    La dirección URL del servicio de escritor de IDFF especifica la ubicación del servlet que escribe la cookie del dominio común. Utilice el formato http:// common-domain-host :port/common/idffwriter .

  5. Escriba un valor para la dirección URL del servicio de lector de IDFF.

    La dirección URL del servicio de lector de IDFF especifica la ubicación del servlet que lee la cookie del dominio común. Utilice el formato http:// common-domain-host :port/common/idffreader .

  6. Escriba un valor para la dirección URL del servicio de escritor de SAML2.

    Este atributo especifica la ubicación del servicio de escritor de SAML2 que escribe la cookie en el dominio común. Utilice el formato http:// common-domain-host :port/common/saml2writer .

  7. Escriba un valor para la dirección URL del servicio de lector de SAML2.

    Este atributo especifica la ubicación del servicio de lector de SAML2 que lee la cookie desde el dominio común. Utilice el formato http:// common-domain-host :port/common/saml2reader .

  8. Seleccione Activo o Inactivo.

    El estado predeterminado es “Activo” (Active). Si selecciona Inactivo, se deshabilita la comunicación dentro del dominio de autenticación.

  9. Seleccione el dominio en el que se creará el círculo de confianza.
  10. Elija uno o varios de los proveedores disponibles y haga clic en la flecha Agregar para seleccionarlos.

    La lista proporcionada contiene los nombres de las entidades creadas y rellenadas con proveedores. Para obtener más información, consulte Para agregar proveedores a un círculo de confianza.

  11. Haga clic en "Aceptar" para completar la configuración.

    El nuevo círculo de confianza se muestra en la lista de círculos de confianza.
