Firmar respuesta SAML

Este atributo especifica si se firman digitalmente todas las respuestas SAML (XML DSIG) antes de su entrega. Seleccione la casilla de verificación para habilitar esta función.

Nota – Todas las respuestas SAML utilizadas por el perfil de POST (envío) del explorador Web se firmarán digitalmente con independencia de que se habilite o no esta opción.
