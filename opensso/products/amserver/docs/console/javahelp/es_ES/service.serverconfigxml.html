Configuración de directorio

Los atributos de configuración del directorio definen la información de configuración básica para el almacén de directorio incrustado:

Configuración de directorio

Los atributos de configuración del directorio son:

Conjunto mínimo de conexiones

Especifica el tamaño mínimo de los grupos de conexiones que se van a utilizar para conectarse a Directory Server, como se especifica en el atributo del servidor LDAP. El valor predeterminado es 1.

Conjunto máximo de conexiones

Este atributo especifica el tamaño máximo de los grupos de conexiones que se van a utilizar para conectarse a Directory Server, como se especifica en el atributo del servidor LDAP. El valor predeterminado es 10.

DN de conexión

Especifica el DN de conexión del servidor LDAP.

Contraseña de conexión

Define la contraseña que se va a utilizar para la conexión con el servidor LDAP. La contraseña amldapuser que se introdujo durante la instalación se utiliza de forma predeterminada como usuario de conexión.

Servidor

Este atributo define el servidor de directorios que actuará como almacén de datos de configuración para la instancia de OpenSSO. Para agregar un servidor de configuración, haga clic en el botón Agregar y facilite valores para los siguientes atributos:

Nom

Introduzca un nombre para el servidor.

Nombre de host

Especifica el nombre de host completo del servidor de directorios. Por ejemplo:

Hostservidordirectorios.nombredominio.com

Port Number

Especifica el número de puerto del servidor de directorios.

Connection Type

Define el tipo de conexión para el servidor de directorios. De forma predeterminada está seleccionado el valor SIMPLE. Tiene la posibilidad de elegir también SSL.

Configuración tradicional

Los siguientes atributos definen las configuraciones básicas del servidor de directorios para las instancias de OpenSSO en el modo tradicional. Estos atributos aparecerán únicamente en una instalación en modo heredado.

Conjunto mínimo de conexiones

Especifica el tamaño mínimo de los grupos de conexiones que se van a utilizar para conectarse a Directory Server, como se especifica en el atributo del servidor LDAP. El valor predeterminado es 1.

Conjunto máximo de conexiones

Este atributo especifica el tamaño máximo de los grupos de conexiones que se van a utilizar para conectarse a Directory Server, como se especifica en el atributo del servidor LDAP. El valor predeterminado es 10.

Servidor

Este atributo muestra el nombre de host, el puerto y el protocolo del equilibrador de carga. Por ejemplo: http://lb.ejemplo.com:80.
