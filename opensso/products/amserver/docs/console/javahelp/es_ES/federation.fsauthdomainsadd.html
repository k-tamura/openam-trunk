Para crear un dominio de autenticación nuevo

Siga este procedimiento para crear un dominio de autenticación nuevo. El punto de partida es Dominios de autenticación en la interfaz Federación.

  1. Haga clic en Nuevo para que se muestren los atributos del dominio de autenticación.

    Aparece la página de perfiles Nuevo dominio de autenticación.

  2. Escriba un nombre para el dominio de autenticación.
  3. Escriba una descripción del dominio de autenticación en el campo Descripción.
  4. Escriba un valor para la dirección URL del servicio de escritor.

    La dirección URL del servicio de escritor especifica la ubicación del servlet que escribe la cookie del dominio común. Utilice el formato http://common-domain-host :port/common/writer.

  5. Escriba un valor para la dirección URL del servicio de lector.

    La dirección URL del servicio de lector especifica la ubicación del servlet que lee la cookie del dominio común. Utilice el formato http://common-domain-host :port/common/transfer.

  6. Seleccione Activo o Inactivo.

    El estado predeterminado es âActivoâ (Active). Si selecciona Inactivo, se deshabilita la comunicación dentro del dominio de autenticación.

  7. Haga clic en "Aceptar" para completar la configuración.

    El nuevo dominio de autenticación se muestra en la pantalla Dominios de autenticación.
