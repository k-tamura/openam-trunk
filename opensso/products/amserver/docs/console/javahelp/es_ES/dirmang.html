Administración de directorios

La ficha Administración de directorio sólo se muestra al instalar OpenSSO en el modo tradicional. La función de administración de directorio proporciona una solución de administración de identidades para las implementaciones de OpenSSO habilitadas para Sun Java System Directory Server.

Para obtener más información acerca de las opciones de instalación del modo heredado, consulte Sun OpenSSO Installation and Configuration Guide.
