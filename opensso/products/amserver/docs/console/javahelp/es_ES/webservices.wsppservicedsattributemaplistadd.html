Para configurar una asignación de atributos

Una asignación es un atributo que define una ubicación en la que se guarda una parte de los datos de identidad. A continuación, se muestra el procedimiento para agregar nuevos atributos al Servicio de perfil personal de Liberty. El punto de partida es la pantalla de configuración del Servicio de perfil personal de Liberty en el módulo Servicios Web.

  1. Haga clic en Nuevo para mostrar los nuevos atributos de asignación de atributos o seleccione el nombre de una asignación configurada para modificarla.
  2. Proporcione los valores de los atributos de contenedores según la siguiente información:
    Prefijo de nombre

    Introduzca el atributo de identidad del Servicio de perfil personal de Liberty que se va a asignar.

    Atributo LDAP

    Introduzca el nombre del atributo LDAP de OpenSSO al que se asigna el prefijo del nombre.

  3. Haga clic en Aceptar para completar la configuración de la asignación.
  4. Haga clic en Guardar en la página del Servicio de perfil personal de Liberty para completar la configuración del servicio.
