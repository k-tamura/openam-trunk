Para crear una entidad de proveedores o una entidad de afiliación

El proceso de establecimiento de una entidad está compuesto por dos pasos. En primer lugar, debe crear la entidad. (Es el momento de determinar si va a ser una entidad de proveedores o una entidad de afiliación.) Después de crear la entidad, debe rellenarla con proveedores. El siguiente procedimiento permite crear una nueva entidad. El punto de partida es Entidades en la interfaz Federación.

Nota – Se puede crear una entidad, pero no estará disponible para asignarla a un círculo de confianza hasta que se hayan agregado los proveedores.

  1. Haga clic en Nuevo para que se muestren los atributos de la entidad.

    Aparece la página Nueva entidad.

  2. Escriba un nombre para la entidad.

    El atributo Nombre de entidad define un identificador de recursos uniformes (URI) para la entidad. Este valor debe ser exclusivo entre los nombres de entidades.

  3. Escriba una descripción para la entidad.
  4. Seleccione un tipo de entidad.

    • Si va a crear una entidad de proveedores, seleccione Proveedor.

    • Si va a crear una entidad de afiliación, seleccione Afiliado, y defina los valores para los atributos Nombre de afiliado y Propietario de afiliado.

      • El nombre de afiliado define un URI para la entidad de afiliación. Este valor debe ser exclusivo entre los nombres de afiliados.

      • El propietario de afiliado especifica un proveedor como administrador principal de la afiliación. El valor de este atributo es un URI que coincidirá con el valor definido en el atributo Nombre de entidad de la entidad a la que pertenece el proveedor/propietario de afiliado.

      Nota – Si se define un proveedor como propietario de afiliado, esto no implica que se incluya automáticamente como miembro de la afiliación. Si un propietario es también miembro, el URI debe definirse en esta ubicación como propietario y, dentro de la propia entidad de afiliación, como miembro. Para agregar miembros a una afiliación, consulte Para agregar miembros a una entidad de afiliación.

  5. Haga clic en Aceptar.

    La nueva entidad aparece en la pantalla Entidades.
