Lista de administradores de solicitudes

El Servicio de enlace de SOAP proporciona la interfaz RequestHandler para procesar el mensaje de solicitud y devolver una respuesta. La interfaz debe implementarse en el servidor por parte de cada servicio de identidad basado en Liberty que utilice el Servicio de enlace de SOAP. El atributo “Lista de administradores de solicitudes” guarda información sobre las clases de implementación de los servicios Web que implementa el administrador de solicitudes. The Request Handler List displays entries that contain key/value pairs separated by a pipe as in: key=disco|class=com.example.identity.liberty.ws.disco.DiscoveryService . Los parámetros se explican en la siguiente tabla.

Parámetro

Encabezado

key

El parámetro key requerido es la última parte de la ruta del URI a un punto final de SOAP. El punto final de SOAP de OpenSSO es el servlet del receptor SOAP. The URI to the SOAP Receiver is: protocol://host:port /deploy-uri/Liberty/key If you define disco as the key, the URI path to the SOAP endpoint for the corresponding Servicio de detección would be: protocol://host:port /amserver/Liberty/disco. Distintos clientes de servicios utilizan distintas claves al conectarse al receptor de SOAP.

class

Este parámetro class requerido especifica el nombre de la clase de implementación del administrador de solicitudes para el servicio de identidades concreto. For example: class=com.example.identity.liberty.ws.disco.DiscoveryService .

Para agregar un nuevo administrador de solicitudes, consulte Para configurar un administrador de solicitudes.
