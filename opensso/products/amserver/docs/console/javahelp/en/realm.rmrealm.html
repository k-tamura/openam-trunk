Realms

An access control realm is a group of authentication properties and authorization policies you can associate with a user or group of users. Realm data is stored in a proprietary information tree that OpenSSO creates within a data store you specify. The OpenSSO framework aggregates policies and properties contained in each realm within the OpenSSO information tree. By default, OpenSSO 8.0 automatically inserts the OpenSSO information tree as a special branch in Sun Java Enterprise System Directory Server, apart from the user data. You can use access control realms while using any LDAPv3 database.
