Supported Containers

The values of this attribute define a list of supported containers in the Liberty Personal Profile Service. A container, as used in this instance, is an attribute of the Liberty Personal Profile Service. For example, Emergency Contact and Common Name are two default containers.

Note – The term container as described here is not related to the OpenSSO identity-related object also called container.

To add a new container, click New, and see To Configure a Supported Container.
