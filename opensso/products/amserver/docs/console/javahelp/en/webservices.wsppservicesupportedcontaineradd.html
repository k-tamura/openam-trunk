To Configure a Supported Container

A container is an attribute that defines a holder for a piece of identity data. Following is the procedure for adding new attributes to the Liberty Personal Profile Service. The starting point is the Liberty Personal Profile Service configuration screen within the Web Services module.

  1. Click New to display the new Supported Container attributes or click on the name of a configured container to modify its profile.
  2. Provide values for the container attributes based on the following information:
    Container Name

    Enter a name for the container such as CreditCard.

    Plugin

    Enter a class name to handle the whole container. This could be used to override the default implementation com.sun.identity.liberty.ws.idpp.plugin.IDPPContainer.

  3. Click OK to complete the Container configuration.
  4. Click Save on the Liberty Personal Profile Service page to complete the service configuration.
