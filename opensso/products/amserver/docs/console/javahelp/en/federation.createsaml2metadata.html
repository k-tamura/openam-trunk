Entity Providers

A entity provider holds the metadata for individual entity providers types. OpenSSO allows you create three types of entity providers based on Liberty protocols: SAMLv2, ID-FF, and WS-FED. Within each entity provider type, you can assign and customize a number of entity provider roles to perform specific functions in federation communication by using the attributes provided in the OpenSSO Console.
