Privileges Properties

Privileges define the access permissions to the authenticated users that exist within a realm. The users are used as policy subjects in privilege definitions. To assign or modify privileges, click the name that you wish to edit. The privileges you can assign are:

  • Read and write access to all log files

  • Read access to all log files

  • Write access to all log files

  • Read and write access only for policy properties

  • Read and write access to all realm and policy properties

For more information, see Policies.
