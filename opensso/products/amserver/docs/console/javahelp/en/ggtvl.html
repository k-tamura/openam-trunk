To Create a New Agent

Perform these steps to create a STS Client agent:

  1. In the Agent section, click New.
  2. Select the appropriate agent type.
    • Discovery agent
    • Security Token Service agent
  3. In the Name field, enter the name for the new agent profile.
  4. Enter and confirm the Password.
  5. Click Create.

    The console creates the agent profile and displays the agent (Discovery, Security Token Service) page again with a link to the new agent profile.

    To do additional configuration for the agent profile, click this link to display the Edit agent page.
