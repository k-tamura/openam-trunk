To Create a New Group

Create a group if you want agents to inherit specific properties from the group. Discovery agents can inherit properties from a Discovery agent group while STS agents can inherit properties from a STS agent group.

  1. In the Group section, click New.
  2. Select the appropriate agent type.
    • Discovery Agent
    • Security Token Agents
  3. In the Name field, enter the name for the new group name.
  4. Click Create.

    The console creates the agent group and displays the agent page again with a link to the group.

    To do additional configuration of the group, click this link to display the Edit agent group page.

    The attributes you can set to configure a group are the same as they are for an individual agent except that the Group, Password, and Password Confirm attributes are not available at the group level.
