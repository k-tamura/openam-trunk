Liste de mappages d'attributs PPLDAP

Chaque attribut d'identité défini par Service de profil personnel Liberty est associé à un attribut LDAP selon une relation bijective. La valeur de cet attribut est une liste spécifiant ces mappages. Par exemple, JobTitle=sunIdentityServerPPEmploymentIdentityJobTitle fait correspondre l'attribut JobTitle de Liberty avec l'attribut sunIdentityServerPPEmploymentIdentityJobTitle d'OpenSSO. Lorsque vous ajoutez de nouveaux attributs d'un côté ou de l'autre de l'équation, assurez-vous que les nouveaux mappages sont bien configurés dans cet attribut. Reportez-vous à Configuration d'un mappage d'attributs.
