Configuration d'un mappeur d'ID de ressource

com.sun.identity.liberty.ws.interfaces.ResourceIDMapper est une interface utilisée pour mapper un ID utilisateur à l'ID de ressource qui lui est associé. Différentes implémentations peuvent être développées avec la classe d'implémentation et ajoutées à l'attribut. La procédure à suivre pour ajouter un nouveau mappeur d'ID de ressource au Service de découverte est décrite ci-dessous. Le point de départ est l'écran de configuration du Service de découverte dans le module Services Web.

  1. Cliquez sur Nouveau pour afficher les attributs du nouveau mappeur ou cliquez sur le nom d'un mappeur configuré pour le modifier.
  2. Fournissez des valeurs pour les attributs de mappeur en vous aidant des informations suivantes :
    ID fournisseur

    Cet attribut accepte comme valeur un URI pointant vers le Service de découverte. La valeur est écrite au format : http:// host:port/amserver/Liberty/disco

    Mappeur d'ID

    Entrez le nom de la classe d'implémentation.

  3. Cliquez sur OK pour terminer la configuration du mappeur.
  4. Cliquez sur Enregistrer dans la page du Service de découverte pour terminer la configuration du service.
