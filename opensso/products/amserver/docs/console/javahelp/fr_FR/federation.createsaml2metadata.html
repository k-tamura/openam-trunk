Fournisseurs d'entités

Un fournisseur d'entités contient les métadonnées qui lui correspondent. OpenSSO vous permet de créer trois types de fournisseurs d'entités basés sur les protocoles Liberty : SAMLv2, ID-FF et WS-FED. Dans chaque type de fournisseurs d'entités, vous pouvez attribuer et personnaliser plusieurs rôles afin d'effectuer des fonctions particulières lors de communications fédérées en utilisant les attributs fournis dans la console OpenSSO.

Reportez-vous aux sections suivantes pour une description des types de fournisseur d'entités et des rôles que vous pouvez attribuer :

  • Entité de fournisseur SAMLv2

  • Entité de fournisseur ID-FF

  • Entité de fournisseur WS-FED
