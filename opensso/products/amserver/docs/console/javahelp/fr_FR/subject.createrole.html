Création ou modification d'un rôle
  1. Cliquez sur l'onglet Rôle.
  2. Cliquez sur Nouveau dans la liste des rôles.
  3. Entrez le nom du rôle.
  4. Cliquez sur OK.

    Une fois le rôle créé, cliquez sur le nom du rôle pour effectuer les tâches suivantes :

    • Ajout de services à une identité

    • Ajout d'utilisateurs à un rôle ou à un groupe
