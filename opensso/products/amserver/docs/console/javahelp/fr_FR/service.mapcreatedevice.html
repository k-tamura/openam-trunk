Ajout d'un nouveau client

  1. Cliquez sur Nouveau dans la liste Type de client.
  2. Sélectionnez le type de périphérique à l'aide des champs suivants :
    Style

    Affiche le style de base du périphérique. Par exemple, HTML.

    Agent utilisateur de l'unité

    Accepte le nom pour le périphérique.

  3. Cliquez sur Suivant.
  4. Entrez les informations suivantes pour le nouveau périphérique :
    Nom du type de client

    Accepte le nom pour le périphérique. Ce nom doit être unique sur tous les périphériques.

    Chaîne utilisateur HTTP

    Définit l'agent utilisateur dans l'en-tête de la demande HTTP. Par exemple, Mozilla/4.0.

  5. Cliquez sur Terminer.
  6. Pour dupliquer un périphérique et ses propriétés, cliquez sur le lien Dupliquer. Les noms de périphérique doivent être uniques. Par défaut, OpenSSO renomme le périphérique en copie_de_nompériphérique.
