Cartes des attributs Serveurs et sites de la console

Le tableau ci-dessous répertorie les propriétés Serveurs et sites qui figuraient dans le fichier AMConfig.properties dans les versions précédentes, mais qui sont maintenant gérées comme des attributs dans la console OpenSSO. Les propriétés sont classées par ordre alphabétique. Pour rechercher une propriété particulière, utilisez la fonction Rechercher de votre navigateur.

Nom de la propriété

Nom de la propriété située dans le fichier AMConfig.properties.

Nom de l'attribut dans la console

Nom de l'attribut tel qu'il apparaît dans la console OpenSSO.

Indique l'emplacement de l'attribut dans la console.
