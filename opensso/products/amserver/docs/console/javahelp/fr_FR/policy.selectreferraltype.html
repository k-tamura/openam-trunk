Ajout d'orientations à une stratégie
  1. Si vous avez déjà créé la stratégie, cliquez sur le nom de la stratégie à laquelle vous voulez ajouter le fournisseur de réponses. Si vous n'avez pas encore créé la stratégie, reportez-vous à Création d'une stratégie d'orientation.
  2. Dans la liste Orientations, cliquez sur Nouveau.
  3. Définissez la ressource dans les champs Règles. Ces champs sont les suivants :

    Orientation— Affiche le type d'orientation en cours.

    Nom— Entrez le nom de l'orientation.

    Contenant — Spécifie un filtre pour les noms de domaine qui seront affichés dans le champ Valeur. Par défaut, tous les noms de domaine sont affichés.

    Valeur— Sélectionnez le nom de domaine de l'orientation.

  4. Cliquez sur Terminer.

    Pour supprimer une orientation d'une stratégie, sélectionnez l'orientation et cliquez sur Supprimer.

    Vous pouvez modifier la définition de l'orientation en cliquant sur le lien Modifier en regard du nom de l'orientation.
