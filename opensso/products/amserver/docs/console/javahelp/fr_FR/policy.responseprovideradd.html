Ajout d'un fournisseur de réponses à une stratégie
  1. Si vous avez déjà créé la stratégie, cliquez sur le nom de la stratégie à laquelle vous souhaitez ajouter le fournisseur de réponses. Si vous n'avez pas encore créé la stratégie, reportez-vous à Création d'une stratégie normale.
  2. Sous la liste Fournisseurs de réponses, cliquez sur Nouveau.
  3. Entrez un nom pour le fournisseur de réponses.
  4. Définissez les valeurs suivantes :
    StaticAttribute

    Il s'agit d'attributs statiques au format de valeur d'attribut, définis dans une instance de IDResponseProviderstockée dans la stratégie.

    DynamicAttribute

    Les attributs de réponse choisis ici doivent être les premiers définis dans le service de Configuration de règles du domaine correspondant. Les noms d'attribut définis doivent être un sous-ensemble de ceux existant dans le magasin de données configuré (IDRepository). Pour plus d'informations sur la définition des attributs, reportez-vous aux définitions d'attribut Configuration de stratégie. Pour sélectionner des attributs spécifiques ou multiples, maintenez la touche Ctrl enfoncée et cliquez sur le bouton gauche de la souris.

  5. Cliquez sur Terminer.
  6. Pour supprimer un fournisseur de réponses d'une stratégie, sélectionnez l'objet et cliquez sur Supprimer. Vous pouvez modifier n'importe quelle définition de fournisseur de réponses en cliquant sur son nom.
