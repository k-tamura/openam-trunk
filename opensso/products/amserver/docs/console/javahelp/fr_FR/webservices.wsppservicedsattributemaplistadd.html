Configuration d'un mappage d'attributs

Un mappage est un attribut qui définit un emplacement réservé pour chaque donnée d'identité. La procédure permettant d'ajouter de nouveaux attributs à Service de profil personnel Liberty est la suivante. Le point de départ est l'écran de configuration de Service de profil personnel Liberty du module Services Web.

  1. Cliquez sur Nouveau pour afficher les nouveaux attributs de mappage d'attributs ou cliquez sur le nom d'un mappage configuré pour le modifier.
  2. Fournissez les valeurs pour les attributs de conteneur en vous basant sur les informations suivantes :
    Préfixe du nom

    Entrez le nom de l'attribut d'identité Service de profil personnel Liberty à mapper.

    Attribut LDAP

    Entrez le nom de l'attribut LDAP d'OpenSSO auquel associer le préfixe du nom.

  3. Cliquez sur OK pour terminer la configuration du mappage.
  4. Cliquez sur Enregistrer dans la page Service de profil personnel Liberty pour terminer la configuration du service.
