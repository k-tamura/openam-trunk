Création d'un plug-in de référentiel Access Manager

Cette section répertorie les étapes permettant de créer un plug-in de référentiel.

Création d'un plug-in de référentiel Access Manager

  1. Sélectionnez le domaine dans lequel vous souhaitez configurer le plug-in de référentiel Access Manager.

  2. Cliquez sur l'onglet Magasin de données.

  3. Cliquez sur Nouveau dans la liste des magasins de données.

  4. Entrez un nom pour le plug-in de référentiel.

  5. Sélectionnez Plug-in du référentiel Access Manager.

  6. Cliquez sur Suivant.

  7. Définissez les champs suivants:

    Nom de classe du plug-in Access Manager

    Spécifie l'emplacement du fichier de classe qui implémente le plug-in du référentiel Access Manager.

    Organisation Access Manager

    DN pointant vers une organisation de Directory Server qui doit être gérée par Access Manager. Ceci constituera le DN de base de toutes les opérations effectuées dans le magasin de données.

  8. Cliquez sur Terminer.
