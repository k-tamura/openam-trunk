Configuration du service Liberty ID-FF

Nom de cookie de fédération

Cet attribut spécifie le nom du cookie des services ID-FF. Ce cookie permet de mémoriser si l'utilisateur est déjà fédéré.

Classe d'implémentation SPI de mécanisme de recherche de proxy IDP

Cet attribut spécifie la classe d'implémentation pour rechercher un fournisseur d'identités préféré à utiliser comme proxy.

Intervalle de nettoyage de cache de demande

Cet attribut spécifie l'intervalle de nettoyage (en secondes) du thread de nettoyage de demande interne ID-FF.

Temporisation de cache de demande

Cet attribut spécifie la valeur de temporisation (en secondes) de la demande d'authentification ID-FF. Tout objet AuthnRequest est purgé de la mémoire s'il dépasse la valeur de temporisation.

URl de connexion IDP

Cet attribut spécifie l'URL de connexion vers laquelle l'IDP redirige l'utilisateur si aucune session valide n'est trouvée pendant le traitement de la demande d'authentification. Si la clé n'est pas spécifiée, une URL de connexion par défaut est utilisée.

Connexion XML

Cet attribut spécifie le niveau de vérification de signature des demandes et des réponses Liberty.
