Composants des tâches courantes

Consultez la documentation en ligne ou cliquez sur l'icône située en regard de chaque tâche courante pour en obtenir les informations spécifiques.

OpenSSO comporte les flux de travaux suivants :

Créer des fournisseurs SAMLv2

Ce flux de travaux vous permet de créer des entités de fournisseur de service et de fournisseur d'identité basées sur le protocole SAMLv2. Les deux types de fournisseur peuvent être hébergés ou distants. Pour les créer, il est nécessaire de fournir les informations de bases relatives aux fournisseurs, telles que le cercle de confiance (COT, Circle of Trust) auquel le fournisseur appartient et les métadonnées du fournisseur. Pour connaître les informations générales sur les fournisseurs SAMLv2, voir Entité de fournisseur SAMLv2. Sinon, suivez les instructions à l'écran.

Créer un Fedlet

Un Fedlet est un petit fichier ZIP pouvant être transmis à un fournisseur de services pour obtenir une fédération immédiate à un fournisseur d'identités configuré avec OpenSSO. C'est la solution idéale pour un fournisseur d'identités qui doit activer un fournisseur de services sans aucune solution de fédération. Le fournisseur de services ajoute simplement le Fedlet à son application et déploie cette dernière pour activer la fédération. Veuillez consultez le manuel Sun OpenSSO Enterprise 8.0 Administration Guide pour plus d'informations sur le déploiement de cette application.

Vérifier la connectivité de la fédération

Ce test vérifie si les connexions de fédération sont établies et localise les problèmes éventuels.

Accéder à la documentation

Cette tâche courante ouvre la page de documentation sur OpenSSO. Celle-ci contient les questions les plus fréquemment posées, la documentation sur les produits, la documentation d'ingénierie, ainsi que des liens pointant vers les blogs de la communauté.

Enregistrer votre produit

Ce lien vous permet d'enregistrer votre produit avec Sun Connection. Vous devez disposer d'un compte en ligne Sun pour effectuer l'enregistrement. Si vous n'en disposez pas, vous pouvez en demander un au cours de ce processus.
