Création d'un cercle de confiance

Suivez la procédure ci-dessous pour créer un cercle de confiance. Le point de départ est Nouveau cercle de confiance dans l'interface Fédération.

  1. Cliquez sur Nouveau pour afficher les attributs du cercle de confiance.

    La page Nouveau profil de cercle de confiance apparaît.

  2. Tapez le nom du cercle de confiance.
  3. Tapez la description du cercle de confiance dans le champ Description.
  4. Tapez la valeur de l'URL du service écriture IDFF.

    L'URL du service écriture IDFF indique l'emplacement du servlet qui écrit le cookie du domaine commun. Utilisez le format http://common-domain-host:port/common/idffwriter.

  5. Tapez la valeur de l'URL du service lecture IDFF.

    L'URL du service lecture IDFF indique l'emplacement du servlet qui lit le cookie du domaine commun. Utilisez le format http://common-domain-host:port/common/idffreader.

  6. Tapez la valeur de l'URL du service écriture SAML2.

    Spécifie l'emplacement du service écriture SAML2 qui écrit le cookie dans le domaine commun. Utilisez le format http://common-domain-host:port/common/saml2writer.

  7. Tapez la valeur de l'URL du service lecture SAML2.

    Spécifie l'emplacement du service lecture SAML2 qui lit le cookie dans le domaine commun. Utilisez le format http://common-domain-host:port/common/saml2reader.

  8. Choisissez Actif ou Inactif.

    Le statut par défaut est Actif. Le choix de Inactif désactive les communications au sein du domaine d'authentification.

  9. Sélectionnez le domaine dans lequel sera créé le cercle de confiance.
  10. Choisissez un ou plusieurs fournisseurs disponibles et cliquez sur la flèche Ajouter pour les sélectionner.

    La liste fournie contient les noms des entités créées et renseignées avec des fournisseurs. Pour plus d'informations, reportez-vous à la rubrique Ajout de fournisseurs à un cercle de confiance.

  11. Cliquez sur OK pour terminer la configuration.

    Le nouveau cercle de confiance s'affiche dans la liste Cercle de confiance.
