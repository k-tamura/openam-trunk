Appartenance à Identity

Vérifie si l'invocateur uuid spécifié dans l'environnement est membre d'au moins un objet AMIdentity déclaré dans la condition. L'invocateur uuid est spécifié comme clé de Condition.INVOCATOR_PRINCIPAL_UUID dans le paramètre environment de la demande d'évaluation. Ceci permet principalement d'appliquer les règles d'autorisation du client de service Web (WSC). L'identité du WSC est transmise en tant que valeur de l'invocateur uuid.
