Configuration du répertoire

Les attributs de configuration du répertoire définissent les informations de configuration de base du magasin de répertoire incorporé :

Configuration du répertoire

Les attributs de configuration du répertoire sont :

Pool de connexion minimum

Spécifie la taille minimale des pools de connexion devant être utilisés pour la connexion au serveur Directory Server, telle que spécifiée par l'attribut du serveur LDAP. La valeur par défaut est 1.

Pool de connexion maximum

Spécifie la taille maximale des pools de connexion devant être utilisés pour la connexion au serveur Directory Server, telle que spécifiée par l'attribut du serveur LDAP. La valeur par défaut est 10.

DN de liaison

Spécifie le DN de liaison dans le serveur LDAP.

Mot de passe de liaison

Définit le mot de passe à utiliser pour établir la liaison avec le serveur LDAP. Par défaut, le mot de passe amldapuser entré lors de l'installation est utilisé comme utilisateur de liaison.

Serveur

Cet attribut définit le serveur d'annuaire utilisé comme magasin de données de configuration pour l'instance OpenSSO. Pour ajouter un serveur de configuration, cliquez sur le bouton Ajouter, puis indiquez la valeur des attributs suivants :

Nom

Indiquez le nom du serveur.

Nom d'hôte

Indique le nom d'hôte complet du Directory Server. Par exemple :

HôteDirectoryServer.nomDomaine.com

Numéro de port

Spécifie le numéro de port du Directory Server.

Type de connexion

Définit le type de connexion du Directory Server. Par défaut, SIMPLE est sélectionné. Vous pouvez également choisir SSL.

Configuration héritée

Les attributs suivants définissent les configurations Directory Server de base des instances en mode hérité d'OpenSSO. Ces attributs ne s'affichent que pour une installation en mode hérité.

Pool de connexion minimum

Spécifie la taille minimale des pools de connexion devant être utilisés pour la connexion au serveur Directory Server, telle que spécifiée par l'attribut du serveur LDAP. La valeur par défaut est 1.

Pool de connexion maximum

Spécifie la taille maximale des pools de connexion devant être utilisés pour la connexion au serveur Directory Server, telle que spécifiée par l'attribut du serveur LDAP. La valeur par défaut est 10.

Serveur

Cet attribut répertorie le protocole d'équilibreur de charge, le nom d'hôte et le port. Par exemple, http://lb.exemple.com:80.
