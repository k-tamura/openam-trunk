Attributs de l'agent de découverte

Le profil de l'agent de découverte contient une configuration d'autorité de confiance que le client/profil des services Web utilise afin de communiquer avec le service de découverte Liberty pour les recherches de services Web et l'enregistrement et pour obtenir les références de sécurité.

Groupe

Le mécanisme de groupe permet de définir un ensemble de types d'agents similaires. Vous devez définir le groupe avant d'inclure l'agent spécifique dans un ensemble.

Mot de passe

Définit le mot de passe de l'agent de découverte.

Confirmation de mot de passe

Confirmez le mot de passe.

Statut

Définit si l'agent sera actif ou inactif au sein du système. Par défaut, cet attribut est activé, ce qui signifie que l'agent va participer à la sécurisation des demandes de services Web sortantes émises par les clients de services Web et va valider les réponses de services Web renvoyées par un fournisseur de services Web.

Emplacement du référentiel de configuration de l'agent

Cet attribut définit l'emplacement d'agent du référentiel de configuration de l'agent de découverte.

Alias de clé privée

Cet attribut définit l'alias de clé de certificat privée, utilisé pour signer la demande de services Web ou déchiffrer la réponse des services Web.

Point d'arrivée du service de découverte

Cet attribut définit le point d'arrivée du service de découverte où le client de l'autorité de confiance établit des communications en vue des enregistrements et des recherches de services.

Point d'arrivée du service Web d'authentification

Cet attribut définit l'extrémité du service d'authentification que le client de services Web utilise pour s'authentifier avec le jeton SSOToken de l'utilisateur final afin de recevoir l'offre de ressources du service de découverte (également appelée offre de ressources d'initialisation).
