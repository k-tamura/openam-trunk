Création d'un agent

Créer un agent est également appelé créer un profil d'agent. Vous pouvez effectuer cette tâche dans la console d'administration d'OpenSSO. Certaines étapes ne concernent que les agents de stratégie J2EE et Web.

  1. Cliquez sur l'onglet Contrôle d'accès.
  2. Cliquez sur le nom du domaine auquel l'agent appartiendra.
  3. Cliquez sur l'onglet Agents.
  4. Cliquez sur l'onglet correspondant au type d'agent approprié.
    • Agents Web
    • Agents J2EE
    • Agents de fournisseur de services Web
    • Agents de client de services Web
    • Agent de client STS

      L'agent de client du service de jeton de sécurité existe en deux types : Agent de découverte et Agent STS. Pour plus d'informations sur ces types d'agent, reportez-vous à la rubrique Client STS.

    • Agents 2.2

      L'onglet Agents 2.2 est destiné aux agents de la suite logicielle Policy Agent 2.2.

    • Authentificateur d'agent
  5. Dans la section Agent, cliquez sur Nouveau.
  6. Dans le champ Nom, entrez le nom du nouveau profil d'agent.
  7. Entrez et confirmez le mot de passe.

    Attention – Ce mot de passe doit être le mot de passe entré dans le fichier de mots de passe de profil d'agent que vous spécifiez lorsque vous exécutez le programme agentadmin pour installer l'agent.

    Les étapes 8 à 10 ne concernent que les agents de stratégie Web et J2EE.

  8. Sélectionnez la configuration appropriée (locale ou centralisée).

    Si vous choisissez la configuration locale, vous ne pouvez pas modifier les propriétés de cet agent dans la console. Dans ce cas, l'agent extrait les informations de configuration des fichiers locaux OpenSSOAgentBootstrap.properties et OpenSSOAgentConfiguration.properties, situés dans le répertoire d'installation de l'agent. Pour un agent configuré localement, modifiez la valeur des propriétés directement dans le fichier OpenSSOAgentConfiguration.properties.

  9. Dans le champ URL du serveur, saisissez l'URL du serveur OpenSSO.

    Par exemple :

    http://OpenSSO_EnterpriseHost.example.com:8080/OpenSSO
  10. Dans le champ URL de l'agent, saisissez l'URL de l'application agent agentapp .

    Par exemple :

    Agents Web

    http://agentHost.example.com:8090

    Agents J2EE

    http://agentHost.example.com:8090/agentapp

  11. Cliquez sur Créer.

    La console crée le profil de l'agent et affiche de nouveau sa page avec un lien pointant vers le nouveau profil.

    Pour effectuer d'autres tâches de configuration du profil d'agent, cliquez sur ce lien afin d'afficher la page Modification de groupe d'agents.
