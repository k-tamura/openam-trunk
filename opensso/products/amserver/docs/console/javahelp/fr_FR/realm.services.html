Services

Dans OpenSSO, un service est un groupe d'attributs gérés ensemble par la console OpenSSO. Ces attributs peuvent être constitués uniquement d'éléments d'information liés, tels qu'un nom d'employé, un intitulé de poste et une adresse e-mail. Mais les attributs sont généralement utilisés comme des paramètres de configuration pour un module logiciel tel qu'une application de messagerie ou un service de gestion des paies.

À l'aide de l'onglet Services, vous pouvez ajouter et configurer plusieurs services par défaut d'OpenSSO sur un domaine. Vous pouvez ajouter les services suivants :

  • Administration

  • Configuration de règles

  • Service de découverte

  • Paramètres de globalisation

  • Réinitialisation du mot de passe

  • Session

  • Utilisateur
