Partenaires de confiance : sélection du type et du profil de partenaire

Cet attribut définit tout partenaire de confiance (distant par rapport au serveur sur lequel OpenSSO est installé) qui communiquera avec OpenSSO.

Remarque – Le site du partenaire de confiance doit avoir une relation de confiance préalable avec un ou plusieurs des sites configurés dans l'attribut Identificateurs de site.

La première étape de configuration d'un partenaire de confiance consiste à déterminer le rôle du partenaire dans la relation de confiance. Un partenaire de confiance peut être un site source (qui génère une assertion de connexion unique) ou un site de destination (qui consomme une assertion de connexion unique). Par exemple, si le partenaire est le site source, la configuration de cet attribut est fonction de la manière dont les assertions seront envoyées. Si le partenaire est le site de destination, la configuration de cet attribut est basée sur le profil selon lequel il recevra les assertions. Voici la première partie de la procédure de configuration d'un partenaire de confiance. Le point de départ est l'écran SAML sous Fédération.

Remarque – Pour modifier ou dupliquer les attributs d'un profil de partenaire de confiance, cliquez sur le bouton correspondant dans la colonne Actions à côté du nom du partenaire de confiance configuré.

  1. Sélectionnez le rôle (Destination ou Source) du site partenaire de confiance en cours de configuration en cochant le ou les profils de communication approprié(s).

    Vous pouvez choisir le profil d'artéfact du navigateur Web ou le profil POST du navigateur Web pour la destination, la source ou les deux, ou Requête SOAP pour la destination. Les choix effectués influent sur les attributs qui devront être configurés dans les étapes qui suivent.

    Remarque – Si vous modifiez un partenaire de confiance existant, cliquez sur Modifier pour changer le rôle du site du partenaire.

  2. Cliquez sur Suivant.
  3. Pour terminer la configuration, consultez Partenaires de confiance : configuration des attributs du partenaire de confiance.
