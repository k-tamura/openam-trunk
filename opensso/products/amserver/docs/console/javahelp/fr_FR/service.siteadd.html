Création d'une instance de site

  1. Cliquez sur Nouveau dans la liste des sites.
  2. Entrez le nom d'hôte et le port dans le champ Serveur.
  3. Entrez le Nom du site.

    Cette valeur identifie le serveur de valeur univoque et permet de spécifier un second point d'entrée (en plus de l'URL principale) au site. Ceci permet également de raccourcir la longueur des cookies en mappant l'URL du serveur à l'ID du serveur.

  4. Indiquez l'URL principale de l'instance du site, y compris l'URI du site.
  5. Cliquez sur Enregistrer.

    Le site créé apparaît dans la liste des sites dans le format correct.
