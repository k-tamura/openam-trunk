Agents

La configuration d'agent centralisée permet à un administrateur d'agents de gérer plusieurs configurations d'agents à partir d'un même emplacement central. Les configurations d'agent sont stockées dans le référentiel de données d'OpenSSO et gérées par un administrateur à la console OpenSSO. Toute modification de configuration d'agent est transmise aux agents affectés et les agents réagissent en conséquence selon la nature des propriétés mises à jour.
