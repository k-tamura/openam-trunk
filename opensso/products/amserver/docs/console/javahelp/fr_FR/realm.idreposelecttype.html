Création d'un magasin de données

  1. Sélectionnez le domaine auquel vous souhaitez ajouter un nouveau magasin de données.
  2. Cliquez sur l'onglet Magasin de données.
  3. Cliquez sur Nouveau dans la liste des magasins de données.
  4. Entrez le nom du magasin de données.
  5. Sélectionnez le type de magasin de données que vous souhaitez créer.
  6. Cliquez sur Suivant.
  7. Configurez le magasin de données en entrant les valeurs d'attribut appropriées. Reportez-vous à Types de magasins de données OpenSSO pour obtenir des liens vers les définitions d'attributs de chaque type de magasin de données.
  8. Cliquez sur Terminer.
