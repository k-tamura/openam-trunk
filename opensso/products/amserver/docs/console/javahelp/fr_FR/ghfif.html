Authentificateur d'agent

Un authentificateur d'agent est un type d'agent qui, une fois qu'il est authentifié, peut obtenir les données en lecture seule de profils d'agent sélectionnés. Les profils d'agent peuvent être de tout type (J2EE, WSP, Détection, etc.) mais doivent appartenir au même domaine. Les utilisateurs qui possèdent les références de l'authentificateur d'agent (nom d'utilisateur et mot de passe) peuvent lire les données du profil agent, mais n'ont pas les droits de création, de mise à jour et de suppression de l'agent Admin.

L'authentificateur d'agent comporte les attributs suivants :

Mot de passe

Le mot de passe a été défini lors de la création du profil d'authentificateur d'agent. Cependant, vous pourrez changer de mot de passe à tout moment.

Confirmation de mot de passe

Le mot de passe a été confirmé lors de la création du profil d'authentificateur d'agent. Si vous modifiez le mot de passe, vous devez confirmer votre modification.

Statut

L'option Active est sélectionnée lors de la création de l'authentificateur d'agent. Ne choisissez Inactive que si vous voulez supprimer la protection assurée par l'agent.

Profils d'agent disposant d'un droit de lecture

Cet attribut définit une liste d'agents OpenSSO dont les données du profil sont lues par l'authentificateur d'agent. Les agents peuvent être de tout type (J2EE, WSP, Détection, etc.), mais doivent appartenir au même domaine. Pour ajouter un agent à la liste, sélectionnez le nom de l'agent et cliquez sur Ajouter.
