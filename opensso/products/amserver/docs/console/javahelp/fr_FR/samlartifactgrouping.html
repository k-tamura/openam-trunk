Artéfact

Voici les attributs du groupe Artéfact.

Délai d'artéfact

Cet attribut spécifie le délai de création d'une assertion pour un artéfact. La valeur par défaut est 400.

Nom de l'artéfact SAML

Cet attribut assigne un nom de variable à un artéfact SAML. Cet artéfact est composé de données avec limite de taille identifiant une assertion et un site source. Il est inclus dans la chaîne de requête URL et redirigé jusqu'au site de destination. Le nom par défaut est SAMLart. Si vous utilisez SAMLart, la chaîne de requête de redirection pourrait être http://hôte:port/URI-déploiement/SamlAwareServlet?TARGET=URL-cible/&SAMLart=artifact123
