Ajout d'un fournisseur de services à une entité fournisseur

La première étape du processus d'ajout d'un fournisseur de services à une entité fournisseur consiste à définir le type de fournisseur de services ajouté.

  • Un fournisseur hébergé est hébergé sur le même serveur qu'Access Manager.

  • Un fournisseur distant n'est pas hébergé sur le même serveur qu'Access Manager.

La procédure suivante décrit l'étape initiale. Le point de départ est Entités sous l'interface Fédération.

  1. Sélectionnez le nom d'une entité fournisseur configurée pour modifier son profil.

    La page du profil de l'entité s'affiche.

  2. Dans le menu Afficher, sélectionnez Fournisseur de services.
  3. Sélectionnez le type du fournisseur que vous ajoutez :

    • Choisissez Nouveau fournisseur central si le fournisseur est hébergé sur le même serveur qu'Access Manager.

    • Choisissez Nouveau fournisseur distant si le fournisseur n'est pas hébergé sur le même serveur que Federated Access Manager.

    La page du profil du fournisseur de services s'affiche.

  4. Selon votre sélection à l'étape précédente, faites votre choix parmi les possibilités suivantes :
    • Ajout d'un nouveau fournisseur de services hébergé à une entité fournisseur
    • Ajout d'un nouveau fournisseur de services distant à une entité fournisseur
