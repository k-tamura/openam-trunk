URL cibles

Si l'URL cible reçu par l'intermédiaire d'un profil SAML est mentionné comme valeur pour cet attribut, les assertions reçues seront adressées à l'URL cible via une commande HTTP FORM POST.

Remarque – Pour modifier ou dupliquer les attributs d'un profil de partenaire de confiance, cliquez sur le bouton correspondant dans la colonne Actions à côté du nom du partenaire de confiance configuré.
