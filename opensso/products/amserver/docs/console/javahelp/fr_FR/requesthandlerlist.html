Liste de gestionnaires de demandes

Service de liaison SOAP fournit l'interface RequestHandler pour traiter le messages de demande et renvoyer une réponse. Cette interface doit être implémentée côté serveur par chaque service d'identité basé sur Liberty utilisant Service de liaison SOAP. L'attribut Liste de gestionnaires de demandes stocke les informations sur les classes d'implémentation des services Web qui implémentent le gestionnaire de demandes. The Request Handler List displays entries that contain key/value pairs separated by a pipe as in: key=disco|class=com.example.identity.liberty.ws.disco.DiscoveryService . Ces paramètres sont expliqués dans le tableau suivant.

Paramètre

En-tête

key

Le paramètre obligatoire key est la dernière partie du chemin de l'URI vers une extrémité SOAP. Le point d'arrivée SOAP dans OpenSSO est le servlet récepteur SOAP. The URI to the SOAP Receiver is: protocol://host:port /deploy-uri/Liberty/key If you define disco as the key, the URI path to the SOAP endpoint for the corresponding Service de découverte would be: protocol://host:port /amserver/Liberty/disco. Des clients de services différents utilisent des clés différentes pour la connexion au récepteur SOAP.

class

Le paramètre obligatoire class spécifie le nom de la classe d'implémentation RequestHandler du service d'identité. For example: class=com.example.identity.liberty.ws.disco.DiscoveryService .

Pour ajouter un nouveau gestionnaire de demandes, reportez-vous à Configuration d'un gestionnaire de demandes.
