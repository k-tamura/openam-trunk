探索代理程式屬性

[探索代理程式] 設定檔保存了信任授權配置，Web 服務的用戶端/設定檔可以使用它來與「Liberty 探索」服務通訊，以進行 Web 服務查找、註冊及取得安全憑證。

群組

群組機制可讓您定義一個相似類型的代理程式的集合。您必須先定義群組，才能將特定代理程式加入集合中。

密碼

定義「探索代理程式」的密碼。

密碼確認

確認密碼。

狀態

定義代理程式在系統中將是使用中或非使用中。依預設，此屬性會設為使用中，表示代理程式將參與保護來自 Web 服務用戶端的外送 Web 服務請求的安全，並參與驗證來自 Web 服務提供者的 Web 服務回應。

代理程式配置儲存庫的位置

此屬性為「探索代理程式」定義配置儲存庫的代理程式位置。

私有金鑰別名

此屬性會定義私有憑證金鑰別名，該別名會用來簽署 Web 服務請求或解密 Web 服務回應。

探索服務端點

此屬性定義探索服務端點，信任授權用戶端會在其中建立通訊，以進行服務註冊與查找。

認證 Web 服務端點

此屬性定義認證服務端點，Web 服務用戶端會使用此端點來進行認證，方法是使用一般使用者的 SSOToken 接收探索服務資源提供 (亦稱為啟動程式資源提供)
