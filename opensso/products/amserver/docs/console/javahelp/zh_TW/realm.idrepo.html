資料存放區

資料存放區是一個資料庫，您可在其中儲存使用者屬性與使用者配置資料。

OpenSSO 提供可連接至 LDAPv3 身份識別儲存庫架構的身份識別儲存庫外掛程式。這樣便可讓您檢視並擷取 OpenSSO 使用者資訊，而無需對您現有的使用者資料庫進行變更。OpenSSO 架構會整合身份識別儲存庫外掛程式與其他 OpenSSO 外掛程式的資料，以組成每個使用者的虛擬身份。OpenSSO 稍後可在多個身份識別儲存庫間的認證與授權程序中使用通用識別。當使用者階段作業結束時，將銷毀虛擬使用者識別。

OpenSSO 支援以下資料存放區：

  • Active Directory

  • 通用 LDAPv3

  • 具有 OpenSSO 模式的 Sun Directory Server

請參閱建立新的資料存放區，以取得如何建立新的資料存放區及配置儲存庫外掛程式的相關資訊。
