自訂 SAMLv2 XACML PDP

XACML PDP 包含下列用於自訂的屬性：

  • 協定支援列舉

  • 簽署金鑰別名

  • 加密金鑰別名

  • 基本授權

  • 已簽名授權決定查詢

  • 授權服務

協定支援列舉

顯示此提供者支援的 XACML PDP 發行版本。

urn:liberty:iff:2003-08 是指 Liberty Identity Federation Framework 1.2 版。

urn:liberty:iff:2002-12 是指 Liberty Identity Federation Framework 1.1 版。

簽署金鑰別名

定義用來簽署請求和回應的金鑰別名。

加密金鑰別名

定義 XACML 加密的金鑰別名。

基本授權

啟用基本授權可保護 SOAP 端點。任何存取這些端點的提供者都必須擁有下列兩項特性中定義的使用者和密碼：使用者名稱和密碼。

已簽名授權決定查詢

啟用時，此屬性會強制 XACML 授權決定的所有查詢都進行簽名。

授權服務

此屬性定義授權請求的類型 (連結)，以及接收請求的 URL 端點。依預設，連結類型為 SOAP。
