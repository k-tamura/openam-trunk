全域化設定

全域化設定服務包含全域屬性，可讓您針對不同語言環境及字元集配置 OpenSSO。這些屬性包括：

  • 每種語言環境支援的字元集

  • 字元集別名

  • 自動產生的共用名稱格式

每種語言環境支援的字元集

此屬性列出每種語言環境支援的字元集，指示語言環境與字元集之間的對映。格式如下所示：

若要增加新建支援的字元集，請按一下 [新增]，然後定義下列參數：

語言環境

您要新增的語言環境。請參閱支援的語言環境，以取得更多資訊。

支援的字元集

輸入指定的語言環境支援的字元集。字元集是以分號隔開。例如，charset=charset1;charset2;charset3;...;charsetn

若要編輯現有的支援字元集，請按一下支援的字元集表格中的名稱。當完成時，請按一下 [確定]。

字元集別名

此屬性列出將用於傳送回應的字碼集名稱 (對映至 IANA 名稱)。這些字碼集名稱不需要與 Java 字碼集名稱相符。目前存在一種雜湊表，可以將 Java 字元集對映至 IANA 字元集，反之亦然。

若要增加新建字元集別名，請按一下 [新增] 按鈕，然後定義下列參數：

MIME 名稱

IANA 對映名稱。例如，Shift_JIS

Java 名稱

要對映至 IANA 字元集的 Java 字元集。

若要編輯任何現有的字元集別名，請按一下表格中的名稱。當完成時，請按一下 [確定]。

自動產生的共用名稱格式

此顯示選項允許您定義自動產生名稱的方式，以容納不同語言環境和字元集的名稱格式。預設語法如下 (請注意，定義中包含的逗號和/或空格將顯示在名稱格式中)：

en_us = {givenname} {initials} {sn}

例如，如果您希望以新的名稱格式，即以中文字元集顯示帶有 uid (11111) 的使用者 (User One)，請以如下格式定義：

zh = {sn}{givenname}({uid})

顯示如下：

OneUser 11111
