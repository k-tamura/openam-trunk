一般

[一般] 屬性配置了集中式伺服器管理的基本配置資料。

網站屬性

網站屬性為：

父系網站

此屬性會將負載平衡器 [網站名稱] (網站 ID) 對映至 OpenSSO 伺服器。請注意，您必須先建立網站，才能新增該網站。

系統屬性

系統屬性會列出伺服器實例的位置資訊：

基底安裝目錄

指定產品資料所在的基底目錄。

預設語言環境

語言環境值是安裝 OpenSSO 所使用的預設語言子類型。預設值為 en_us。

通知 URL

通知服務端點的位置。此值會在安裝期間設定。

XML 驗證

預設值為 [否]。決定在使用 OpenSSO XMLUtils 類別剖析 XML 文件時是否需要驗證。此特性只有在 [除錯等級] 屬性設為「警告」或「訊息」時才會生效。允許的值為「是」和「否」。只有在此特性的值為「是」，且 [除錯等級] 屬性的值設為警告或訊息時 XML 文件驗證才會啟用。

除錯屬性

[除錯] 屬性會列出基本的錯誤檢查資訊：

除錯等級

指定除錯等級。預設值為「錯誤」。可能的值包括：

  • 關閉 — 不會建立除錯檔案。

  • 錯誤 — 只會記錄錯誤訊息。

  • 警告 — 只會記錄警告訊息。

  • 訊息 — 錯誤、警告和資訊訊息都會記錄。

除錯目錄

指定將在其中建立除錯檔案的輸出目錄。值會在安裝期間設定。範例：OpenSSO-base /debug_URI/debug。

郵件伺服器

[郵件伺服器] 屬性會列出郵件伺服器的主機名稱和連接埠：

郵件伺服器主機名稱

預設值為 localhost。指定郵件伺服器主機。

郵件伺服器連接埠號碼

預設值為 25。指定郵件伺服器連接埠。
