階段作業視窗

顯示目前登入 OpenSSO 的使用者及每一次登入的階段作業時間。這些顯示欄位包括：

使用者 ID

顯示目前登入使用者的使用者 ID。

剩餘時間 (分鐘)

顯示必須重新認證之前，使用者此階段作業所具有的剩餘時間 (以分鐘計算)。

最長階段作業時間 (分鐘)

顯示階段作業過期之前使用者可以登入，並且必須重新認證以重新取得存取權限的最長時間 (以分鐘計算)。

閒置時間 (分鐘)

顯示使用者已閒置的時間 (分鐘)。

最長閒置時間 (分鐘)

顯示在必須重新認證之前，使用者可以閒置的最大時間 (以分鐘計算)。時間限制由管理員在階段作業服務中定義。請參閱階段作業，以取得階段作業屬性定義。
