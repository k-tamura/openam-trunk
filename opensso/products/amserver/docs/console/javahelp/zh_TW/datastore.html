資料存放區

資料存放區是一個資料庫，您可在其中儲存使用者屬性與使用者配置資料。

Access Manager 提供一個連接至 LDAPv3 識別儲存庫架構的識別儲存庫外掛程式。這個新的模型可讓您檢視並擷取 Access Manager 使用者資訊，而不需對您現有的使用者資料庫進行變更。Access Manager 架構整合識別儲存庫外掛程式的資料與其他 Access Manager 外掛程式的資料以形成每位使用者的虛擬識別。Access Manager 稍後可在多個識別儲存庫間的認證與授權程序中使用通用識別。當使用者階段作業結束時，將銷毀虛擬使用者識別。

請參閱建立新的 LDAPv3 資料存放區，以取得如何建立新的資料存放區及配置儲存庫外掛程式的相關資訊。

如果以範圍模式安裝 Access Manager，則您可以配置 Access Manager SDK 以管理 Directory Server 資料。請參閱建立新的 Access Manager 儲存庫外掛程式。

建立新的 LDAPv3 資料存放區

  1. 選取要新增資料存放區的範圍。

  2. 按一下 [資料存放區] 標籤。

  3. 按一下 [資料存放區] 清單中的 [新建]。

  4. 輸入資料存放區的名稱。

  5. 定義 LDAPv3 儲存庫外掛程式的屬性。如需屬性的說明，請參閱 LDAPv3 儲存庫外掛程式屬性。

  6. 按一下 [完成]。
