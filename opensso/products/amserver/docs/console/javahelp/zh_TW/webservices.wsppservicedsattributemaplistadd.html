配置屬性對映

對映是一種定義識別資料片段保存器的屬性。以下是新增屬性至 Liberty 個人設定檔服務 的程序。起點是 Web 服務模組內的 Liberty 個人設定檔服務 配置畫面。

  1. 按一下 [新建] 以顯示新的屬性對映屬性，或按一下已配置的對映名稱以修改它。
  2. 根據下列資訊提供容器屬性值：
    名稱前綴

    輸入要對映的 Liberty 個人設定檔服務 識別屬性的名稱。

    LDAP 屬性

    輸入名稱前綴要對映至的 OpenSSO LDAP 屬性的名稱。

  3. 按一下 [確定] 以完成對映的配置。
  4. 按一下Liberty 個人設定檔服務 頁面上的 [儲存] 以完成服務配置。
