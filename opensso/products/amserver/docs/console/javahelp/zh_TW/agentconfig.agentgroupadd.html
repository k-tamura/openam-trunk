建立新群組

如果您要讓代理程式繼承群組的特定特性，則請建立群組。代理程式只能從其群組類型繼承特性。例如，Web 代理程式可以從 Web 代理程式群組繼承特性。

  1. 按一下 [存取控制] 標籤。
  2. 按一下群組所屬的範圍的名稱。
  3. 按一下 [代理程式] 標籤。
  4. 如有必要，請選取適當代理程式類型的標籤。
  5. 在 [群組] 區段中，按一下 [新建]。

    在 [名稱] 欄位中，輸入新群組的名稱。

  6. 只為 Web 代理程式和 J2EE 代理程式輸入 OpenSSO 伺服器 URL。

    例如，http://OpenSSO_EnterpriseHost.example.com:8080/OpenSSO 。

  7. 按一下 [建立]。

    主控台會建立代理程式群組，並且再次顯示包含至群組的連結的代理程式頁面。

    若要進行額外的群組配置，請按一下此連結，以顯示 [編輯] 代理程式群組頁面。

    您可設定用來配置群組的特性與用於個別代理程式的特性相同，僅有的不同之處是 [群組]、[密碼] 和 [密碼確認] 特性無法在群組層級使用。

    備註 – 同時，務必注意部份群組特性已指定了變數值，而且在大部分情況下不應該變更。以下為這類值的範例：

    @AGENT_PROTO@://@AGENT_HOST@:@AGENT_PORT@/amagent
