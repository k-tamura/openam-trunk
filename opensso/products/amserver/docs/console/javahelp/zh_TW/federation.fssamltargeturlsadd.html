配置目標 URL

可以針對每個將要透過 POST 接收指定的目標 URL，定義 (或修改) 以下子屬性。

  1. 按一下 [新建] 以新增目標 URL。

    會顯示 [將新 Post 新增至目標 URL]。您也可以選取已定義目標 URL 的 [編輯] 或 [複製] 按鈕來到達此頁面。

  2. 鍵入屬性的值。
    協定

    選擇 http 或 https。

    伺服器名稱

    TARGET URL 所在伺服器的名稱，例如 www.sun.com。

    連接埠

    此屬性包含了如 58080 的連接埠號。

    路徑

    此屬性包含 URI，如 /amserver/console。

  3. 按一下 [確定] 以完成目標 URL 的配置。
  4. 按一下 [SAML 設定檔 ] 頁面上的 [儲存] 以完成 SAML 配置。
