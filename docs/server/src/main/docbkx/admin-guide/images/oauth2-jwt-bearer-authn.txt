title JWT Bearer Client Authentication
autonumber

box "Client can be Issuer" #FAFAFA
    participant "Client" as Client
    participant "Issuer" as Issuer
end box
participant "OpenAM\nAuthorization Server" as AuthzServer

Client->Issuer: Request JWT

Issuer->Client: JWT (signed)

Client->AuthzServer: POST with client_assertion=The JWT\nclient_assertion_type=\nurn:ietf:params:oauth:client-assertion-type:jwt-bearer

AuthzServer->AuthzServer: Validate JWT

AuthzServer->Client: Response (for example with an access token)
